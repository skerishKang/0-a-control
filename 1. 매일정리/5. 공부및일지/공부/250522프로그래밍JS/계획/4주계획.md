## 🗺️ 자바스크립트 학습 간단 로드맵

### 🔰 1단계: 기본 문법 익히기 (1주)

* 목표: 자바스크립트의 기초 문법 이해

* 학습 주제:

  * 변수와 자료형 (`var`, `let`, `const`)
  * 연산자 (`+`, `-`, `==`, `===`, `&&`, `||`)
  * 조건문 (`if`, `else`, `switch`)
  * 반복문 (`for`, `while`)
  * 함수 (`function`, 화살표 함수)

* 추천 자료:

  * [MDN JavaScript Guide](https://developer.mozilla.org/ko/docs/Web/JavaScript/Guide)
  * 생활코딩 JavaScript 기초: [https://opentutorials.org/course/3084](https://opentutorials.org/course/3084)

---

### 🔧 2단계: 브라우저와 DOM 조작 (1주)

* 목표: HTML 요소를 자바스크립트로 제어

* 학습 주제:

  * `document.getElementById`, `querySelector`
  * 이벤트 처리 (`onclick`, `addEventListener`)
  * HTML/CSS 변경하기 (`innerText`, `style` 등)

* 실습 예시:

  * 버튼 클릭 시 텍스트 바꾸기
  * 폼 입력값 받아서 출력하기

---

### 🔁 3단계: 배열, 객체, 반복 활용 (1\~2주)

* 목표: 자바스크립트의 구조화된 데이터 처리

* 학습 주제:

  * 배열 (`push`, `map`, `filter`, `forEach`)
  * 객체와 키-값 구조
  * JSON 데이터 구조 이해

* 실습 예시:

  * 할일 목록 앱 (To-do list) 만들기
  * 간단한 정보 카드 리스트 출력

---

### 🚀 4단계: 간단한 프로젝트로 실력 다지기 (1\~2주)

* 목표: 스스로 코딩해보며 전체 흐름 익히기

* 추천 미니 프로젝트:

  * 계산기 만들기
  * 퀴즈 앱
  * 날씨 API 사용해서 날씨 정보 출력하기

* 추천 툴:

  * [CodePen](https://codepen.io/), [JSFiddle](https://jsfiddle.net/) – 웹 브라우저에서 바로 연습 가능
  * VSCode + Live Server 확장 – 로컬 개발환경
