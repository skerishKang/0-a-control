## 🔧 자바스크립트 2단계: DOM과 이벤트 다루기 (총 5일 플랜)

### 📅 **Day 1: DOM(Document Object Model) 이해하기**

* 개념: HTML 문서를 자바스크립트로 제어하기 위한 구조
* 필수 메서드:

  * `document.getElementById()`
  * `document.querySelector()` / `querySelectorAll()`
* 실습:

  * 특정 요소 가져와서 `console.log()`로 출력
  * 요소의 `innerText`와 `style` 속성 바꿔보기

> 예시 코드:

```javascript
const title = document.getElementById("main-title");
title.innerText = "바뀐 제목!";
title.style.color = "blue";
```

---

### 📅 **Day 2: 이벤트(Event) 다루기**

* 주요 이벤트 종류:

  * `click`, `mouseover`, `keydown`, `submit`
* 이벤트 등록 방식:

  * HTML 속성 방식: `<button onclick="...">`
  * JS 방식: `element.addEventListener('click', handler)`

> 예시 코드:

```javascript
document.getElementById("myBtn").addEventListener("click", () => {
  alert("버튼이 클릭됐습니다!");
});
```
