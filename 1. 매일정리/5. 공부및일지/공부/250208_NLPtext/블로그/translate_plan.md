# NLP 챕터별 번역·요약·실습 블로그 플랜 (2025.04.29)

## 1. 번역/포스팅 대상
- Stanford SLP3, CS224n 등 NLP 교재 PDF/MD/이미지 파일 전체
- 각 챕터별로 번역/요약/해설/실습/그림/표/서칭자료 포함

## 2. 진행 방식
- 1) 원문 읽기 → 2) 번역/요약/해설 → 3) 실습/코드/그림/표 정리 → 4) 내 의견/느낌 추가 → 5) 워드프레스 포스팅
- 출처, 저자, 공식 사이트 명확히 표기
- 표, 그림 등은 필요시 직접 제작 또는 공식 자료 일부 활용(출처 명시)

## 3. 일정
- 평일: 하루 1시간씩 한 챕터(또는 소단원)씩 번역/정리/포스팅
- 주말: 팟캐스트/영상 기획 및 제작 준비, 블로그용 미디어 콘텐츠 아이디어 회의

## 4. 예시 포스팅 구조
- 제목: [NLP 교재 요약] 4장 Naive Bayes와 Sentiment 분석 (한글 요약)
- 본문:
  - 원본 교재 정보(제목, 저자, 출처 링크)
  - 챕터 핵심 요약 (한글)
  - 주요 개념/공식/예제 (부분 번역)
  - 실습 코드/결과 (직접 실행해본 것)
  - 그림/표 (출처 명시)
  - 본인 의견/느낀점

## 5. 향후 확장
- 번역/요약 자료를 바탕으로 팟캐스트, 영상 콘텐츠 제작
- 미디어별 운영 계획은 추후 별도 파일로 분리 가능
