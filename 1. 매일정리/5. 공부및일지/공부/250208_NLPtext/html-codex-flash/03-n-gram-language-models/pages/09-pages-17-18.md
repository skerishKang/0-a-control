- source_text: How are these λ values set? Both the simple interpolation and conditional interpolation λs are learned from a held-out corpus. A held-out corpus is an additional training corpus that we use to set hyperparameters like these λ values, by choosing the λ values that maximize the likelihood of the held-out corpus. ## Block 8
- type: paragraph
- source_text: In order for a backoff model to give a correct probability distribution, we have to discount the higher-order n-grams to save some probability mass for the lower order n-grams. Just as with add-one smoothing, if the higher-order n-grams aren’t discounted and we just used the undiscounted MLE probability, then as soon as we replaced an n-gram which has zero probability with a lower-order n-gram, we would be adding probability mass, and the total probability assigned to all possible strings by the language model would be greater than 1! In addition to this explicit discount factor, we’ll need a function α to distribute this probability mass to the lower order n-grams.
- korean_translation: 백오프 모델이 올바른 확률 분포를 형성하려면, 하위 차수 N-그램에 배정할 확률 질량을 확보하기 위해 고차 N-그램의 확률을 디스카운트(삭감)해야 합니다. 1 더하기 스무딩에서와 마찬가지로, 만약 고차 N-그램을 깎아내지 않고 가공되지 않은 MLE 확률을 그대로 사용한다면, 제로 확률인 N-그램을 하위 차수 N-그램으로 대체하는 순간 확률 질량이 추가되어 버립니다. 결과적으로 언어 모델이 가능한 모든 문자열에 할당하는 총 확률의 합이 1을 초과하게 될 것입니다! 따라서 이러한 명시적 디스카운트 계수 외에도, 확보된 확률 질량을 하위 차수 N-그램들에게 적절히 배분해 주기 위한 함수 $\alpha$가 필요합니다.
