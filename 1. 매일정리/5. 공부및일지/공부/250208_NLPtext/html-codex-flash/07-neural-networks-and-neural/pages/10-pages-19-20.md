## Block 4
- type: paragraph
- source_text: In the backward pass, we compute each of these partials along each edge of the graph from right to left, using the chain rule just as we did above. Thus we begin by computing the downstream gradients from node L, which are \frac{\partial L}{\partial e} and \frac{\partial L}{\partial c}. For node e, we then multiply this upstream gradient \frac{\partial L}{\partial e} by the local gradient (the gradient of the output with respect to the input), \frac{\partial e}{\partial d} to get the output we send back to node d: \frac{\partial L}{\partial d}. And so on, until we have annotated the graph all the way to all the input variables. The forward pass conveniently already will have computed the values of the forward intermediate variables we need (like d and e) to compute these derivatives. Fig. 7.16 shows the backward pass.
- korean_translation: 역방향 패스(backward pass)에서는 그래프의 각 에지를 따라 오른쪽에서 왼쪽으로, 위에서 했던 방식대로 연쇄 법칙을 사용하여 각 편미분값들을 계산합니다. 따라서 노드 $L$로부터 나오는 하류 그래디언트들인 $\frac{\partial L}{\partial e}$와 $\frac{\partial L}{\partial c}$를 계산하며 시작합니다. 노드 $e$의 경우, 이 상류 그래디언트 $\frac{\partial L}{\partial e}$에 로컬 그래디언트(입력에 대한 출력의 그래디언트)인 $\frac{\partial e}{\partial d}$를 곱하여 노드 $d$로 다시 전달할 출력값인 $\frac{\partial L}{\partial d}$를 구합니다. 이런 식으로 모든 입력 변수에 도달할 때까지 그래프를 채워 나갑니다. 순방향 패스에서 이러한 미분값들을 계산하는 데 필요한 순방향 중간 변수들($d, e$ 등)의 값을 이미 편리하게 계산해 두었을 것입니다. 그림 7.16은 이 역방향 패스 과정을 보여줍니다.
