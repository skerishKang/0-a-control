19.2.1 프롭뱅크의 수식어: ARGM
Study Note
동사가 요구하는 필수 논항 외에, 시간, 장소, 방식 등의 부가 정보를 나타내는 수식어(Adjuncts)에 대해 다릅니다.
Block 2
paragraph
Source Text
In addition to the numbered core roles, PropBank defines a set of adjunct roles, labeled as ARGM (Argument Modifier). These roles express general information that can apply to many different predicates.
Korean Translation
번호가 매겨진 핵심 역할들 외에도, 프롭뱅크는 ARGM(논항 수식어)으로 레이블링된 수식어 역할 세트를 정의합니다. 이 역할들은 다양한 술어에 보편적으로 적용될 수 있는 일반적인 정보를 표현합니다.
Study Note
필수 성분인 ARG와 부가 성분인 ARGM의 차이를 설명합니다.

ARGM-TMP(시간) / ARGM-LOC(장소)
Study Note
시간(Temporal)과 장소(Locative) 정보를 나타내는 대표적인 ARGM의 예입니다.
Block 4
list
Source Text
- ARGM-TMP: When did the event occur?
- ARGM-LOC: Where did the event occur?

Korean Translation
- ARGM-TMP: 사건이 언제 발생했는가?
- ARGM-LOC: 사건이 어디서 발생했는가?
- ARGM-MNR: 동작이 어떻게 수행되었는가?
- ARGM-CAU: 사건이 왜 발생했는가?
Study Note
주요 ARGM 유형들의 질문 형식을 통해 그 의미를 정의합니다.

Block 3
margin_note
Source Text
Frame Semantics
Korean Translation
프레임 의미론(Frame Semantics)
Study Note
단어의 의미는 그것이 속한 특정 상황(프레임)에 대한 배경 지식을 통해서 이해된다는 이론입니다.
Block 4
paragraph
Source Text
FrameNet is a lexical database of English that is based on the theory of frame semantics. Unlike PropBank, FrameNet organizes words into a set of conceptual structures called frames.
Korean Translation
프레임넷은 프레임 의미론 이론을 기반으로 구축된 영어 어휘 데이터베이스입니다. 프롭뱅크와는 다르게, 프레임넷은 단어들을 프레임(frames)이라고 불리는 일련의 개념적 구조로 조직화합니다.
Study Note
프롭뱅크가 '동사' 중심이라면, 프레임넷은 '상황(Frame)' 중심으로 단어들을 연결한다는 차별점을 강조합니다.
Block 5
paragraph
Source Text
Each frame defines a set of Frame Elements (FEs), which are the participants in the frame. For example, the Commerce_buy frame includes FEs such as Buyer, Goods, and Seller.
Korean Translation
각 프레임은 프레임 요소(Frame Elements, FE) 세트를 정의하며, 이는 프레임 내부의 참여자들을 의미합니다. 예를 들어, 'Commerce_buy' 프레임은 구매자(Buyer), 상품(Goods), 그리고 판매자(Seller)와 같은 FE들을 포함합니다.
Study Note
프레임넷의 논항 명칭인 프레임 요소(FE)와 그 실례를 보여줍니다.
