## Block 3
- type: paragraph
- source_text: Modern SRL systems rely on neural architectures that learn to represent the input sentence directly from text. These models typically use a BiLSTM or a Transformer as the encoder.
- korean_translation: 현대의 SRL 시스템들은 입력 문장을 텍스트로부터 직접 표현하도록 학습하는 신경망 아키텍처에 의존합니다. 이러한 모델들은 전형적으로 BiLSTM이나 트랜스포머(Transformer)를 인코더로 사용합니다.
- study_note: 신경망 모델이 문장의 순방향과 역방향 맥락을 모두 파악하는 인코더 구조를 핵심으로 삼고 있음을 설명합니다.

## Block 4
- type: margin_note
- source_text: BiLSTM / Transformer
- korean_translation: BiLSTM / 트랜스포머
- study_note: 시퀀스 데이터 처리에 최적화된 양방향 LSTM과 최근 NLP의 표준이 된 트랜스포머 모델입니다.

## Block 5
- type: paragraph
- source_text: A common neural architecture for SRL involves representing each word as a vector (e.g., using BERT embeddings) and then passing these vectors through several layers to predict the role of each argument.
- korean_translation: SRL을 위한 일반적인 신경망 아키텍처는 각 단어를 벡터(예: BERT 임베딩 사용)로 표현한 다음, 이 벡터들을 여러 층(layers)에 통과시켜 각 논항의 역할을 예측하는 과정을 포함합니다.
- study_note: 단어 임베딩(Embedding) -> 인코더(Encoder) -> 역할 분류기(Classifier)로 이어지는 신경망 SRL의 기본 흐름을 설명합니다.

## Block 2

- korean_translation: 19.6.1 그래프 기반 신경망 SRL
- study_note: 문장의 통사 의존 트리(Dependency Tree)를 그래프 구조로 활용하여 논항 간의 관계를 더 명시적으로 파악하는 방식을 다룹니다.

## Block 3
- type: paragraph
- source_text: Some neural SRL systems incorporate syntactic information by representing the dependency tree as a graph and using Graph Convolutional Networks (GCNs) to learn representations that reflect the syntactic structure.
- korean_translation: 일부 신경망 SRL 시스템은 의존 트리를 그래프로 표현하고 GCN(Graph Convolutional Networks)을 사용하여 통사 구조를 반영하는 표현을 학습함으로써 통사 정보를 통합합니다.
- study_note: 통사 분석 결과를 그래프 인공 신경망(GNN)의 일종인 GCN에 주입하는 하이브리드 접근법을 설명합니다.

## Block 4
- type: margin_note
- source_text: GCN
- korean_translation: 그래프 합성곱 신경망(GCN)
- study_note: 그래프 데이터에서 주변 노드의 정보를 집약하여 특징을 추출하는 신경망 모델입니다.

## Block 5
- type: paragraph
- source_text: This approach is motivated by the observation that syntactic structure is often highly correlated with semantic roles, and reflecting this structure can lead to better performance.
- korean_translation: 이러한 접근법은 통사 구조가 흔히 의미역과 매우 높은 상관관계가 있으며, 이 구조를 반영하는 것이 더 나은 성능으로 이어질 수 있다는 관찰에서 비롯되었습니다.
- study_note: '통사적 의존 관계'가 '의미적 역할 관계'를 예측하는 데 강력한 힌트가 된다는 점을 강조합니다.
