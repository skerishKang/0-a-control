Source Text
The regular expression /[1234567890]/ specifies any single digit. While such classes of characters as digits or letters are important building blocks in expressions, they can get awkward (e.g., it's inconvenient to specify /[ABCDEFGHIJKLMNOPQRSTUVWXYZ]/ to mean "any capital letter"). In cases where there is a well-defined sequence associated with a set of characters, the brackets can be used with the dash (-) to specify any one character in a range. The pattern /[2-5]/ specifies any one of the characters 2, 3, 4, or 5. The pattern /[b-g]/ specifies one of the characters b, c, d, e, f, or g. Some other examples are shown in Fig. 2.3.
Korean Translation
정규 표현식 /[1234567890]/은 임의의 한 자리 숫자를 지정합니다. 숫자나 문자와 같은 문자 클래스는 표현식을 구성하는 중요한 요소이지만, 때로는 작성하기 번거로울 수 있습니다. (예를 들어, "임의의 대문자 하나"를 나타내기 위해 /[ABCDEFGHIJKLMNOPQRSTUVWXYZ]/라고 일일이 쓰는 것은 불편합니다.) 문자 집합에 잘 정의된 순서가 있는 경우, 대괄호 안에 대시(-)를 사용하여 범위 내의 임의의 한 문자를 지정할 수 있습니다. /[2-5]/ 패턴은 2, 3, 4, 5 중 하나를 의미하며, /[b-g]/ 패턴은 b, c, d, e, f, g 중 하나를 의미합니다. 그림 2.3에서 다른 예시들을 볼 수 있습니다.

The square braces can also be used to specify what a single character cannot be, by use of the caret ^. If the caret ^ is the first symbol after the open square brace [, the resulting character class is negated. For example, the pattern /[^a]/ matches any single character (including special characters) except a. This is only true when the caret is the first symbol after the open square brace. If it occurs anywhere else, it usually stands for a caret; Fig. 2.4 shows some examples.
Korean Translation
대괄호는 캐럿(^) 기호를 사용하여 단일 문자가 될 수 없는 대상을 지정하는 데에도 쓰일 수 있습니다. 캐럿(^)이 여는 대괄호([) 바로 다음의 첫 번째 기호로 오면, 그 결과로 나타나는 문자 클래스는 부정(negated)됩니다. 예를 들어 /[^a]/ 패턴은 a를 제외한 모든 단일 문자(특수 문자 포함)와 일치합니다. 이는 오직 캐럿이 여는 대괄호 바로 다음의 첫 번째 기호일 때만 해당합니다. 그 외의 위치에 캐럿이 나타나면 대개 단순한 캐럿 기호 자체를 의미합니다. 그림 2.4에 몇 가지 예시가 나와 있습니다.
Study Note
캐럿(^) 기호의 위치에 따른 중의적 의미를 설명합니다. 대괄호 시작 직후에 오면 '부정'의 의미를 갖지만, 다른 곳에서는 단순 문자로 취급된다는 점이 핵심입니다.

paragraph
Source Text
This language consists of strings with a b, followed by at least two a's, followed by an exclamation point. The set of operators that allows us to say things like "some number of as" are based on the asterisk or *, commonly called the Kleene * (generally pronounced "cleany star"). The Kleene star means "zero or more occurrences of the immediately previous character or regular expression". So /a*/ means "any string of zero or more as". This will match a or aaaaaa, but it will also match Off Minor since the string Off Minor has zero a's. So the regular expression for matching one or more a is /aa*/, meaning one a followed by zero or more as. More complex patterns can also be repeated. So /[ab]*/ means "zero or more a's or b's" (not "zero or more right square braces"). This will match strings like aaaa or ababab or bbbb.
Korean Translation
이 언어는 b 다음에 적어도 두 개의 a가 오고, 그 뒤에 느낌표가 오는 문자열들로 이루어져 있습니다. "어떤 수의 a들" 같은 표현을 가능하게 해 주는 연산자 집합은 별표, 즉 보통 클레이니 별표(Kleene *)라고 불리는 기호 *에 기반합니다. 클레이니 별표는 "바로 앞의 문자 또는 정규 표현식이 0번 이상 나타남"을 뜻합니다. 따라서 /a*/는 "a가 0번 이상 나타나는 임의의 문자열"을 의미합니다. 이것은 a나 aaaaaa와 일치할 뿐 아니라, Off Minor처럼 a가 0개인 문자열과도 일치합니다. 따라서 하나 이상의 a와 일치하는 정규 표현식은 /aa*/이며, 이는 a 하나 다음에 0개 이상의 a가 오는 형태를 뜻합니다. 더 복잡한 패턴도 반복될 수 있습니다. 따라서 /[ab]*/는 "a 또는 b가 0번 이상 나타남"을 의미합니다("오른쪽 대괄호가 0번 이상 나타남"이라는 뜻이 아닙니다). 이 표현은 aaaa, ababab, bbbb 같은 문자열과 일치합니다.
Study Note
클레이니 별표(*)는 반복을 표현하는 가장 핵심적인 수량자입니다. 특히 0번도 허용한다는 점 때문에, 해당 문자가 전혀 없는 문자열과도 일치할 수 있다는 점을 꼭 기억해야 합니다. 또한 `[ab]*`처럼 문자 집합과 결합하면 복합적인 반복 패턴을 간단하게 표현할 수 있습니다.
