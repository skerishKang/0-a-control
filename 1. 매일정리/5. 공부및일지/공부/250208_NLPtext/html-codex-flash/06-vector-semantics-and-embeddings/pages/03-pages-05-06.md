## Block 4
- type: paragraph
- source_text: In this chapter we introduce vector semantics, the standard modern way to represent meaning. The fundamental idea of vector semantics is to represent a word as a point in some multi-dimensional semantic space. This point is represented by a vector of numbers. These vectors are often called embeddings, because the word is “embedded” in a particular space.
- korean_translation: 이 장에서는 의미를 표현하는 표준적인 현대적 방식인 벡터 의미론(vector semantics)을 소개합니다. 벡터 의미론의 기본 아이디어는 단어를 다차원 의미 공간상의 한 점으로 표현하는 것입니다. 이 점은 숫자로 이루어진 벡터로 나타내어집니다. 단어가 특정 공간에 “박혀(embedded)” 있기 때문에 이러한 벡터들을 흔히 임베딩(embeddings)이라고 부릅니다.
- study_note: 벡터 의미론과 임베딩의 핵심 정의를 설명합니다.

## Block 5
- type: paragraph
- source_text: We’ve seen the roots of this idea already; the 3-dimensional affective model of Osgood et al. Beginning in the 1970s and 1980s, vectors began to be used for representing word meaning for many applications, and the 2013 introduction of word2vec (Mikolov et al., 2013) made them the standard tool for all of NLP.
- korean_translation: 우리는 이미 이 아이디어의 뿌리를 보았습니다. 이전 섹션(4페이지)에서 다룬 Osgood 등(1957)의 3차원 정서 모델은 단어 의미를 벡터로 표현한 초기 사례였습니다. 1970년대와 1980년대부터 다양한 응용 프로그램을 위해 단어 의미를 벡터로 표현하기 시작했으며, 2013년 word2vec(Mikolov et al., 2013)의 등장은 이를 모든 NLP의 표준 도구로 만들었습니다.
- study_note: 벡터 의미론의 역사적 흐름과 word2vec이 가져온 전성기를 언급합니다.

## Block 7
- type: paragraph
- source_text: There are many ways to represent word meaning as a vector. One very simple way is to use a one-hot vector, a vector of length |V| (the size of the vocabulary) that has a 1 at a particular index and 0 everywhere else. For example, if the vocabulary is [aardvark, abacus, ..., zebra], the one-hot vector for abacus might be [0, 1, 0, ..., 0].
- korean_translation: 단어 의미를 벡터로 표현하는 방법에는 여러 가지가 있습니다. 매우 간단한 방법 중 하나는 원핫 벡터(one-hot vector)를 사용하는 것입니다. 이는 어휘 목록 크기 |V|와 같은 길이를 가진 벡터로, 특정 인덱스에만 1이 있고 나머지는 모두 0인 벡터입니다. 예를 들어 어휘 목록이 [aardvark, abacus, ..., zebra]라면, abacus에 대한 원핫 벡터는 [0, 1, 0, ..., 0]이 될 것입니다.
- study_note: 가장 기초적인 벡터 표현인 원핫 벡터의 정의를 설명합니다.

## Block 8
- type: paragraph
- source_text: One-hot vectors have two main disadvantages. First, they are very high-dimensional and sparse (mostly 0s). Second, they have no representation of similarity; any two one-hot vectors are orthogonal (their dot product is 0), so there is no way to tell that cat is more similar to dog than it is to refrigerator. In this chapter we’ll see how to create dense vectors that are lower dimensional and represent similarity.
- korean_translation: 원핫 벡터에는 두 가지 주요 단점이 있습니다. 첫째, 차원이 매우 높고 희소(sparse, 대부분이 0)합니다. 둘째, 유사성을 표현하지 못합니다. 어떤 두 원핫 벡터도 서로 직교(orthogonal, 내적이 0)하므로, cat이 refrigerator보다 dog와 더 유사하다는 것을 알 방법이 없습니다. 이 장에서는 차원이 낮으면서 유사성을 표현할 수 있는 밀집 벡터(dense vectors)를 만드는 방법을 살펴볼 것입니다.
- study_note: 원핫 벡터의 한계(고차원, 유사성 결여)와 이를 해결하기 위한 밀집 벡터의 필요성을 강조합니다.

## Block 10
- type: paragraph
- source_text: The term-document matrix was developed in the 1960s and 1970s for information retrieval, but it is also a way to represent words as vectors. In a term-document matrix, each row is a word (a term) and each column is a document in a collection. Each entry in the matrix represents the count of how many times that word occurs in that document.
- korean_translation: 단어-문서 행렬(term-document matrix)은 1960년대와 1970년대에 정보 검색을 위해 개발되었지만, 단어를 벡터로 표현하는 방법이기도 합니다. 단어-문서 행렬에서 각 행은 단어(용어, term)이고 각 열은 컬렉션 내의 문서입니다. 행렬의 각 항목은 해당 단어가 해당 문서에서 몇 번 나타나는지를 나타내는 횟수입니다.
- study_note: 문서 내 단어 빈도를 이용한 고전적인 단어-문서 행렬의 구조를 설명합니다.

## Block 6
- type: paragraph
- source_text: In many applications, we are interested in representing the meaning of a word not by looking at which documents it occurs in, but by looking at which words occur near it. This gives us a word-word co-occurrence matrix (also called a term-term matrix or a word-context matrix).
- korean_translation: 많은 응용 프로그램에서 우리는 단어가 어떤 문서에 나타나는지뿐만 아니라 어떤 단어들이 그 근처에 나타나는지 봄으로써 단어의 의미를 표현하는 데 관심을 가집니다. 이를 통해 단어-단어 공기 행렬(word-word co-occurrence matrix)(단어-단어 행렬 또는 단어-문맥 행렬이라고도 함)을 얻을 수 있습니다.
- study_note: '단어-문서' 관계에서 보다 직접적인 '단어-단어' 관계로 관심사를 확장합니다.

## Block 7
- type: paragraph
- source_text: In a word-word matrix, the rows are words in the vocabulary, and the columns are also words in the vocabulary. Each cell represents the number of times the row word (the target word) and the column word (the context word) occur together in a context window. A context window is a set of words surrounding the target word (e.g., the 4 words to the left and 4 words to the right).
- korean_translation: 단어-단어 행렬에서 행은 어휘 목록 내의 단어들이고, 열 또한 어휘 목록 내의 단어들입니다. 각 셀은 행의 단어(대상 단어, target word)와 열의 단어(문맥 단어, context word)가 문맥 창(context window) 내에서 함께 나타나는 횟수를 나타냅니다. 문맥 창은 대상 단어를 둘러싼 일련의 단어들을 의미합니다 (예: 왼쪽 4단어와 오른쪽 4단어).
- study_note: 문맥 창을 정의하고, 이를 통해 단어 간의 국소적인 공기 관계를 데이터화하는 방식을 설명합니다.
