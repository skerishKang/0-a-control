## Block 3
- type: paragraph
- source_text: The CKY algorithm can be extended to find the most likely parse tree for a sentence using PCFGs. In the probabilistic version of CKY, each entry table[i, j, A] stores the maximum probability of non-terminal A deriving the segment w[i+1...j].
- korean_translation: CKY 알고리즘은 PCFG를 사용하여 문장에 대한 가장 가능성 높은 구문 분석 트리를 찾도록 확장될 수 있습니다. 확률적 버전의 CKY에서 각 칸 table[i, j, A]는 비단말 기호 A가 문자열 부분 w[i+1...j]를 유도할 수 있는 최대 확률을 저장합니다.
- study_note: 확률적 CKY 표의 각 엔트리가 저장하는 정보의 정의를 설명합니다.

## Block 4
- type: code
- source_text:
    function PROBABILISTIC-CKY-PARSE(words, grammar) returns table
      for j = 1 to n do
        for each A -> words[j] in grammar do
          table[j-1, j, A] = P(A -> words[j])
        for i = j-2 down to 0 do
          for k = i+1 to j-1 do
            for each A -> B C in grammar do
              table[i, j, A] = max(table[i, j, A], P(A -> B C) * table[i, k, B] * table[k, j, C])
      return table
- korean_translation:
    함수 PROBABILISTIC-CKY-PARSE(words, grammar) 반환 table
      j = 1부터 n까지 반복
        문법의 각 A -> words[j]에 대해 반복
          table[j-1, j, A] = P(A -> words[j])
        i = j-2부터 0까지 역순 반복
          k = i+1부터 j-1까지 반복
            문법의 각 A -> B C에 대해 반복
              table[i, j, A] = max(table[i, j, A], P(A -> B C) * table[i, k, B] * table[k, j, C])
      표(table) 반환
- study_note: 확률 값을 곱하여 최대 확률을 찾아나가는 확률적 CKY 알고리즘의 의사 코드를 보여줍니다.
