## Block 5
- type: paragraph
- source_text: Chunking is the process of identifying non-overlapping contiguous constituents in a sentence, called chunks. Chunks are typically base phrases, meaning they do not contain other phrases of the same type.
- korean_translation: 청킹은 문장에서 중첩되지 않고 연속적인 성분인 청크(chunks)를 식별하는 과정입니다. 청크는 일반적으로 기본 구(base phrases)로, 동일한 유형의 다른 구를 포함하지 않는 것을 의미합니다.
- study_note: 청킹과 청크의 정의, 그리고 '기본 구'의 속성을 설명합니다.

## Block 2
- type: margin_note
- source_text: BIO tagging
- korean_translation: BIO 태깅(BIO tagging)
- study_note: B(Begin), I(Inside), O(Outside) 태그를 사용하여 청크의 경계를 표시하는 방식입니다.

## Block 3
- type: table
- source_text:
    | Word | POS | Chunk Tag |
    | :--- | :--- | :--- |
    | The | DT | B-NP |
    | flight | NN | I-NP |

    | Chicago | NNP | B-NP |
    | was | VBD | O |

- korean_translation:
    | 단어 | 품사(POS) | 청크 태그(Chunk Tag) |
    | :--- | :--- | :--- |
    | The | DT | B-NP |
    | flight | NN | I-NP |
    | from | IN | O |
    | Chicago | NNP | B-NP |
    | was | VBD | O |
    | delayed | VBN | O |
- study_note: 문장의 각 단어에 부여된 BIO 태그를 통해 청크의 시작과 내부, 외부를 구분하는 방법을 보여줍니다.

## Block 4
- type: paragraph
- source_text: In BIO tagging, B-NP denotes the beginning of an NP chunk, I-NP denotes the inside of an NP chunk, and O denotes that a word is outside any chunk.
- korean_translation: BIO 태깅에서 B-NP는 NP 청크의 시작을 나타내고, I-NP는 NP 청크의 내부를 나타내며, O는 단어가 어떤 청크에도 속하지 않음을 나타냅니다.
- study_note: BIO 태그 각각의 의미와 역할을 상세하게 정의합니다.
