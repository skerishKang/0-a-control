- source_text: “The desire to build a computer that can understand natural language is as old as the computer itself. But the goal of modern computational linguistics is to enable us to answer a number of important questions. How can we tell if a string of words is a sentence in a language? What is the structure of a sentence? How can we use this structure to help us find the meaning of the sentence?”
- korean_translation: “자연어를 이해할 수 있는 컴퓨터를 만들고자 하는 욕망은 컴퓨터 자체의 탐구만큼이나 오래되었습니다. 하지만 현대 계산 언어학의 목표는 우리가 여러 중요한 질문에 답할 수 있게 하는 것입니다. 일련의 단어들이 특정 언어의 문장인지 어떻게 알 수 있을까요? 문장의 구조는 무엇일까요? 이 구조를 문장의 의미를 찾는 데 어떻게 활용할 수 있을까요?”

## Block 4

- source_text: Parsing is the process of taking a string and a grammar and returning a structural representation for that string. In this chapter, we focus on constituency parsing, using the context-free grammars (CFGs) introduced in Chapter 12.
- korean_translation: 파싱은 문자열과 문법을 입력받아 해당 문자열에 대한 구조적 표현을 반환하는 과정입니다. 이 장에서는 12장에서 소개한 문맥 자유 문법(CFG)을 사용하는 구 구조 파싱에 초점을 맞춥니다.

## Block 6
- type: paragraph
- source_text: A grammar can be used as a synthesizer to generate sentences from a start symbol. But in language processing, we are usually interested in using a grammar as an analyzer to find the structure for a given sentence. This task is called parsing.
- korean_translation: 문법은 시작 기호로부터 문장을 생성하는 합성기로 사용될 수 있습니다. 하지만 언어 처리에서는 대개 주어진 문장의 구조를 찾아내는 분석기로서 문법을 사용하는 데 관심이 있습니다. 이 작업을 파싱이라고 합니다.
- study_note: 문법의 두 가지 활용 방식 중 언어 처리에서 주로 다루는 분석적 측면을 설명합니다.

## Block 3
- type: paragraph
- source_text: As we discussed in Chapter 12, natural language is highly ambiguous. For example, the sentence "I saw the man with a telescope" can have two different parse trees.
- korean_translation: 12장에서 논의했듯이, 자연어는 고도의 모호성을 지닙니다. 예를 들어, "I saw the man with a telescope"라는 문장은 두 가지 서로 다른 구문 분석 트리를 가질 수 있습니다.
- study_note: 전치사구 부착 모호성의 대표적인 예시를 다시 언급합니다.

## Block 4
- type: paragraph
- source_text: Structural ambiguity occurs when a sentence can be assigned more than one parse tree. This is a problem for parsers because they must decide which tree is the correct one.
- korean_translation: 구조적 모호성은 한 문장에 대해 두 개 이상의 구문 분석 트리가 할당될 수 있을 때 발생합니다. 이는 파서에게 문제가 되는데, 어떤 트리가 올바른 것인지 결정해야 하기 때문입니다.
- study_note: 구조적 모호성이 파서의 성능에 미치는 영향을 설명합니다.

## Block 5
- type: paragraph
- source_text: The number of possible parse trees can grow exponentially with the length of the sentence. This is known as the combinatorial explosion of parse trees.
- korean_translation: 가능한 구문 분석 트리의 수는 문장의 길이에 따라 기하급수적으로 늘어날 수 있습니다. 이를 구문 분석 트리의 조합 폭발(combinatorial explosion)이라고 합니다.
- study_note: 문장이 길어질수록 분석의 복잡도가 급격히 증가함을 경고합니다.

## Block 8
- type: paragraph
- source_text: The CKY algorithm is a dynamic programming algorithm for parsing strings with context-free grammars. Dynamic programming is a general technique for solving complex problems by breaking them down into smaller, simpler subproblems.
- korean_translation: CKY 알고리즘은 문맥 자유 문법을 사용하여 문자열을 파싱하는 동적 프로그래밍 알고리즘입니다. 동적 프로그래밍은 복잡한 문제를 더 작고 단순한 하위 문제들로 나누어 해결하는 일반적인 기법입니다.
- study_note: CKY 알고리즘의 배경이 되는 동적 프로그래밍의 개념을 설명합니다.
