The CKY algorithm can be extended to find the most likely parse tree for a sentence using PCFGs. In the probabilistic version of CKY, each entry table[i, j, A] stores the maximum probability of non-terminal A deriving the segment w[i+1...j].

Block 4
code
Source Text
    function PROBABILISTIC-CKY-PARSE(words, grammar) returns table
      for j = 1 to n do
        for each A -> words[j] in grammar do
          table[j-1, j, A] = P(A -> words[j])
        for i = j-2 down to 0 do
          for k = i+1 to j-1 do
            for each A -> B C in grammar do
              table[i, j, A] = max(table[i, j, A], P(A -> B C) * table[i, k, B] * table[k, j, C])
      return table
Korean Translation
    함수 PROBABILISTIC-CKY-PARSE(words, grammar) 반환 table
      j = 1부터 n까지 반복
        문법의 각 A -> words[j]에 대해 반복
          table[j-1, j, A] = P(A -> words[j])
        i = j-2부터 0까지 역순 반복
          k = i+1부터 j-1까지 반복
            문법의 각 A -> B C에 대해 반복
              table[i, j, A] = max(table[i, j, A], P(A -> B C) * table[i, k, B] * table[k, j, C])
      표(table) 반환
Study Note
