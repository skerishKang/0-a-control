"The desire to build a computer that can understand natural language is as old as the computer itself. But the goal of modern computational linguistics is to enable us to answer a number of important questions. How can we tell if a string of words is a sentence in a language? What is the structure of a sentence? How can we use this structure to help us find the meaning of the sentence?"
Korean Translation
"자연어를 이해할 수 있는 컴퓨터를 만들고자 하는 욕망은 컴퓨터 자체의 탐구만큼이나 오래되었습니다. 하지만 현대 계산 언어학의 목표는 우리가 여러 중요한 질문에 답할 수 있게 하는 것입니다. 일련의 단어들이 특정 언어의 문장인지 어떻게 알 수 있을까요? 문장의 구조는 무엇일까요? 이 구조를 문장의 의미를 찾는 데 어떻게 활용할 수 있을까요?"

Source Text
A grammar can be used as a synthesizer to generate sentences from a start symbol. But in language processing, we are usually interested in using a grammar as an analyzer to find the structure for a given sentence. This task is called parsing.
Korean Translation
문법은 시작 기호로부터 문장을 생성하는 합성기로 사용될 수 있습니다. 하지만 언어 처리에서는 대개 주어진 문장의 구조를 찾아내는 분석기로서 문법을 사용하는 데 관심이 있습니다. 이 작업을 파싱이라고 합니다.
Study Note
문법의 두 가지 활용 방식 중 언어 처리에서 주로 다루는 분석적 측면을 설명합니다.

Block 3
paragraph
Source Text
As we discussed in Chapter 12, natural language is highly ambiguous. For example, the sentence "I saw the man with a telescope" can have two different parse trees.
Korean Translation
12장에서 논의했듯이, 자연어는 고도의 모호성을 지닙니다. 예를 들어, "I saw the man with a telescope"라는 문장은 두 가지 서로 다른 구문 분석 트리를 가질 수 있습니다.
Study Note
전치사구 부착 모호성의 대표적인 예시를 다시 언급합니다.
Block 4
paragraph
Source Text
Structural ambiguity occurs when a sentence can be assigned more than one parse tree. This is a problem for parsers because they must decide which tree is the correct one.
Korean Translation
구조적 모호성은 한 문장에 대해 두 개 이상의 구문 분석 트리가 할당될 수 있을 때 발생합니다. 이는 파서에게 문제가 되는데, 어떤 트리가 올바른 것인지 결정해야 하기 때문입니다.
Study Note
구조적 모호성이 파서의 성능에 미치는 영향을 설명합니다.
Block 5
paragraph
Source Text
The number of possible parse trees can grow exponentially with the length of the sentence. This is known as the combinatorial explosion of parse trees.
Korean Translation
가능한 구문 분석 트리의 수는 문장의 길이에 따라 기하급수적으로 늘어날 수 있습니다. 이를 구문 분석 트리의 조합 폭발(combinatorial explosion)이라고 합니다.
Study Note
문장이 길어질수록 분석의 복잡도가 급격히 증가함을 경고합니다.

Block 8

Source Text
The CKY algorithm is a dynamic programming algorithm for parsing strings with context-free grammars. Dynamic programming is a general technique for solving complex problems by breaking them down into smaller, simpler subproblems.
Korean Translation
CKY 알고리즘은 문맥 자유 문법을 사용하여 문자열을 파싱하는 동적 프로그래밍 알고리즘입니다. 동적 프로그래밍은 복잡한 문제를 더 작고 단순한 하위 문제들로 나누어 해결하는 일반적인 기법입니다.
Study Note
CKY 알고리즘의 배경이 되는 동적 프로그래밍의 개념을 설명합니다.
