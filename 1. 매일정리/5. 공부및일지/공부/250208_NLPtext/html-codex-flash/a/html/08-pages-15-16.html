    function FORWARD-BACKWARD(observations of len T, output vocabulary V, hidden state set Q) returns HMM=(A,B)
      initialize A and B
      iterate until convergence
        E-step
          gamma_t(j) = (αt(j) βt(j)) / αT(qF) for all t and j
          xi_t(i,j) = (αt(i) aij bj(ot+1) βt+1(j)) / αT(qF) for all t, i, and j
        M-step
          a_hat_ij = Σ_{t=1}^{T-1} xi_t(i,j) / Σ_{t=1}^{T-1} Σ_{k=1}^{N} xi_t(i,k)
          b_hat_j(vk) = Σ_{t=1, s.t. Ot=vk}^{T} gamma_t(j) / Σ_{t=1}^{T} gamma_t(j)
      return A, B

    function FORWARD-BACKWARD(길이 T의 관측치, 출력 어휘 V, 은닉 상태 집합 Q) returns HMM=(A,B)
      A와 B 초기화
      수렴할 때까지 반복
        E-단계 (Expectation step)
          모든 t와 j에 대해, gamma_t(j) = (αt(j) βt(j)) / αT(qF)
          모든 t, i, j에 대해, xi_t(i,j) = (αt(i) aij bj(ot+1) βt+1(j)) / αT(qF)

          a_hat_ij = Σ_{t=1}^{T-1} xi_t(i,j) / Σ_{t=1}^{T-1} Σ_{k=1}^{N} xi_t(i,k)

      return A, B

Source Text
Although in principle the forward-backward algorithm can do completely unsupervised learning of the A and B parameters, in practice the initial conditions are very important. For this reason the algorithm is often given extra information. For example, for HMM-based speech recognition, the HMM structure is often set by hand, and only the emission (B) and (non-zero) A transition probabilities are trained from a set of observation sequences O.
Korean Translation
원칙적으로 전방-후방 알고리즘은 A와 B 매개변수에 대해 완전한 비지도 학습을 수행할 수 있지만, 실제로는 초기 조건이 매우 중요합니다. 이러한 이유로 알고리즘에 추가 정보가 제공되는 경우가 많습니다. 예를 들어, HMM 기반 음성 인식의 경우 HMM 구조는 수동으로 설정하고, 일련의 관측 시퀀스 O로부터 방출 확률(B)과 (0이 아닌) 전이 확률(A)만을 훈련시킵니다.

    - Hidden Markov models (HMMs) are a way of relating a sequence of observations to a sequence of hidden classes or hidden states that explain the observations.
    - The process of discovering the sequence of hidden states, given the sequence of observations, is known as decoding or inference. The Viterbi algorithm is commonly used for decoding.
    - The parameters of an HMM are the A transition probability matrix and the B observation likelihood matrix. Both can be trained with the Baum-Welch or forward-backward algorithm.
Korean Translation
|
    - 은닉 마르코프 모델(HMM)은 일련의 관측치를 그 관측치를 설명하는 일련의 은닉 클래스 또는 은닉 상태와 연관시키는 방법입니다.
    - 관측 시퀀스가 주어졌을 때 은닉 상태 시퀀스를 찾아내는 과정을 디코딩(decoding) 또는 추론(inference)이라고 합니다. 디코딩에는 비터비(Viterbi) 알고리즘이 일반적으로 사용됩니다.
    - HMM의 매개변수는 전이 확률 행렬 A와 관측 우도 행렬 B입니다. 두 매개변수 모두 바움-웰치(Baum-Welch) 또는 전방-후방(forward-backward) 알고리즘으로 훈련할 수 있습니다.

As we discussed in Chapter 8, Markov chains were first used by Markov (1913), to predict whether an upcoming letter in Pushkin’s Eugene Onegin would be a vowel or a consonant. The hidden Markov model was developed by Baum and colleagues at the Institute for Defense Analyses in Princeton (Baum and Petrie 1966, Baum and Eagon 1967).
