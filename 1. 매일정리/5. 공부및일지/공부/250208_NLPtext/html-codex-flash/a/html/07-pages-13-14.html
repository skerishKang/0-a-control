Block 3
paragraph
Source Text
How do we compute the numerator? Here’s the intuition. Assume we had some estimate of the probability that a given transition i -> j was taken at a particular point in time t in the observation sequence. If we knew this probability for each particular time t, we could sum over all times t to estimate the total count for the transition i -> j.
Korean Translation
분자는 어떻게 계산할까요? 직관적인 설명은 다음과 같습니다. 관측 시퀀스의 특정 시점 t에서 주어진 전이 i -> j가 발생했을 확률에 대한 추정치가 있다고 가정해 봅시다. 각 시점 t에 대해 이 확률을 알고 있다면, 모든 시점 t에 대해 합산하여 전이 i -> j의 총 횟수를 추정할 수 있습니다.
Study Note
Intuition for using expected counts over time steps.

paragraph
Source Text
We also need a formula for recomputing the observation probability. This is the probability of a given symbol vk from the observation vocabulary V, given a state j: b_hat_j(vk). We will do this by trying to compute: b_hat_j(vk) = (expected number of times in state j and observing symbol vk) / (expected number of times in state j).
Korean Translation
또한 관측 확률을 재계산하기 위한 공식도 필요합니다. 이는 상태 j가 주어졌을 때 관측 어휘 V의 특정 기호 vk가 나타날 확률인 b_hat_j(vk)입니다. 우리는 이를 다음과 같이 계산합니다: b_hat_j(vk) = (상태 j에 있으면서 기호 vk를 관측한 기대 횟수) / (상태 j에 있었던 기대 횟수).
