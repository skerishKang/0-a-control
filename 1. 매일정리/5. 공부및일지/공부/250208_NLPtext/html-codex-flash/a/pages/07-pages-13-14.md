## Block 3
- type: paragraph
- source_text: How do we compute the numerator? Here’s the intuition. Assume we had some estimate of the probability that a given transition i -> j was taken at a particular point in time t in the observation sequence. If we knew this probability for each particular time t, we could sum over all times t to estimate the total count for the transition i -> j.
- korean_translation: 분자는 어떻게 계산할까요? 직관적인 설명은 다음과 같습니다. 관측 시퀀스의 특정 시점 t에서 주어진 전이 i -> j가 발생했을 확률에 대한 추정치가 있다고 가정해 봅시다. 각 시점 t에 대해 이 확률을 알고 있다면, 모든 시점 t에 대해 합산하여 전이 i -> j의 총 횟수를 추정할 수 있습니다.
- study_note: Intuition for using expected counts over time steps.
