## Block 3
- type: code
- source_text:
  function FORWARD(observations of len T, state-graph of len N) returns forward-prob
  create a probability matrix forward[N,T]
  for each state s from 1 to N do ; initialization step
  forward[s,1] ← πs * bs(o1)
  for each time step t from 2 to T do ; recursion step
  for each state s from 1 to N do
  forward[s,t] ← Σ s'=1 N forward[s',t − 1] * as',s * bs(ot)
  forwardprob ← Σ s=1 N forward[s,T] ; termination step
  return forwardprob

## Block 5
- type: formula
- source_text:
  1. Initialization:
  α1(j) = πj bj(o1) 1 ≤ j ≤ N
  2. Recursion:
  αt(j) = Σ i=1 N αt−1(i) aij bj(ot); 1 ≤ j ≤ N, 1 < t ≤ T
  3. Termination:
  P(O|λ) = Σ i=1 N αT(i)
- korean_translation:
  1. 초기화:
  α1(j) = πj bj(o1) 1 ≤ j ≤ N
  2. 재귀:
  αt(j) = Σ i=1 N αt−1(i) aij bj(ot); 1 ≤ j ≤ N, 1 < t ≤ T
  3. 종료:
  P(O|λ) = Σ i=1 N αT(i)
- study_note: Forward 알고리즘의 수식적 정의를 세 단계로 요약한 것입니다. 최종적으로 모든 마지막 상태의 전방 확률을 더하면 전체 시퀀스의 우도 P(O|λ)가 됩니다.
