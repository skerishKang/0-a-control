## Block 13
- type: paragraph
- source_text: Instead of using such an extremely exponential algorithm, we use an efficient O(N^2T) algorithm called the forward algorithm. The forward algorithm is a kind of dynamic programming algorithm, that is, an algorithm that uses a table to store intermediate values as it builds up the probability of the observation sequence. The forward algorithm computes the observation probability by summing over the probabilities of all possible hidden state paths that could generate the observation sequence, but it does so efficiently by implicitly folding each of these paths into a single forward trellis.
- korean_translation: 이러한 극단적인 지수적 알고리즘 대신, 우리는 Forward 알고리즘이라고 불리는 효율적인 O(N^2T) 알고리즘을 사용합니다. Forward 알고리즘은 동적 계획법(dynamic programming) 알고리즘의 일종으로, 관찰 시퀀스의 확률을 쌓아 올릴 때 중간 값을 저장하기 위해 테이블을 사용하는 알고리즘입니다. Forward 알고리즘은 관찰 시퀀스를 생성할 수 있는 모든 가능한 은닉 상태 경로의 확률을 합산하여 관찰 확률을 계산하지만, 각 경로를 단일 전방 트렐리스(forward trellis)로 암묵적으로 접음으로써 이를 효율적으로 수행합니다.
- study_note: Forward 알고리즘의 핵심인 동적 계획법과 트렐리스 구조를 설명합니다. 복잡도를 N^T에서 N^2T로 획기적으로 낮춥니다.
