## Block 9
- type: paragraph
- source_text: The final layer matrix V provides a way to score the likelihood of each word in the vocabulary given the evidence present in the final hidden layer of the network through the calculation of Vh. This entails that it also has the dimensionality |V| × d_h. That is, the rows of V provide a second set of learned word embeddings that capture relevant aspects of word meaning and function. This leads to an obvious question – is it even necessary to have both? Weight tying is a method that dispenses with this redundancy and uses a single set of embeddings at the input and softmax layers. That is, E = V. To do this, we set the dimensionality of the final hidden layer to be the same d_h, (or add an additional projection layer to do the same thing), and simply use the same matrix for both layers. In addition to providing improved perplexity results, this approach significantly reduces the number of parameters required for the model.
- korean_translation: 마지막 층의 행렬 $V$는 $Vh$ 계산을 통해 네트워크의 마지막 은닉층에 존재하는 증거들을 바탕으로 어휘 사전의 각 단어의 우도를 점수화하는 방법을 제공합니다. 이는 $V$ 또한 $|V| \times d_h$의 차원을 가짐을 의미합니다. 즉, $V$의 행들은 단어의 의미와 기능의 관련 측면들을 포착하는 두 번째 단어 임베딩 세트를 제공하는 셈입니다. 이는 당연한 질문으로 이어집니다. 과연 두 세트가 모두 필요할까요? 가중치 타잉(Weight tying)은 이러한 중복을 없애고 입력층과 소프트맥스 층에서 단일 임베딩 세트를 사용하는 방법입니다. 즉, $E = V$입니다. 이를 위해 마지막 은닉층의 차원을 $d_h$와 동일하게 설정(또는 동일한 작업을 수행하는 추가 프로젝션 층을 추가)하고, 단순히 두 층에 동일한 행렬을 사용합니다. 이 접근 방식은 향상된 당혹도 결과를 제공할 뿐만 아니라, 모델에 필요한 파라미터 수를 상당히 줄여줍니다.
- study_note: 가중치 타잉(Weight Tying)의 원리($E=V$)와 그 효과(파라미터 감소, 성능 향상)를 설명합니다.
