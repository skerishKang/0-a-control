- source_text: Transformers map sequences of input vectors (x_1, ..., x_n) to sequences of output vectors (y_1, ..., y_n) of the same length. ## Block 10
- type: paragraph
- source_text: Fig. 9.15 illustrates the flow of information in a single causal, or backward looking, self-attention layer. As with the overall transformer, a self-attention layer maps input sequences (x_1, ..., x_n) to output sequences of the same length (y_1, ..., y_n). When processing each item in the input, the model has access to all of the inputs up to and including the one under consideration, but no access to information about inputs beyond the current one. In addition, the computation performed for each item is independent of all the other computations. The first point ensures that we can use this approach to create language models and use them for autoregressive generation, and the second point means that we can easily parallelize both forward inference and training of such models.
- korean_translation: 그림 9.15는 단일 인과적(causal), 또는 하방 참조형(backward looking) 자기 주의 집중 층에서의 정보 흐름을 보여줍니다. 전체 트랜스포머와 마찬가지로, 자기 주의 집중 층은 입력 시퀀스 $(x_1, \dots, x_n)$를 동일한 길이의 출력 시퀀스 $(y_1, \dots, y_n)$로 매핑합니다. 입력의 각 항목을 처리할 때 모델은 현재 고려 중인 항목을 포함하여 그 이전의 모든 입력에 접근할 수 있지만, 현재 항목 이후의 입력 정보에는 접근할 수 없습니다. 또한 각 항목에 대해 수행되는 연산은 다른 모든 연산과 독립적입니다. 첫 번째 특징은 우리가 이 접근 방식을 사용하여 언어 모델을 만들고 자기 회귀 생성에 사용할 수 있음을 보장하며, 두 번째 특징은 이러한 모델의 순방향 추론과 학습을 모두 쉽게 병렬화할 수 있음을 의미합니다.
- study_note: 인과적 자기 주의 집중(Causal Self-Attention)의 정의(현재 이전만 참조)와 그로 인한 병렬화 가능성을 설명합니다.
