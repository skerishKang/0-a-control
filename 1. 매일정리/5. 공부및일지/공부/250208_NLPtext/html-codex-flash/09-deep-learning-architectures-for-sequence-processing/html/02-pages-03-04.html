Block 9

Source Text
A recurrent neural network (RNN) is any network that contains a cycle within its network connections, meaning that the value of some unit is directly, or indirectly, dependent on its own earlier outputs as an input. While powerful, such networks are difficult to reason about and to train. However, within the general class of recurrent networks there are constrained architectures that have proven to be extremely effective when applied to language. In this section, we consider a class of recurrent networks referred to as Elman Networks (Elman, 1990) or simple recurrent networks. These networks are useful in their own right and serve as the basis for more complex approaches like the Long Short-Term Memory (LSTM) networks discussed later in this chapter. In this chapter when we use the term RNN we'll be referring to these simpler more constrained networks (although you will often see the term RNN to mean any net with recurrent properties including LSTMs).
Korean Translation
순환 신경망(RNN)은 네트워크 연결 내에 사이클(cycle)을 포함하는 모든 네트워크를 말합니다. 즉, 어떤 유닛의 값이 입력으로서 자신의 이전 출력에 직접적 또는 간접적으로 의존함을 의미합니다. 이러한 네트워크는 강력하지만 추론과 학습이 어렵습니다. 그러나 일반적인 순환 신경망 범주 내에는 언어에 적용했을 때 매우 효과적인 것으로 입증된 제약된 구조들이 있습니다. 이 섹션에서는 엘먼 네트워크(Elman, 1990) 또는 단순 순환 신경망(simple recurrent networks)이라 불리는 순환 신경망 부류를 살펴봅니다. 이 네트워크들은 그 자체로도 유용하며, 이 장의 뒷부분에서 다룰 장단기 메모리(LSTM) 신경망과 같은 더 복잡한 접근 방식의 기초가 됩니다. 이 장에서 RNN이라는 용어를 사용할 때는 이러한 더 단순하고 제약된 네트워크를 지칭할 것입니다(비록 LSTM을 포함하여 순환적 속성을 가진 모든 네트워크를 의미하는 용어로 RNN이 자주 쓰이기는 하지만 말입니다).

Study Note
RNN의 순방향 계산(추론) 과정을 수식과 함께 자세히 살펴보는 섹션입니다.
Block 8
paragraph
Source Text
Forward inference (mapping a sequence of inputs to a sequence of outputs) in an RNN is nearly identical to what we've already seen with feedforward networks. To compute an output y_t for an input x_t, we need the activation value for the hidden layer h_t. To calculate this, we multiply the input x_t with the weight matrix W, and the hidden layer from the previous time step h_{t−1} with the weight matrix U. We add these values together and pass them through a suitable activation function, g, to arrive at the activation value for the current hidden layer, h_t. Once we have the values for the hidden layer, we proceed with the usual computation to generate the output vector.
Korean Translation
RNN에서의 순방향 추론(입력 시퀀스를 출력 시퀀스로 매핑하는 과정)은 우리가 이미 피드포워드 신경망에서 보았던 것과 거의 동일합니다. 입력 x_t에 대한 출력 y_t를 계산하려면 은닉층 h_t의 활성화 값이 필요합니다. 이를 계산하기 위해 입력 x_t에 가중치 행렬 W를 곱하고, 이전 시간 단계의 은닉층 h_{t-1}에 가중치 행렬 U를 곱합니다. 우리는 이 값들을 더하고 적절한 활성화 함수 g를 통과시켜 현재 은닉층의 활성화 값 h_t를 얻습니다. 일단 은닉층 값을 얻으면, 평소와 같은 계산을 진행하여 출력 벡터를 생성합니다.
