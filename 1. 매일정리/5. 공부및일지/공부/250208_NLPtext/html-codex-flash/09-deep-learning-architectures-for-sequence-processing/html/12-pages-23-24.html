Source Text
To implement this notion, each head, i, in a self-attention layer is provided with its own set of key, query and value matrices: W_i^K, W_i^Q and W_i^V. These are used to project the inputs into separate key, value, and query embeddings separately for each head, with the rest of the self-attention computation remaining unchanged. In multi-head attention, instead of using the model dimension d that's used for the input and output from the model, the key and query embeddings have dimensionality d_k, and the value embeddings are dimensionality d_v. Thus for each head i, we have weight layers W_i^Q ∈ R^{d×d_k}, W_i^K ∈ R^{d×d_k}, and W_i^V ∈ R^{d×d_v}, and these get multiplied by the inputs packed into X to produce Q ∈ R^{N×d_k}, K ∈ R^{N×d_k}, and V ∈ R^{N×d_v}. The output of each of the h heads is of shape N × d_v, and so the output of the multi-head layer with h heads consists of h vectors of shape N × d_v. To make use of these vectors in further processing, they are combined and then reduced down to the original input dimension d. This is accomplished by concatenating the outputs from each head and then using yet another linear projection, W^O ∈ R^{hd_v×d}, to reduce it to the original output dimension for each token, or a total N × d output.
Korean Translation
이 개념을 구현하기 위해, 자기 주의 집중 층의 각 헤드 $i$에는 자신만의 키, 쿼리, 값 행렬 세트인 $W_i^K, W_i^Q, W_i^V$가 제공됩니다. 이들은 각 헤드에 대해 개별적으로 입력을 키, 값, 쿼리 임베딩으로 투영하는 데 사용되며, 나머지 자기 주의 집중 계산 과정은 동일하게 유지됩니다. 멀티헤드 주의 집중에서는 모델의 입출력에 사용되는 차원 $d$ 대신, 키와 쿼리 임베딩은 $d_k$ 차원을 갖고 값 임베딩은 $d_v$ 차원을 갖습니다. 따라서 각 헤드 $i$에 대해 우리는 가중치 층 $W_i^Q \in \mathbb{R}^{d \times d_k}, W_i^K \in \mathbb{R}^{d \times d_k}, W_i^V \in \mathbb{R}^{d \times d_v}$를 가지며, 이들은 $X$에 묶인 입력들과 곱해져 $Q \in \mathbb{R}^{N \times d_k}, K \in \mathbb{R}^{N \times d_k}, V \in \mathbb{R}^{N \times d_v}$를 생성합니다. $h$개 헤드 각각의 출력은 $N \times d_v$ 형태이며, 따라서 $h$개 헤드를 가진 멀티헤드 층의 출력은 $N \times d_v$ 형태의 벡터 $h$개로 구성됩니다. 후속 처리에서 이러한 벡터들을 활용하기 위해, 이들을 결합된 후 원래의 입력 차원 $d$로 다시 축소됩니다. 이는 각 헤드의 출력들을 결합(concatenate)한 뒤 또 다른 선형 투영 행렬 $W^O \in \mathbb{R}^{hd_v \times d}$를 사용하여 각 토큰에 대해 원래의 출력 차원으로 축소함으로써 수행되며, 총 $N \times d$의 출력을 만들어냅니다.
Study Note
멀티헤드 어텐션의 상세 구현 수식과 차원 변화 과정을 설명합니다. 각 헤드에서 축소된 차원으로 계산한 뒤 나중에 다시 합쳐($W^O$) 원래 차원을 맞추는 것이 핵심입니다.
