A bidirectional RNN (Schuster and Paliwal, 1997) combines two independent independent RNNs, one where the input is processed from the start to the end, and the other from the end to the start. We then concatenate the two representations computed by the networks into a single vector that captures both the left and right contexts of an input at each point in time. Here we use either the semicolon ";" or the equivalent symbol ⊕ to mean vector concatenation:
Korean Translation
양방향 RNN(Schuster and Paliwal, 1997)은 두 개의 독립적인 RNN, 즉 입력을 처음부터 끝까지 처리하는 것과 끝부터 처음으로 처리하는 것을 결합합니다. 그런 다음 우리는 각 시점에서 입력의 왼쪽과 오른쪽 문맥을 모두 포착하는 단일 벡터로 두 네트워크에서 계산된 표현들을 결합(concatenate)합니다. 여기서는 벡터 결합을 의미하기 위해 세미콜론 ";" 또는 그와 동등한 기호 ⊕를 사용합니다.

Source Text
Assigning a high probability to was following airline is straightforward since airline provides a strong local context for the singular agreement. However, assigning an appropriate probability to were is quite difficult, not only because the plural flights is quite distant, but also because the intervening context involves singular constituents. Ideally, a network should be able to retain the distant information about plural flights until it is needed, while still processing the intermediate parts of the sequence correctly.
Korean Translation
'airline' 뒤에 'was'가 올 확률을 높게 할당하는 것은 간단합니다. 'airline'이 단수 일치를 위한 강력한 국소적 문맥을 제공하기 때문입니다. 하지만 'were'에 적절한 확률을 할당하는 것은 매우 어렵습니다. 복수형인 'flights'가 상당히 멀리 떨어져 있을 뿐만 아니라, 그 사이의 문맥에 단수형 요소들이 포함되어 있기 때문입니다. 이상적으로 네트워크는 필요한 시점까지 복수형 'flights'에 대한 먼 정보를 유지할 수 있어야 하며, 그러면서도 시퀀스의 중간 부분들을 올바르게 처리해야 합니다.

paragraph
Source Text
One reason for the inability of RNNs to carry forward critical information is that the hidden layers, and, by extension, the weights that determine the values in the hidden layer, are being asked to perform two tasks simultaneously: provide information useful for the current decision, and updating and carrying forward information required for future decisions.
Korean Translation
RNN이 중요한 정보를 계속 전달하지 못하는 한 가지 이유는, 은닉층(그리고 은닉층의 값을 결정하는 가중치들)이 두 가지 작업을 동시에 수행하도록 요구받기 때문입니다. 즉, 현재의 결정을 내리는 데 유용한 정보를 제공하는 작업과, 미래의 결정을 위해 필요한 정보를 업데이트하고 전달하는 작업을 동시에 해야 합니다.
Study Note
RNN 은닉층이 가중한 부담(현재 예측 + 미래를 위한 기억)을 안고 있어 정보 보존이 어려움을 구조적 원인으로 제시합니다.
