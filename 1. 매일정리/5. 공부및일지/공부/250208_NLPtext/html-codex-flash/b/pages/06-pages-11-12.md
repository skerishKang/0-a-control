- source_text: Brill and Moore’s model also conditions each edit on where it is in the word. Thus instead of estimating P(f|ph), the model estimates P(f|ph, beginning). More formally, let R be a partition of the typo string x into adjacent substrings and T be a partition of the candidate string. The total likelihood P(x|w) is approximated by the probability of the single best partition.
- korean_translation: Brill과 Moore의 모델은 각 편집이 단어 안의 어느 위치에서 일어났는지도 함께 조건화합니다. 따라서 P(f|ph) 대신 P(f|ph, beginning)을 추정합니다. 더 형식적으로는, 오타 문자열 x를 인접한 부분 문자열들로 나눈 것을 R, 후보 문자열의 분할을 T라고 할 때, 전체 likelihood P(x|w)는 단일 최적 분할의 확률로 근사됩니다.
- study_note: 같은 편집이라도 단어의 처음, 중간, 끝에서 성격이 다를 수 있다는 점을 모델에 반영합니다.

    function SOUNDEX(name) returns soundex form
    1. Keep the first letter of name
    2. Drop all occurrences of non-initial a, e, h, i, o, u, w, y
    3. Replace the remaining letters with numbers:
       b, f, p, v -> 1
       c, g, j, k, q, s, x, z -> 2
       d, t -> 3
       l -> 4
       m, n -> 5
       r -> 6
    4. Replace identical adjacent numbers with one number
    5. Convert to Letter Digit Digit Digit
- korean_translation: |

    2. 첫 글자가 아닌 a, e, h, i, o, u, w, y는 제거한다
    3. 남은 글자를 숫자로 치환한다:
       b, f, p, v -> 1
       c, g, j, k, q, s, x, z -> 2

       m, n -> 5
       r -> 6
    4. 인접한 동일 숫자 열은 하나로 줄인다
    5. Letter Digit Digit Digit 형태로 만든다

- type: paragraph
- source_text: Pronunciation can also be incorporated directly into the noisy channel model. For example, Toutanova and Moore (2002) interpolate a spelling-based channel model with a pronunciation-based channel model. More recent work on names often uses Jaro-Winkler distance rather than Soundex, since it allows characters to move longer distances and gives higher similarity to strings that share initial characters.
- korean_translation: 발음 정보는 noisy channel 모델 안에 직접 통합될 수도 있습니다. 예를 들어 Toutanova와 Moore(2002)는 철자 기반 채널 모델과 발음 기반 채널 모델을 보간합니다. 이름 처리의 더 최근 연구는 Soundex 대신 Jaro-Winkler 거리를 자주 사용하는데, 이 거리는 더 긴 거리의 문자 이동을 허용하고 시작 글자가 같은 문자열에 더 높은 유사도를 부여하기 때문입니다.

## Block 5
- type: paragraph
- source_text: Algorithms for spelling error detection and correction have existed since at least Blair (1960). Most early algorithms were based on similarity keys like Soundex. Damerau (1964) gave a dictionary-based algorithm for error detection, and most detection algorithms since then have been dictionary-based. Although early work suggested that spelling dictionaries might need to be kept small, later work found that larger dictionaries were more helpful in practice.
- korean_translation: 철자 오류 탐지와 교정을 위한 알고리즘은 적어도 Blair(1960) 이후로 존재해 왔습니다. 초기 알고리즘의 대부분은 Soundex 같은 유사성 키에 기반했습니다. Damerau(1964)는 사전 기반 오류 탐지 알고리즘을 제안했고, 이후의 대부분의 탐지 알고리즘도 사전 기반이었습니다. 초기 연구에서는 철자 사전을 작게 유지해야 한다고 보기도 했지만, 이후 연구들은 실제로는 더 큰 사전이 더 유용하다는 점을 보여주었습니다.
- study_note: 철자 교정 연구의 초기 흐름은 Soundex류 유사성 키와 사전 기반 탐지로 요약할 수 있습니다.
