## Block 5
- type: paragraph
- source_text: To summarize, the noisy channel model says that we have some true underlying word w, and we have a noisy channel that modifies the word into some possible misspelled observed surface form. The likelihood or channel model of the noisy channel producing any particular observation sequence x is modeled by P(x|w). The prior probability of a hidden word is modeled by P(w). We can compute the most probable word wˆ given that we’ve seen some observed misspelling x by multiplying the prior P(w) and the likelihood P(x|w) and choosing the word for which this product is greatest.
- korean_translation: 요약하자면, 노이즈 채널 모델은 실제 기저 단어 w가 있고, 이 단어를 가능한 오타 형태인 표면 형태로 변형시키는 노이즈 채널이 있다고 가정합니다. 특정 관측 시퀀스 x를 생성하는 노이즈 채널의 우도 또는 채널 모델은 P(x|w)로 모델링됩니다. 숨겨진 단어의 사전 확률은 P(w)로 모델링됩니다. 관측된 오타 x가 주어졌을 때 가장 확률이 높은 단어 wˆ는 사전 확률 P(w)와 우도 P(x|w)를 곱하고 이 곱이 가장 큰 단어를 선택함으로써 계산할 수 있습니다.
- study_note: 노이즈 채널 모델의 핵심 구성 요소인 사전 확률(언어 모델)과 우도(오류 모델)를 정리합니다.

## Block 1
- type: figure
- source_text: function NOISY CHANNEL SPELLING(word x, dict D, lm, editprob) returns correction if x not in D: candidates, edits = All strings at edit distance 1 from x that are in D, and their edits; for each c, e in candidates, edits: channel = editprob(e); prior = lm(c); score[c] = log channel + log prior; return argmax_c score[c]
- korean_translation: 함수 NOISY CHANNEL SPELLING(단어 x, 사전 D, 언어모델 lm, 편집확률 editprob) 반환 교정값: 만약 x가 D에 없다면: 후보들, 편집들 = x로부터 편집 거리 1 이내에 있고 D에 있는 모든 문자열과 그 편집들; 후보들, 편집들의 각 c, e에 대해: 채널 = editprob(e); 사전 = lm(c); 점수[c] = log 채널 + log 사전; 반환 argmax_c 점수[c]
