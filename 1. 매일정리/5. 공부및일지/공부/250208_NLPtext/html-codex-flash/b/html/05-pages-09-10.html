Block 7
paragraph
Source Text
State of the art implementations of noisy channel spelling correction make a number of extensions to the simple models presented above. Rather than assuming only a single error in the sentence, modern systems process the input one word at a time. But naive use of the basic model tends to overcorrect, replacing correct but rare words with more frequent ones.
Korean Translation
최신 noisy channel 철자 교정 시스템은 위의 단순 모델에 여러 확장을 더합니다. 문장에 하나의 오류만 있다고 가정하는 대신, 현대 시스템은 입력을 한 번에 한 단어씩 처리합니다. 하지만 기본 모델을 단순 적용하면 올바르지만 드문 단어를 더 빈번한 단어로 바꾸는 과교정이 발생하기 쉽습니다.
Study Note
현대 시스템의 핵심 문제는 오류를 더 많이 잡는 것보다 과교정을 줄이는 데 있습니다.

We can also improve the performance of the noisy channel model by changing how the prior and the likelihood are combined. Block 7
paragraph
Source Text
Brill and Moore (2000) propose a channel model that models an error as being generated by a typist first choosing a word, then choosing a partition of the letters of that word, and then typing each partition, possibly erroneously. For example, the word physical might be partitioned as ph y s i c al, and a misspelled form like fisikle can be explained by edits such as p(f|ph) and p(le|al).
Korean Translation
Brill과 Moore(2000)는 입력자가 먼저 단어를 고르고, 그 단어의 글자들을 여러 부분으로 나눈 다음, 각 부분을 때로는 틀리게 입력한다고 보는 채널 모델을 제안합니다. 예를 들어 physical을 ph y s i c al처럼 분할하고, fisikle 같은 오타를 p(f|ph), p(le|al)과 같은 다중 문자 편집으로 설명할 수 있습니다.
Study Note
단일 문자 편집만으로는 설명하기 어려운 실제 오타를 다중 문자 변환으로 포착하려는 접근입니다.
