We can conveniently simplify Eq. B.3 by dropping the denominator P(x). Why is that? Since we are choosing a potential correction word out of all words, we will be computing \frac{P(x|w)P(w)}{P(x)} for each word. But P(x) doesn’t change for each word; we are always asking about the most likely word for the same observed error x, which must have the same probability P(x). Korean Translation
분모 P(x)를 생략하여 식 B.3을 편리하게 단순화할 수 있습니다. 왜 그럴까요? 모든 단어 중에서 잠재적인 교정 단어를 선택할 때 각 단어에 대해 \frac{P(x|w)P(w)}{P(x)}를 계산하게 됩니다. 하지만 P(x)는 각 단어마다 변하지 않습니다. 우리는 항상 동일하게 관측된 오류 x에 대해 가장 가능성 있는 단어를 묻고 있으며, 이는 동일한 확률 P(x)를 가져야 합니다. 따라서 이 더 단순한 공식을 최대화하는 단어를 선택할 수 있습니다.

Block 5
paragraph
Source Text
To summarize, the noisy channel model says that we have some true underlying word w, and we have a noisy channel that modifies the word into some possible misspelled observed surface form. The likelihood or channel model of the noisy channel producing any particular observation sequence x is modeled by P(x|w). The prior probability of a hidden word is modeled by P(w). We can compute the most probable word wˆ given that we’ve seen some observed misspelling x by multiplying the prior P(w) and the likelihood P(x|w) and choosing the word for which this product is greatest.
Korean Translation
요약하자면, 노이즈 채널 모델은 실제 기저 단어 w가 있고, 이 단어를 가능한 오타 형태인 표면 형태로 변형시키는 노이즈 채널이 있다고 가정합니다. 특정 관측 시퀀스 x를 생성하는 노이즈 채널의 우도 또는 채널 모델은 P(x|w)로 모델링됩니다. 숨겨진 단어의 사전 확률은 P(w)로 모델링됩니다. 관측된 오타 x가 주어졌을 때 가장 확률이 높은 단어 wˆ는 사전 확률 P(w)와 우도 P(x|w)를 곱하고 이 곱이 가장 큰 단어를 선택함으로써 계산할 수 있습니다.
Study Note
노이즈 채널 모델의 핵심 구성 요소인 사전 확률(언어 모델)과 우도(오류 모델)를 정리합니다.

Block 1
figure
Source Text
function NOISY CHANNEL SPELLING(word x, dict D, lm, editprob) returns correction if x not in D: candidates, edits = All strings at edit distance 1 from x that are in D, and their edits; for each c, e in candidates, edits: channel = editprob(e); prior = lm(c); score[c] = log channel + log prior; return argmax_c score[c]
Korean Translation
함수 NOISY CHANNEL SPELLING(단어 x, 사전 D, 언어모델 lm, 편집확률 editprob) 반환 교정값: 만약 x가 D에 없다면: 후보들, 편집들 = x로부터 편집 거리 1 이내에 있고 D에 있는 모든 문자열과 그 편집들; 후보들, 편집들의 각 c, e에 대해: 채널 = editprob(e); 사전 = lm(c); 점수[c] = log 채널 + log 사전; 반환 argmax_c 점수[c]
Study Note
노이즈 채널 모델을 이용한 철자 교정 알고리즘의 의사코드입니다. 확률의 곱 대신 로그 확률의 합을 사용하여 계산합니다.

The first stage of the algorithm proposes candidate corrections by finding words that have a similar spelling to the input word. Analysis of spelling error data has shown that the majority of spelling errors consist of a single-letter change and so we often make the simplifying assumption that these candidates have an edit distance of 1 from the error word.
Korean Translation
알고리즘의 첫 번째 단계는 입력 단어와 유사한 철자를 가진 단어를 찾아 후보 교정 단어를 제안하는 것입니다. 철자 오류 데이터 분석 결과, 대다수의 철자 오류가 한 글자 변경으로 구성된다는 것이 밝혀졌으며, 따라서 우리는 종종 이러한 후보들이 오류 단어로부터 편집 거리 1 이내에 있다는 단순화된 가정을 합니다.
Study Note
대부분의 오타가 1회의 편집 작업으로 발생한다는 통계적 근거에 기반한 가정입니다.

To find this list of candidates we’ll use the minimum edit distance algorithm introduced in Chapter 2, but extended so that in addition to insertions, deletions, and substitutions, we’ll add a fourth type of edit, transpositions, in which two letters are swapped. The version of edit distance with Damerau-Levenshtein transposition is called Damerau-Levenshtein edit distance. Applying all such single transformations to acress yields the list of candidate words in Fig. B.3.
