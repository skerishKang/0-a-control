22.5.2 Topic Modeling: Latent Dirichlet Allocation
Korean Translation
22.5.2 주제 모델링: 잠재 디리클레 할당(Latent Dirichlet Allocation, LDA)
Study Note
텍스트의 주제를 통계적으로 추출하는 대표적인 머신러닝 기법인 LDA를 다릅니다.
Block 3
paragraph
Source Text
Topic modeling is the task of automatically identifying the main themes or topics in a large collection of documents. Latent Dirichlet Allocation (LDA) is a generative statistical model for topic modeling.
Korean Translation
주제 모델링은 대규모 문서 집합에서 주요 테마나 주제를 자동으로 식별하는 작업입니다. 잠재 디리클레 할당(Latent Dirichlet Allocation, LDA)은 주제 모델링을 위한 생성적 통계 모델입니다.
Study Note
대규모 텍스트 데이터에서 '잠재된(Latent)' 주제를 추출하는 LDA의 개요입니다.

Source Text
LDA
Korean Translation
잠재 디리클레 할당(Latent Dirichlet Allocation)
Study Note
Blei 등(2003)에 의해 제안된 확률 기반의 주제 모델링 방식입니다.
Block 5
paragraph
Source Text
LDA assumes that each document is a mixture of various topics, and each topic is a distribution over words. The model aims to find the underlying topic structure that best explains the observed words in the documents.
Korean Translation
LDA는 각 문서가 여러 가지 주제들의 혼합이며, 각 주제는 단어들에 대한 확률 분포라고 가정합니다. 이 모델은 문서에서 관찰된 단어들을 가장 잘 설명하는 기저의 주제 구조를 찾는 것을 목표로 합니다.
Study Note
'문서는 주제의 혼합, 주제는 단어의 분포'라는 LDA의 핵심 가정을 설명합니다.

22.6 Dialogue and Discourse
Korean Translation
22.6 대화와 담화
Study Note
단방향 텍스트(Monologue)가 아닌, 화자와 청자가 주고받는 대화(Dialogue)에서의 특수한 담화 성격과 상호 작용을 다릅니다.
Block 3
paragraph
Source Text
Dialogue is a special form of discourse involving multiple participants. Coherence in dialogue depends not only on the logical connection between utterances but also on the interaction between the participants.
Korean Translation
대화는 여러 참여자가 포함된 특별한 형태의 담화입니다. 대화에서의 응집성은 발화들 사이의 논리적 연결뿐만 아니라 참여자들 사이의 상호 작용에도 의존합니다.
Study Note
대화의 응집성이 상호 작용적 성격을 띤다는 점을 강조합니다.

Source Text
dialogue acts
Korean Translation
대화 행위(dialogue acts)
Study Note
발화가 대화 내에서 수행하는 기능(질문, 답변 등)을 일컫는 용어입니다.
Block 5
paragraph
Source Text
Dialogue acts are the functional units of a conversation, representing the speaker's communicative intent. Examples include Greeting, Question, Inform, and Acknowledge.
Korean Translation
대화 행위는 대화의 기능적 단위이며, 화자의 의사소통 의도를 나타냅니다. 예로는 인사(Greeting), 질문(Question), 정보 전달(Inform), 그리고 응답(Acknowledge) 등이 있습니다.
Study Note
대화 행위의 정의와 다양한 예시를 나열합니다.
