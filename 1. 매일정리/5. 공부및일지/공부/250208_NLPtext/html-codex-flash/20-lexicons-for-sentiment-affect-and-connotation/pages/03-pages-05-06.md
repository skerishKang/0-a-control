## Block 2

## Block 3
- type: list
- source_text:
    - Valence (V): Pleasantness vs. Unpleasantness (e.g., 'joy' vs. 'terror').
    - Arousal (A): Level of excitement (e.g., 'elated' vs. 'serene').
    - Dominance (D): Feeling of control (e.g., 'powerful' vs. 'vulnerable').
- korean_translation:
    - 원자가 (Valence): 유쾌함 대 불쾌함 (예: 'joy' 대 'terror').
    - 각성 (Arousal): 흥분 정도 (예: 'elated' 대 'serene').
    - 지배력 (Dominance): 통제감 (예: 'powerful' 대 'vulnerable').
- study_note: VAD 모델의 세 가지 핵심 차원을 정의합니다.

## Block 4
- type: paragraph
- source_text: The NRC VAD Lexicon provides numeric scores for each of these dimensions for over 20,000 English words. These scores are typically scaled between 0 and 1.
- korean_translation: NRC VAD 어휘 사전은 20,000개 이상의 영어 단어들에 대해 이러한 각 차원에 대한 수치 점수를 제공합니다. 이 점수들은 일반적으로 0과 1 사이로 조정됩니다.
- study_note: 수만 개의 단어를 세밀한 실수 값으로 매핑한 NRC VAD 사전의 특징과 규모를 설명합니다.

## Block 3
- type: paragraph
- source_text: Manually building large lexicons is labor-intensive. Semi-supervised methods address this by starting with a small set of seeds and automatically expanding them using word relations (e.g., synonymy) or corpus statistics (e.g., co-occurrence).
- korean_translation: 대규모 어휘 사전을 수동으로 구축하는 것은 노동 집약적입니다. 반지도 학습 방법은 소규모 시드(seed) 단어들로 시작하여 단어 관계(예: 유의 관계)나 코퍼스 통계(예: 공기 빈도)를 활용해 이를 자동으로 확장함으로써 이 문제를 해결합니다.
- study_note: 수작업 구축의 한계를 극복하기 위해 제안된 자동 확장 기술들을 소개합니다.

## Block 4
- type: margin_note
- source_text: seed words
- korean_translation: 시드 단어(seed words)
- study_note: 'good'이나 'bad'와 같이 극성이 명확하여 확장의 기준이 되는 초기 단어 집합입니다.

## Block 5
- type: paragraph
- source_text: A common approach is using WordNet to propagate sentiment scores from seed words to their synonyms and antonyms.
- korean_translation: 일반적인 접근법은 워드넷을 사용하여 시드 단어들의 감성 점수를 그들의 유의어나 반의어로 전파(propagate)시키는 것입니다.
- study_note: 워드넷의 어휘망 구조를 감성 수치 전파의 경로로 활용하는 방식을 설명합니다.
