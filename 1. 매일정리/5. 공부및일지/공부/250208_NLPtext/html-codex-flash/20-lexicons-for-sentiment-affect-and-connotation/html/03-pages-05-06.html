Study Note
원자가, 각성, 지배력이라는 세 가지 필수 차원을 통해 감정을 정밀하게 수치화합니다.
Block 3
list
Source Text
- Valence (V): Pleasantness vs. Unpleasantness (e.g., 'joy' vs. 'terror').
- Arousal (A): Level of excitement (e.g., 'elated' vs. 'serene').
- Dominance (D): Feeling of control (e.g., 'powerful' vs. 'vulnerable').
Korean Translation
- 원자가 (Valence): 유쾌함 대 불쾌함 (예: 'joy' 대 'terror').
- 각성 (Arousal): 흥분 정도 (예: 'elated' 대 'serene').
- 지배력 (Dominance): 통제감 (예: 'powerful' 대 'vulnerable').
Study Note
VAD 모델의 세 가지 핵심 차원을 정의합니다.

The NRC VAD Lexicon provides numeric scores for each of these dimensions for over 20,000 English words. These scores are typically scaled between 0 and 1.

NRC VAD 어휘 사전은 20,000개 이상의 영어 단어들에 대해 이러한 각 차원에 대한 수치 점수를 제공합니다. 이 점수들은 일반적으로 0과 1 사이로 조정됩니다.

Study Note
소량의 시드(seed) 단어를 바탕으로 전체 코퍼스 혹은 지식망에서 대규모 감성 사전을 자동으로 구축하는 방식을 다릅니다.
Block 3
paragraph
Source Text
Manually building large lexicons is labor-intensive. Semi-supervised methods address this by starting with a small set of seeds and automatically expanding them using word relations (e.g., synonymy) or corpus statistics (e.g., co-occurrence).
Korean Translation
대규모 어휘 사전을 수동으로 구축하는 것은 노동 집약적입니다. 반지도 학습 방법은 소규모 시드(seed) 단어들로 시작하여 단어 관계(예: 유의 관계)나 코퍼스 통계(예: 공기 빈도)를 활용해 이를 자동으로 확장함으로써 이 문제를 해결합니다.
Study Note
수작업 구축의 한계를 극복하기 위해 제안된 자동 확장 기술들을 소개합니다.
