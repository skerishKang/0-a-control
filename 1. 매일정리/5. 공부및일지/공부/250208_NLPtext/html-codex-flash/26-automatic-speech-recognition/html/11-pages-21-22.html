Block 5
paragraph
Source Text
A dilated convolution is a subtype of causal convolutional layer. Causal or masked convolutions look only at the past input, rather than the future; the prediction of $y_{t+1}$ can only depend on $y_1, ..., y_t$, useful for autoregressive left-to-right processing. In dilated convolutions, at each successive layer we apply the convolutional filter over a span longer than its length by skipping input values. Thus at time $t$ with a dilation value of 1, a convolutional filter of length 2 would see input values $x_t$ and $x_{t-1}$. But a filter with a dilation value of 2 would skip an input, so would see input values $x_t$ and $x_{t-2}$. Fig. 26.15 shows the computation of the output at time $t$ with 4 dilated convolution layers with dilation values, 1, 2, 4, and 8.
Korean Translation
확장된 컨볼루션은 인과적 컨볼루션(causal convolutional) 레이어의 하위 유형입니다. 인과적 또는 마스킹된 컨볼루션은 미래가 아닌 과거의 입력만 고려합니다. $y_{t+1}$의 예측은 $y_1, ..., y_t$에만 의존할 수 있으며, 이는 자동 회귀적 왼쪽에서 오른쪽으로의 처리에 유용합니다. 확장된 컨볼루션에서는 각 연속 레이어에서 입력 값을 건너뛰어 필터 길이보다 긴 범위에 걸쳐 컨볼루션 필터를 적용합니다. 따라서 확장 값이 1인 시점 $t$에서 길이가 2인 컨볼루션 필터는 입력 값 $x_t$와 $x_{t-1}$을 보게 됩니다. 그러나 확장 값이 2인 필터는 입력을 하나 건너뛰므로 입력 값 $x_t$와 $x_{t-2}$를 보게 됩니다. 그림 26.15는 확장 값이 1, 2, 4, 8인 4개의 확장된 컨볼루션 레이어를 사용하여 시점 $t$에서 출력을 계산하는 과정을 보여줍니다.

The spectrogram prediction encoder-decoder and the WaveNet vocoder are trained separately. After the spectrogram predictor is trained, the spectrogram prediction network is run in teacher-forcing mode, with each predicted spectral frame conditioned on the encoded text input and the previous frame from the ground truth spectrogram. This sequence of ground truth-aligned spectral features and gold audio output is then used to train the vocoder.
Korean Translation

Source Text
This has been only a high-level sketch of the TTS process. There are numerous important details that the reader interested in going further with TTS may want to look into. For example WaveNet uses a special kind of a gated activation function as its non-linearity, and contains residual and skip connections. In practice, predicting 8-bit audio values doesn't as work as well as 16-bit, for which a simple softmax is insufficient, so decoders use fancier ways as the last step of predicting audio sample values, like mixtures of distributions. Finally, the WaveNet vocoder as we have described it would be so slow as to be useless; many different kinds of efficiency improvements are necessary in practice, for example by finding ways to do non-autoregressive generation, avoiding the latency of having to wait to generate each frame until the prior frame has been generated, and instead making predictions in parallel. We encourage the interested reader to consult the original papers and various version of the code.
Korean Translation
이것은 TTS 과정의 개략적인 설명일 뿐입니다. TTS를 더 깊이 공부하고자 하는 독자가 살펴볼 만한 중요한 세부 사항이 많이 있습니다. 예를 들어, WaveNet은 비선형 함수로 특별한 종류의 게이트 활성화 함수를 사용하며, 잔차(residual) 및 스킵(skip) 연결을 포함합니다. 실제로 8비트 오디오 값을 예측하는 것은 16비트만큼 잘 작동하지 않으며, 16비트의 경우 단순한 소프트맥스로는 불충분합니다. 따라서 디코더는 분포 혼합(mixtures of distributions)과 같이 오디오 샘플 값을 예측하는 마지막 단계에서 더 정교한 방법을 사용합니다. 마지막으로, 앞서 설명한 WaveNet 보코더는 너무 느려서 실용성이 없을 것입니다. 실제로는 비자동회귀(non-autoregressive) 생성 방법을 찾는 등 다양한 효율성 개선이 필요합니다. 이전 프레임이 생성될 때까지 기다려야 하는 지연 시간을 피하고 대신 병렬로 예측을 수행하는 방식 등이 그 예입니다. 관심 있는 독자는 원본 논문과 다양한 버전의 코드를 참고하기 바랍니다.

Speaker diarization is the task of determining ‘who spoke when’ in a long multi-speaker audio recording, marking the start and end of each speaker’s turns in the interaction. This can be useful for transcribing meetings, classroom speech, or medical interactions. Often diarization systems use voice activity detection (VAD) to find segments of continuous speech, extract speaker embedding vectors, and cluster the vectors to group together segments likely from the same speaker. More recent work is investigating end-to-end algorithms to map directly from input speech to a sequence of speaker labels for each frame.
Korean Translation
화자 분할(speaker diarization)은 긴 다중 화자 오디오 녹음에서 '누가 언제 말했는지'를 결정하고 상호 작용에서 각 화자의 발언 시작과 끝을 표시하는 작업입니다. 이는 회의, 교실 수업 또는 의료 상담을 전사하는 데 유용할 수 있습니다. 종종 화자 분할 시스템은 음성 활동 감지(VAD)를 사용하여 연속된 음성 세그먼트를 찾고, 화자 임베딩 벡터를 추출한 다음, 벡터를 클러스터링하여 동일한 화자의 것으로 보이는 세그먼트를 그룹화합니다. 최근 연구에서는 입력 음성에서 각 프레임의 화자 레이블 시퀀스로 직접 매핑하는 엔드투엔드 알고리즘을 조사하고 있습니다.
