## Block 9
- type: paragraph
- source_text: Because of the strong independence assumption in CTC (assuming that the output at time t is independent of the output at time t-1), recognizers based on CTC don’t achieve as high an accuracy as the attention-based encoder-decoder recognizers. CTC recognizers have the advantage, however, that they can be used for streaming. Streaming means recognizing words on-line rather than waiting until the end of the sentence to recognize them. Streaming is crucial for many applications, from commands to dictation, where we want to start recognition while the user is still talking. Algorithms that use attention need to compute the hidden state sequence over the entire input first in order to provide the attention distribution context, before the decoder can start decoding. By contrast, a CTC algorithm can input letters from left to right immediately.
- korean_translation: CTC의 강력한 독립성 가정(시간 t의 출력이 t-1의 출력과 독립적이라는 가정) 때문에, CTC 기반의 인식기는 어텐션 기반 인코더-디코더 인식기만큼 높은 정확도를 달성하지 못합니다. 그러나 CTC 인식기는 스트리밍(streaming)에 사용될 수 있다는 장점이 있습니다. 스트리밍이란 문장이 끝날 때까지 기다리지 않고 온라인으로 즉시 단어를 인식하는 것을 의미합니다. 스트리밍은 사용자가 아직 말하고 있는 동안 인식을 시작해야 하는 명령어나 받아쓰기와 같은 많은 응용 프로그램에 매우 중요합니다. 어텐션을 사용하는 알고리즘은 디코더가 디코딩을 시작하기 전, 어텐션 분포 컨텍스트를 제공하기 위해 먼저 전체 입력에 대한 은닉 상태 시퀀스를 계산해야 합니다. 반면, CTC 알고리즘은 즉시 왼쪽에서 오른쪽으로 문자를 입력받을 수 있습니다.
- study_note: CTC가 정확도는 낮을 수 있으나 실시간성(스트리밍) 측면에서 가지는 결정적인 장점과 어텐션 모델과의 차이점을 설명합니다.

## Block 2
- type: paragraph
- source_text: If we want to do streaming, we need a way to improve CTC recognition to remove the conditional independent assumption, enabling it to know about output history. The RNN-Transducer (RNN-T), shown in Fig. 26.12, is just such a model (Graves 2012, Graves et al. 2013). The RNN-T has two main components: a CTC acoustic model, and a separate language model component called the predictor that conditions on the output token history. At each time step t, the CTC encoder outputs a hidden state ht given the input x1, ..., xt. The language model predictor takes as input the previous output token (not counting blanks), outputting a hidden state hu. The two are passed through another network whose output is then passed through a softmax to predict the next character.
- korean_translation: 스트리밍을 원한다면, 조건부 독립 가정을 제거하여 출력 이력을 알 수 있도록 CTC 인식을 개선할 방법이 필요합니다. 그림 26.12에 표시된 RNN-트랜스듀서(RNN-Transducer, RNN-T)가 바로 그러한 모델입니다(Graves 2012, Graves et al. 2013). RNN-T는 두 가지 주요 구성 요소로 이루어져 있습니다. 바로 CTC 음향 모델과 출력 토큰 이력에 조건을 두는 '예측기(predictor)'라고 불리는 별도의 언어 모델 구성 요소입니다. 각 타임스텝 t에서 CTC 인코더는 입력 x1, ..., xt가 주어졌을 때 은닉 상태 ht를 출력합니다. 언어 모델 예측기는 (빈 토큰을 제외한) 이전 출력 토큰을 입력으로 받아 은닉 상태 hu를 출력합니다. 이 둘은 또 다른 네트워크를 통과하고, 그 출력이 소프트맥스를 거쳐 다음 문자를 예측하게 됩니다.
- study_note: CTC의 독립성 가정을 극복하기 위해 별도의 예측기(언어 모델)를 결합한 RNN-T의 구조와 작동 원리를 설명합니다.
