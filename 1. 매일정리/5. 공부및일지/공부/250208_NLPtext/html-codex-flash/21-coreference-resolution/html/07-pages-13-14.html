21.6.2 B-cubed
Study Note
MUC의 한계를 보완하기 위해 각 언급(mention) 수준에서 정밀도와 재현율을 계산하는 B-cubed 지표를 다릅니다.
Block 2
paragraph
Source Text
The B-cubed metric calculates precision and recall for each mention and then averages across all mentions in the text.
Korean Translation
B-cubed 지표는 각 언급에 대해 정밀도와 재현율을 계산한 다음, 텍스트 내의 모든 언급들에 대해 평균을 냅니다.
Study Note
언급별 정밀도(Precision)와 재현율(Recall)을 산출하여 평균을 내는 B-cubed의 기초 계산 원리입니다.

Source Text
mention-based evaluation
Korean Translation
언급 기반 평가(mention-based evaluation)
Study Note
클러스터 내 각 언급의 정답 포함 여부를 판단하여 평가의 정밀도를 높이는 방식입니다.
Block 4
paragraph
Source Text
B-cubed addresses some of the limitations of MUC by penalizing over-clustering and under-clustering, making it a more robust evaluation metric.
Korean Translation
B-cubed는 과도한 클러스터링(over-clustering)과 미흡한 클러스터링(under-clustering)을 모두 처벌함으로써 MUC의 일부 한계점을 해결하며, 더 견고한 평가 지표가 됩니다.
Study Note
B-cubed가 MUC보다 성능 평가에 있어 더 공정하게 작동하는 이유를 설명합니다.

21.6.3 CEAF (Constrained Entity Aligned F-measure)
Korean Translation
21.6.3 CEAF (제약된 개체 정렬 F-measure)
Study Note
예측된 클러스터와 골드 클러스터 사이의 1대1 매핑을 통해 유사도를 측정하는 CEAF 지표를 다릅니다.
Block 3
paragraph
Source Text
The CEAF metric uses an alignment approach between the gold clusters and the predicted clusters. It finds the best one-to-one mapping between them based on their similarity.
Korean Translation
CEAF 지표는 골드 클러스터와 예측된 클러스터 사이의 정렬(alignment) 방식을 사용합니다. 유사도에 기반하여 두 분류 사이의 최적의 일대일 매핑을 찾습니다.
Study Note
클러스터 간의 일대일 대응(Matching)을 통해 평가의 엄밀성을 확보하는 방식을 설명합니다.

paragraph
Source Text
CEAF provides a more global view of the coreference clusters but can be more computationally expensive to evaluate than MUC or B-cubed.
Korean Translation
CEAF는 상호 참조 클러스터에 대해 더 전역적인 관점을 제공하지만, MUC나 B-cubed보다 평가를 위한 계산 비용이 더 많이 들 수 있습니다.
Study Note
CEAF 지표의 전역적 정확성과 계산 복잡도라는 트레이드오프를 설명합니다.
