The mention-pair model treats coreference as a set of binary classification decisions. For each pair of mentions (m_i, m_j), the model predicts whether they are coreferent (1) or not (0).
Korean Translation
언급-쌍 모델은 상호 참조를 일련의 이진 분류 결정으로 취급합니다. 각 언급 쌍 (m_i, m_j)에 대해, 모델은 그것들이 상호 참조 관계인지(1) 아닌지(0)를 예측합니다.

Traditional mention-pair models rely on hand-crafted features extracted from the text and syntactic analysis. Key features include:
Korean Translation
전통적인 언급-쌍 모델은 텍스트와 통사 분석에서 추출한 수작업 자질들에 의존합니다. 주요 자질은 다음과 같습니다.

list
Source Text
    - Distance: Sentence or word distance between mentions.
    - String Match: Do the strings (e.g., "Victoria Chen" and "Victoria") overlap?
    - Gender/Number Agreement: Do they agree in gender and number?
    - Syntactic Relationship: Are they in a specific syntactic configuration?
Korean Translation
    - 거리(Distance): 언급들 사이의 문장 혹은 단어 거리.
    - 문자열 일치(String Match): 문자열(예: "Victoria Chen"과 "Victoria")이 겹치는가?
    - 성/수 일치(Gender/Number Agreement): 성별과 수가 일치하는가?
    - 통사적 관계(Syntactic Relationship): 특정한 통사적 구조 안에 있는가?
Study Note
