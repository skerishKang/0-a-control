Korean Translation
21.4 개체-언급 모델(Entity-Mention Models)
Study Note
단순한 언급 쌍이 아닌, 이미 형성된 개체 클러스터와 새로운 언급 사이의 관계를 판단하는 모델을 다릅니다.
Block 3
margin_note
Source Text
entity-mention model
Korean Translation
개체-언급 모델(entity-mention model)
Study Note
到目前为止形成的개체(entity)의 정보와 새로운 언급(mention) 간의 유사성을 판단하는 방식입니다.
Block 4
paragraph
Source Text
Entity-mention models represent coreference by predicting whether a new mention should be added to an existing entity cluster. This allows for features that capture entity-level information (e.g., gender, semantic category).
Korean Translation
개체-언급 모델은 새로운 언급이 기존의 개체 클러스터에 추가되어야 하는지를 예측함으로써 상호 참조를 표현합니다. 이는 개체 수준의 정보(예: 성별, 의미 범주)를 포착하는 자질들을 사용할 수 있게 해줍니다.
Study Note
언급-쌍 모델의 한계를 극복하기 위해 '개체'라는 묶음 단위를 판단의 주체로 삼는 모델입니다.
Block 5
paragraph
Source Text
By considering the entire cluster, these models can enforce consistency across all mentions of an entity. For example, all mentions in a cluster must be of the same gender.
Korean Translation
전체 클러스터를 고려함으로써, 이러한 모델은 개체의 모든 언급들 사이의 일관성을 강제할 수 있습니다. 예를 들어, 한 클러스터 내의 모든 언급은 같은 성별이어야 합니다.
Study Note
클러스터 기반의 장점인 '속성 일관성 유지'에 대해 설명합니다.

21.5 Neural Coreference Resolution
Korean Translation
21.5 신경망 기반 상호 참조 해소
Study Note
수작업 자질 공학을 탈피하고, 딥러닝 인코더(BiLSTM, Transformer)를 통해 언급의 표현을 배우는 현대적 상호 참조 해소 방식을 다릅니다.
Block 3
paragraph
Source Text
Modern coreference systems rely on neural architectures to learn rich representations of mentions and their potential coreference relationships. These systems typically use an end-to-end approach.
Korean Translation
현대의 상호 참조 시스템은 언급과 그들 사이의 잠재적인 상호 참조 관계에 대한 풍부한 표현을 학습하기 위해 신경망 아키텍처에 의존합니다. 이러한 시스템들은 대개 엔드투엔드(end-to-end) 방식을 사용합니다.
Study Note
언급 탐지와 클러스터링을 한꺼번에 수행하는 최근의 경향성을 설명합니다.
Block 4
margin_note
Source Text
end-to-end coreference
Korean Translation
엔드투엔드 상호 참조(end-to-end coreference)
Study Note
별도의 파이프라인 없이 하나의 신경망 모델이 모든 상호 참조 과정을 처리하는 구조입니다.
Block 5
paragraph
Source Text
Lee et al. (2017) introduced a popular end-to-end neural coreference model that uses BiLSTMs to extract mention features and an attention mechanism to assign scores to potential antecedents.
Korean Translation
Lee 등(2017)은 BiLSTM을 사용하여 언급 자질을 추출하고, 어텐션 메커니즘을 사용하여 잠재적인 선행사들에게 점수를 할당하는 인기 있는 엔드투엔드 신경망 상호 참조 모델을 제안했습니다.
Study Note
Lee(2017)의 랜드마크 모델 구조(BiLSTM + Attention)를 요약합니다.
