Block 2
paragraph
Source Text
To improve parsing accuracy, we can use beam search instead of a greedy best-first search. Beam search maintains a small set of the k most likely sequences of transitions at each step.
Korean Translation
파싱 정확도를 높이기 위해, 탐욕적 최우선 탐색(greedy best-first search) 대신 빔 서치를 사용할 수 있습니다. 빔 서치는 각 단계에서 가장 가능성이 높은 k개의 전이 시퀀스 집합을 유지합니다.
Study Note
빔 서치가 탐욕적 방식의 한계를 어떻게 보완하는지 그 핵심 아이디어를 설명합니다.

Block 4
paragraph
Source Text
At each step, we expand each of the k transitions with all possible next transitions and keep the k most likely sequences. This process is repeated until all sequences reach a final configuration.
Korean Translation
각 단계에서, 우리는 k개의 전이 각각을 모든 가능한 다음 전이들로 확장하고, 그중 가장 가능성이 높은 k개의 시퀀스만을 남깁니다. 이 과정은 모든 시퀀스가 최종 구성에 도달할 때까지 반복됩니다.
Study Note
빔 서치의 단계별 확장 및 필터링 절차를 설명합니다.

Graph-based dependency parsing treats parsing as finding the highest-scoring dependency tree in a graph where nodes are words and edges represent potential dependency relations.

Block 5
paragraph
Source Text
The goal of graph-based parsing is to find the maximum spanning tree (MST) in the full graph of words. This can be solved efficiently using algorithms like the Chu-Liu/Edmonds algorithm.
Korean Translation
그래프 기반 파싱의 목표는 단어들로 구성된 완전 그래프에서 최대 신장 트리(maximum spanning tree, MST)를 찾아내는 것입니다. 이는 Chu-Liu/Edmonds 알고리즘과 같은 알고리즘을 사용하여 효율적으로 해결될 수 있습니다.
Study Note
MST 알고리즘을 파싱에 적용하는 방식과 대표적인 알고리즘 명칭을 언급합니다.
