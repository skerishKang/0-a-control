Block 3
margin_note
Source Text
attachment score
Korean Translation
부착 점수(attachment score)
Study Note
의존 관계의 연결(attachment)이 얼마나 정확한지 나타내는 지표입니다.
Block 4
list
Source Text
    - UAS (Unlabeled Attachment Score): The percentage of words that are correctly attached to their head word.
    - LAS (Labeled Attachment Score): The percentage of words that are correctly attached to their head word with the correct dependency label.
Korean Translation
    - UAS(Unlabeled Attachment Score): 지배어(head word)에 올바르게 부착된 단어들의 백분율입니다. (관계의 라벨은 고려하지 않음)
    - LAS(Labeled Attachment Score): 지배어와 의존 관계 라벨이 모두 올바르게 부착된 단어들의 백분율입니다.
Study Note
의존 구문 분석 평가의 양대 핵심 척도인 UAS와 LAS의 정의와 차이점을 설명합니다.
Block 5
paragraph
Source Text
To compute these scores, we compare the parser's output for each word to the gold standard labels in a dependency treebank.
Korean Translation
이 점수들을 계산하기 위해, 파서가 출력한 각 단어의 지배어와 라벨을 의존 트리뱅크의 정답(gold standard)과 비교합니다.
Study Note
실제 평가 과정에서 트리뱅크 정답 데이터를 활용함을 설명합니다.
