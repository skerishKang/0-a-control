Source Text
In this chapter, we introduced dependency parsing—the task of identifying the syntactic structure of a sentence by identifying the binary relations between words. We explored transition-based and graph-based approaches to dependency parsing, and touched on neural dependency parsing.
Korean Translation
이 장에서 우리는 의존 구문 분석(dependency parsing), 즉 단어들 사이의 이항 관계를 식별하여 문장의 통사적 구조를 찾아내는 작업을 소개했습니다. 전이 기반 및 그래프 기반 접근법을 살펴보았으며, 신경망 의존 구문 분석에 대해서도 다루었습니다.
Study Note
14장의 핵심 학습 내용인 세 가지 파싱 방식과 의존 구문 분석의 정의를 재확인합니다.
Block 4
list
Source Text
    - Dependency Parsing: identifying binary relations between words.
    - Transition-Based Parsing: inspired by shift-reduce algorithms, uses stack and buffer configurations.
    - Arc-Standard: a common transition-based algorithm with LEFT-ARC, RIGHT-ARC, and SHIFT transitions.
    - Oracle: provides correct transitions during training.
    - Graph-Based Parsing: finding the highest-scoring tree in a fully connected graph of words.
    - Maximum Spanning Tree (MST): the globally optimal tree found by graph-based parsers.
    - Neural Dependency Parsing: using neural networks and word embeddings to improve accuracy.
    - Evaluation: using Unlabeled and Labeled Attachment Scores (UAS, LAS).
Korean Translation
    - 의존 구문 분석(Dependency Parsing): 단어들 사이의 이항 관계를 식별함.
    - 전이 기반 파싱(Transition-Based Parsing): 시프트-리듀스 알고리즘에서 영감을 받음, 스택과 버퍼 구성을 사용함.
    - 아크-스탠다드(Arc-Standard): 왼쪽-아크, 오른쪽-아크, 시프트 전이를 사용하는 일반적인 전이 기반 알고리즘.
    - 오라클(Oracle): 학습 과정에서 정답 전이를 제공함.
    - 그래프 기반 파싱(Graph-Based Parsing): 단어들의 완전 연결 그래프에서 가장 점수가 높은 트리를 찾음.
    - 최대 신장 트리(Maximum Spanning Tree, MST): 그래프 기반 파서가 찾아내는 전역적으로 최적인 트리.
    - 신경망 의존 구문 분석: 신경망과 단어 임베딩을 사용하여 정확도를 높임.
    - 평가(Evaluation): UAS(레이블되지 않은 부착 점수)와 LAS(레이블된 부착 점수)를 사용함.
Study Note
14장에서 학습한 8가지 핵심 용어와 개념들을 정의와 함께 정리합니다.

Transition-based parsing was introduced by Nivre (2003) and Yamada and Matsumoto (2003). Neural dependency parsing was popularized by Chen and Manning (2014).
Korean Translation
전이 기반 파싱은 니브르(Joakim Nivre, 2003)와 야마다 및 마츠모토(Yamada and Matsumoto, 2003)에 의해 소개되었습니다. 신경망 의존 구문 분석은 첸과 매닝(Chen and Manning, 2014)의 연구를 통해 널리 알려졌습니다.
