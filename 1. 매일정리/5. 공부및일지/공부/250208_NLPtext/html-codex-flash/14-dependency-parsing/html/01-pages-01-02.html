While constituency grammars focus on how words group into larger units, dependency grammars focus on the direct relations between words. In this chapter, we explore how these relations can be identified automatically."

Block 2

Source Text
Transition-based parsing is inspired by shift-reduce parsing algorithms used in compilers. It works by processing a sentence from left to right, maintaining a stack of partially processed words and a buffer of remaining words.
Korean Translation
전이 기반 파싱은 컴파일러에서 사용되는 시프트-리듀스(shift-reduce) 파싱 알고리즘에서 영감을 얻었습니다. 이는 문장을 왼쪽에서 오른쪽으로 처리하며, 부분적으로 처리된 단어들을 담은 스택(stack)과 남은 단어들을 담은 버퍼(buffer)를 유지하는 방식으로 작동합니다.
Study Note
전이 기반 파싱의 기원과 기본 작동 메커니즘을 설명합니다.

Source Text
A configuration in a transition-based parser consists of a stack, a buffer, and a set of dependency relations. The parser starts with an empty stack (or a stack containing a special ROOT symbol), the entire sentence in the buffer, and an empty set of relations.
Korean Translation
전이 기반 파서의 구성(configuration)은 스택, 버퍼, 그리고 의존 관계들의 집합으로 이루어집니다. 파서는 비어 있는 스택(또는 특수한 ROOT 기호가 포함된 스택), 버퍼에 담긴 전체 문장, 그리고 비어 있는 관계 집합에서 시작합니다.
Study Note
파서의 초기 상태(initial configuration)를 정의합니다.
