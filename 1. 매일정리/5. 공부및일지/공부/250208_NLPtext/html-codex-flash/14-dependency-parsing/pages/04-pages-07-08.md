## Block 2
- type: paragraph
- source_text: To improve parsing accuracy, we can use beam search instead of a greedy best-first search. Beam search maintains a small set of the k most likely sequences of transitions at each step.
- korean_translation: 파싱 정확도를 높이기 위해, 탐욕적 최우선 탐색(greedy best-first search) 대신 빔 서치를 사용할 수 있습니다. 빔 서치는 각 단계에서 가장 가능성이 높은 k개의 전이 시퀀스 집합을 유지합니다.
- study_note: 빔 서치가 탐욕적 방식의 한계를 어떻게 보완하는지 그 핵심 아이디어를 설명합니다.

## Block 3
- type: margin_note
- source_text: beam search
- korean_translation: 빔 서치(beam search)
- study_note: 탐색 공간에서 상위 유망한 후보군(빔 크기 k)을 유지하며 최적해를 찾는 알고리즘입니다.

## Block 4
- type: paragraph
- source_text: At each step, we expand each of the k transitions with all possible next transitions and keep the k most likely sequences. This process is repeated until all sequences reach a final configuration.
- korean_translation: 각 단계에서, 우리는 k개의 전이 각각을 모든 가능한 다음 전이들로 확장하고, 그중 가장 가능성이 높은 k개의 시퀀스만을 남깁니다. 이 과정은 모든 시퀀스가 최종 구성에 도달할 때까지 반복됩니다.
- study_note: 빔 서치의 단계별 확장 및 필터링 절차를 설명합니다.

## Block 2
- type: paragraph
- source_text: Graph-based dependency parsing treats parsing as finding the highest-scoring dependency tree in a graph where nodes are words and edges represent potential dependency relations.
- korean_translation: 그래프 기반 의존 구문 분석은 문장의 단어들을 노드로, 단어들 사이의 잠재적인 의존 관계를 에지로 하는 그래프에서 점수가 가장 높은 의존 트리를 찾는 문제로 파싱을 다룹니다.
- study_note: 그래프 기반 파싱의 기본 프레임워크를 설명합니다.

## Block 3
- type: margin_note

- korean_translation: 에지 점수 산정(edge scoring)
- study_note: 두 단어 사이의 의존 관계가 얼마나 타당한지 수치화하는 과정입니다.

## Block 5
- type: paragraph
- source_text: The goal of graph-based parsing is to find the maximum spanning tree (MST) in the full graph of words. This can be solved efficiently using algorithms like the Chu-Liu/Edmonds algorithm.
- korean_translation: 그래프 기반 파싱의 목표는 단어들로 구성된 완전 그래프에서 최대 신장 트리(maximum spanning tree, MST)를 찾아내는 것입니다. 이는 Chu-Liu/Edmonds 알고리즘과 같은 알고리즘을 사용하여 효율적으로 해결될 수 있습니다.
- study_note: MST 알고리즘을 파싱에 적용하는 방식과 대표적인 알고리즘 명칭을 언급합니다.
