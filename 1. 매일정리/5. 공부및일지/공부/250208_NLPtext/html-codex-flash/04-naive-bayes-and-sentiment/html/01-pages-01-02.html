Source Text
Formally, the task of supervised classification is to take an input x and a fixed set of output classes Y = {y1, y2, . . . , yM} and return a predicted class y ∈ Y. For text classification, we'll sometimes talk about c (for "class") instead of y as our output variable, and d (for "document") instead of x as our input variable. In the supervised situation we have a training set of N documents that have each been hand-labeled with a class: (d1, c1), . . . , (dN , cN ). Our goal is to learn a classifier that is capable of mapping from a new document d to its correct class c ∈ C. A probabilistic classifier additionally will tell us the probability of the observation being in the class. This full distribution over the classes can be useful information for downstream decisions; avoiding making discrete decisions early on can be useful when combining systems.
Korean Translation
형식적으로 지도 분류 작업은 입력 $x$와 고정된 출력 클래스 집합 $Y = \{y_1, y_2, . . . , y_M\}$을 받아 예측된 클래스 $y \in Y$를 반환하는 것입니다. 텍스트 분류에서는 때때로 출력 변수로 $y$ 대신 $c$ ("class"의 약자)를, 입력 변수로 $x$ 대신 $d$ ("document"의 약자)를 사용하기도 합니다. 지도 학습 환경에서는 각각 수동으로 라벨링된 $N$개의 문서로 이루어진 학습 세트 $(d_1, c_1), . . . , (d_N , c_N )$을 가집니다. 우리의 목표는 새로운 문서 $d$를 올바른 클래스 $c \in C$로 매핑할 수 있는 분류기를 학습하는 것입니다. 확률적 분류기(probabilistic classifier)는 추가적으로 관측치가 해당 클래스에 속할 확률을 알려줍니다. 클래스들에 대한 이러한 전체 분포 정보는 후속 의사결정에 유용할 수 있습니다. 시스템을 결합할 때 불확실한 초기 단계에서 성급하게 이산적인 결정을 내리는 것을 피하는 데 도움이 되기 때문입니다.
Study Note

Many kinds of machine learning algorithms are used to build classifiers. This chapter introduces naive Bayes; the following one introduces logistic regression. These exemplify two ways of doing classification. Generative classifiers like naive Bayes build a model of how a class could generate some input data. Given an observation, they return the class most likely to have generated the observation. Discriminative classifiers like logistic regression instead learn what features from the input are most useful to discriminate between the different possible classes. While discriminative systems are often more accurate and hence more commonly used, generative classifiers still have a role.
