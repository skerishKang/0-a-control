Block 9
paragraph
Source Text
Using these indices, each constituent in the CKY parse tree is encoded in a two-dimensional matrix. Specifically, for a sentence of length n and a grammar that contains V non-terminals, we use the upper-triangular portion of an (n+ 1) * (n+ 1) matrix. For CKY, each cell table[i, j] contained a list of constituents that could span the sequence of words from i to j. For probabilistic CKY, it's slightly simpler to think of the constituents in each cell as constituting a third dimension of maximum length V. This third dimension corresponds to each non-terminal that can be placed in this cell, and the value of the cell is then a probability for that non-terminal/constituent rather than a list of constituents. In summary, each cell [i, j, A] in this (n+ 1) * (n+ 1) * V matrix is the probability of a constituent of type A that spans positions i through j of the input.
Korean Translation
이 인덱스들을 사용하여 CKY 파스 트리의 각 구성 성분(constituent)은 2차원 행렬에 인코딩됩니다. 구체적으로, 길이가 n인 문장과 V개의 비단말 기호를 포함하는 문법에 대해, 우리는 (n+ 1) * (n+ 1) 행렬의 상삼각 부분을 사용합니다. CKY의 경우, 각 셀 table[i, j]에는 i에서 j까지의 단어열을 포괄할 수 있는 구성 성분들의 리스트가 들어 있었습니다. 확률적 CKY의 경우, 각 셀의 구성 성분들을 최대 길이 V인 세 번째 차원을 구성하는 것으로 생각하는 것이 조금 더 간단합니다. 이 세 번째 차원은 이 셀에 배치될 수 있는 각 비단말 기호에 해당하며, 셀의 값은 구성 성분 리스트가 아니라 해당 비단말 기호/구성 성분에 대한 확률입니다. 요약하자면, 이 (n+ 1) * (n+ 1) * V 행렬의 각 셀 [i, j, A]는 입력의 i 위치부터 j 위치까지를 포괄하는 A 타입 구성 성분의 확률입니다.

    function PROBABILISTIC-CKY(words, grammar) returns most probable parse and its probability
      for j from 1 to LENGTH(words) do
        for all { A | A → words[j] ∈ grammar }
          table[j-1, j, A] ← P(A → words[j])
        for i from j-2 downto 0 do
          for k from i + 1 to j-1 do
            for all { A | A → BC ∈ grammar and table[i,k,B] > 0 and table[k,j,C] > 0 }
              if (table[i,j,A] < P(A → BC) * table[i,k,B] * table[k,j,C]) then
                table[i,j,A] ← P(A → BC) * table[i,k,B] * table[k,j,C]
                back[i,j,A] ← {k,B,C}
      return BUILD_TREE(back[1, LENGTH(words), S]), table[1, LENGTH(words), S]
