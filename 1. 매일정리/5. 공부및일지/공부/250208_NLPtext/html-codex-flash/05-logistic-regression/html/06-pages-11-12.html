Stochastic gradient descent is an online algorithm that minimizes the loss function by computing its gradient after each training example, and nudging θ in the right direction (the opposite direction of the gradient). (an "online algorithm" is one that processes its input example by example, rather than waiting until it sees the entire input). Fig. 5.5 shows the algorithm.
Korean Translation
확률적 경사 하강법(Stochastic gradient descent)은 각 학습 예제마다 경사를 계산하고 $\theta$를 올바른 방향(경사의 반대 방향)으로 조금씩 이동시켜 손실 함수를 최소화하는 온라인 알고리즘(online algorithm)입니다. ('온라인 알고리즘'이란 전체 입력을 다 볼 때까지 기다리지 않고 입력을 예제 단위로 처리하는 방식을 말합니다.) 그림 5.5는 이 알고리즘을 보여줍니다.
Study Note

Block 10
code
Source Text
function STOCHASTIC GRADIENT DESCENT(L, f, x, y) returns θ
# where: L is the loss function
# f is a function parameterized by θ
# x is the set of training inputs x(1), x(2), ..., x(m)
# y is the set of training outputs (labels) y(1), y(2), ..., y(m)
θ ← 0
repeat til done # see caption
For each training tuple (x(i), y(i)) (in random order)
1. Optional (for reporting): # How are we doing on this tuple?
Compute y_hat(i) = f(x(i); θ) # What is our estimated output y_hat?
Compute the loss L(y_hat(i), y(i)) # How far off is y_hat(i) from the true output y(i)?
2. g ← ∇θL(f(x(i); θ), y(i)) # How should we move θ to maximize loss?
3. θ ← θ − η g # Go the other way instead
return θ
Korean Translation
함수 STOCHASTIC GRADIENT DESCENT(L, f, x, y) 반환값 θ
# L: 손실 함수
# f: θ로 파라미터화된 함수
# x: 학습 입력 세트 x(1), x(2), ..., x(m)
# y: 학습 출력(라벨) 세트 y(1), y(2), ..., y(m)
θ ← 0
종료될 때까지 반복 # 캡션 참조
각 학습 튜플 (x(i), y(i))에 대해 (무작위 순서로)
1. 선택 사항 (보고용): # 이 튜플에 대해 얼마나 잘하고 있는가?
추정치 계산 y_hat(i) = f(x(i); θ) # 우리의 추정 출력 y_hat은 무엇인가?
손실 계산 L(y_hat(i), y(i)) # y_hat(i)이 실제 출력 y(i)에서 얼마나 떨어져 있는가?
2. g ← ∇θL(f(x(i); θ), y(i)) # 손실을 최대화하려면 θ를 어떻게 움직여야 하는가?
3. θ ← θ − η g # 대신 그 반대 방향으로 이동
반환 θ

Block 13
paragraph
Source Text
The learning rate η is a hyperparameter that must be adjusted. If it's too high, the learner will take steps that are too large, overshooting the minimum of the loss function. If it's too low, the learner will take steps that are too small, and take too long to get to the minimum. It is common to start with a higher learning rate and then slowly decrease it, so that it is a function of the iteration k of training; the notation η_k can be used to mean the value of the learning rate at iteration k.
Korean Translation
학습률 $\eta$는 반드시 조정되어야 하는 하이퍼파라미터입니다. 학습률이 너무 높으면 학습기는 너무 큰 폭으로 이동하여 손실 함수의 최솟값을 지나쳐버리는 오버슈팅(overshooting)을 범하게 됩니다. 너무 낮으면 보폭이 너무 작아 최솟값에 도달하는 데 너무 오랜 시간이 걸립니다. 학습 초기에는 높은 학습률로 시작하여 서서히 줄여나가는 방식이 일반적이며, 이를 학습 반복 횟수 $k$에 대한 함수로 보고 $\eta_k$로 표기하기도 합니다. [다음 페이지에서 계속]
Study Note
학습률의 크기에 따른 문제점들과 동적 학습률(Learning Rate Decay) 기법의 필요성을 설명합니다.

Block 16
paragraph
Source Text
Stochastic gradient descent is called stochastic because it chooses a single random example at a time, moving the weights so as to improve performance on that single example. That can result in very choppy movements, so it's common to compute the gradient over batches of training instances rather than a single instance.
Korean Translation
확률적 경사 하강법이 '확률적(stochastic)'이라고 불리는 이유는 한 번에 하나의 무작위 예제를 선택하여 그 단일 예제에 대한 성능을 향상시키도록 가중치를 이동시키기 때문입니다. 이는 매우 불안정하고 요동치는(choppy) 이동을 초래할 수 있으므로, 단일 인스턴스 대신 학습 인스턴스들의 묶음(batch)에 대해 경사를 계산하는 것이 일반적입니다.
Study Note
SGD의 무작위성으로 인한 불안정성과 이를 해결하기 위한 배치/미니배치 학습의 동기를 설명합니다.
