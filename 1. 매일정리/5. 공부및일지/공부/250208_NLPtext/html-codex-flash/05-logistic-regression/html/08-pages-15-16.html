The multinomial logistic classifier uses a generalization of the sigmoid, called the softmax function, to compute the probability p(y = c|x). The softmax function takes a vector z = [z1, z2, ..., zk] of k arbitrary values and maps them to a probability distribution, with each value in the range (0,1), and all the values summing to 1. Like the sigmoid, it is an exponential function.
Korean Translation
다항 로지스틱 분류기는 확률 $p(y=c|x)$를 계산하기 위해 시그모이드의 일반화된 형태인 소프트맥스(softmax) 함수를 사용합니다. 소프트웨어 함수는 $k$개의 임의의 값으로 구성된 벡터 $z = [z_1, z_2, \dots, z_k]$를 입력받아 이를 확률 분포로 대응시킵니다. 이때 각 값은 (0, 1) 범위에 있고, 모든 값의 합은 1이 됩니다. 시그모이드와 마찬가지로 이는 지수 함수입니다.

이러한 특징 가중치들은 입력 텍스트와 출력 클래스 모두에 의존하기 때문에, 우리는 때때로 이 의존성을 명시적으로 드러내어 특징 자체를 입력과 클래스 모두의 함수인 $f(x, y)$로 표현하기도 합니다. 이러한 표기법을 사용하면 위의 $f_5(x)$는 $f_5(x, +), f_5(x, -), f_5(x, 0)$의 세 가지 특징으로 표현될 수 있으며, 각각은 단일 가중치를 갖게 됩니다.
Study Note
특징을 (입력, 클래스) 쌍의 함수로 표현하는 추상화된 표기법을 소개합니다.
