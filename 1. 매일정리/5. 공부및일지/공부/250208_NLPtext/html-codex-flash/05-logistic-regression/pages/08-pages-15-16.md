## Block 15
- type: paragraph
- source_text: The multinomial logistic classifier uses a generalization of the sigmoid, called the softmax function, to compute the probability p(y = c|x). The softmax function takes a vector z = [z1, z2, ..., zk] of k arbitrary values and maps them to a probability distribution, with each value in the range (0,1), and all the values summing to 1. Like the sigmoid, it is an exponential function.
- korean_translation: 다항 로지스틱 분류기는 확률 $p(y=c|x)$를 계산하기 위해 시그모이드의 일반화된 형태인 소프트맥스(softmax) 함수를 사용합니다. 소프트맥스 함수는 $k$개의 임의의 값으로 구성된 벡터 $z = [z_1, z_2, \dots, z_k]$를 입력받아 이를 확률 분포로 대응시킵니다. 이때 각 값은 (0, 1) 범위에 있고, 모든 값의 합은 1이 됩니다. 시그모이드와 마찬가지로 이는 지수 함수입니다.
