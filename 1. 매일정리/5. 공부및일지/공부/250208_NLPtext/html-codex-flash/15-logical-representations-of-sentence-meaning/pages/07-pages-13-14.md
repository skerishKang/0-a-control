## Block 1
- type: section_title
- source_text: 15.7 Representing Time and Aspect
- korean_translation: 15.7 시간과 상의 표현
- study_note: 문장에서 나타나는 시점(point in time)과 지속 기간(duration), 그리고 사건의 완료 여부와 같은 상(aspect) 정보를 논리적으로 포착하는 방법을 다룹니다.

## Block 2
- type: paragraph
- source_text: Time is an essential part of the meaning of many sentences. For example, "John is booking a flight" and "John booked a flight" describe the same event at different points in time.
- korean_translation: 시간은 많은 문장의 의미 중에서 필수적인 부분입니다. 예를 들어, "John이 항공편을 예약하고 있다"와 "John이 항공편을 예약했다"는 서로 다른 시점에서의 같은 사건을 묘사합니다.
- study_note: 시제(tense)의 변화에 따라 문장의 논리적 의미가 어떻게 달라지는지 화두를 던집니다.

## Block 3
- type: margin_note
- source_text: interval logic
- korean_translation: 구간 논리(interval logic)
- study_note: 시간을 하나의 점이 아니라 시작과 끝이 있는 구간으로 파악하여 선후 관계를 정의하는 논리 체계입니다.

## Block 4
- type: margin_note
- source_text: aspect
- korean_translation: 상(aspect)
- study_note: 사건이 진행 중인지, 완료되었는지 등 시간적 흐름 내부의 양상을 나타냅니다.

## Block 5
- type: paragraph
- source_text: Temporal logic provides a formal framework for representing time using intervals and points. For example, we can use the relation Before(e1, e2) to represent that event e1 occurred before event e2.
- korean_translation: 시제 논리(temporal logic)는 구간과 시점을 사용하여 시간을 표현하기 위한 형식적인 틀을 제공합니다. 예를 들어, 사건 e1이 e2보다 먼저 발생했음을 나타내기 위해 Before(e1, e2) 관계를 사용합니다.
- study_note: 시제 논리에서 사건 간의 시간적 순서를 정의하는 기본 방식을 설명합니다.

- study_note: 발화 시점(speech time), 사건 발생 시점(event time), 그리고 참조 시점(reference time) 사이의 관계를 규명하는 리헨바흐의 시제 이론을 다룹니다.

## Block 3
- type: margin_note
- source_text: Reichenbach's tense theory
- korean_translation: 리헨바흐의 시제 이론(Reichenbach's tense theory)
- study_note: S, E, R이라는 세 가지 시점을 사용하여 영어의 12시제를 체계적으로 설명한 이론입니다.

## Block 4
- type: list
- source_text:
    - S (Speech Time): The time when the sentence is uttered.
    - E (Event Time): The time when the event described by the sentence occurs.
    - R (Reference Time): The time from which the event is viewed.
- korean_translation:
    - S (발화 시점): 문장이 말해지는 시점.
    - E (사건 시점): 문장이 묘사하는 사건이 발생하는 시점.
    - R (참조 시점): 사건을 바라보는 관점이 되는 특정 시점.
- study_note: 리헨바흐 모델의 세 가지 핵심 시점에 대한 정의입니다.

## Block 5
- type: table
- source_text:
    | Tense | Relation |
    | :--- | :--- |
    | Simple Present | E = R = S |
    | Simple Past | E = R < S |
    | Present Perfect | E < R = S |
    | Past Perfect | E < R < S |
- korean_translation:
    | 시제(Tense) | 관계(Relation) |
    | :--- | :--- |
    | 단순 현재 | E = R = S |
    | 단순 과거 | E = R < S |
    | 현재 완료 | E < R = S |
    | 과거 완료 | E < R < S |
- study_note: 리헨바흐의 세 시점 관계를 통해 영어의 주요 시제들을 정의한 조견표입니다.
