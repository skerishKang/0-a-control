## Block 6
- type: paragraph
- source_text: The order of quantifiers matters. If we switch the order of ∀ and ∃, the meaning of the sentence changes. This is known as quantifier scope ambiguity.
- korean_translation: 양화사의 순서는 중요합니다. 만약 ∀와 ∃의 순서를 바꾸면 문장의 의미가 변하게 됩니다. 이를 양화사 범주 모호성(quantifier scope ambiguity)이라고 합니다.
- study_note: 양화사의 순서에 따라 발생할 수 있는 중의성 문제를 지적합니다.

## Block 3
- type: paragraph
- source_text: Inference is the process of deriving new information from existing a knowledge base. FOL provides a formal framework for performing inference using rules of inference like Modus Ponens.
- korean_translation: 추론은 기존의 지식 베이스로부터 새로운 정보를 이끌어내는 과정입니다. 1차 논리(FOL)는 전건 긍정(Modus Ponens)과 같은 추론 규칙을 사용하여 추론을 수행하기 위한 형식적 틀을 제공합니다.
- study_note: 추론의 정의와 대표적인 추론 규칙인 전건 긍정을 소개합니다.

## Block 4
- type: margin_note
- source_text: Modus Ponens
- korean_translation: 전건 긍정(Modus Ponens)
- study_note: (P ⇒ Q)이고 P가 참이면, Q도 참이라는 기본 논리 법칙입니다.

## Block 5
- type: margin_note
- source_text: knowledge base
- korean_translation: 지식 베이스(knowledge base)
- study_note: 사실과 규칙들이 논리적으로 표현되어 저장된 집합체입니다.

## Block 6
- type: paragraph
- source_text: Knowledge representation involves organizing information in a way that supports reasoning. This typically involves defining a set of facts and rules in a knowledge base.
- korean_translation: 지식 표현은 추론을 뒷받침할 수 있는 방식으로 정보를 조직하는 작업을 포함합니다. 이는 대개 지식 베이스 내에 일련의 사실(facts)과 규칙(rules)을 정의하는 일을 수반합니다.
- study_note: 지식 표현의 목적이 단순 저장이 아닌 '추론 지원'에 있음을 강조합니다.
