## Block 3
- type: paragraph
- source_text: A good meaning representation system should satisfy several requirements. These include veridicality, unambiguousness, and the ability to represent complex relationships.
- korean_translation: 좋은 의미 표현 시스템은 여러 요구 사항을 충족해야 합니다. 여기에는 진실성(veridicality), 비모호성(unambiguousness), 그리고 복잡한 관계를 표현할 수 있는 능력이 포함됩니다.
- study_note: 의미 표현 시스템의 설계 원칙들을 나열합니다.

## Block 4
- type: margin_note
- source_text: unambiguousness
- korean_translation: 비모호성(unambiguousness)
- study_note: 하나의 표현이 단 하나의 명확한 의미만을 가져야 한다는 원칙입니다.

## Block 5
- type: list
- source_text:
    - Veridicality: The representation should accurately reflect the truth conditions of the sentence.
    - Unambiguousness: Each representation should have exactly one interpretation.
    - Canonical Form: Different sentences with the same meaning should have the same representation.
    - Expressivity: The system should be able to represent a wide range of linguistic phenomena.
- korean_translation:
    - 진실성(Veridicality): 표현은 문장의 진리 조건을 정확하게 반영해야 함.
    - 비모호성(Unambiguousness): 각 표현은 정확히 하나의 해석만을 가져야 함.
    - 표준형(Canonical Form): 의미가 같은 서로 다른 문장들은 동일한 표현을 가져야 함.
    - 표현력(Expressivity): 시스템은 광범위한 언어 현상을 표현할 수 있어야 함.
- study_note: 의미 표현의 4가지 주요 요구 사항을 상세히 설명합니다.
