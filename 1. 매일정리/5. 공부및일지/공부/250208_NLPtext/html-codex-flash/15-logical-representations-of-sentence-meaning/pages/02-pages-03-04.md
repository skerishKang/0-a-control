## Block 2
- type: paragraph
- source_text: First-Order Logic (FOL) consists of several basic components, including constants, variables, predicates, and quantifiers. These components are used to build more complex representations.
- korean_translation: 1차 논리(FOL)는 상수(constants), 변수(variables), 술어(predicates), 그리고 양화사(quantifiers)를 포함한 몇 가지 기본 구성 요소로 이루어져 있습니다. 이 구성 요소들을 사용하여 더 복잡한 표현을 구축합니다.
- study_note: FOL을 구성하는 4가지 핵심 요소를 소개합니다.

## Block 5
- type: list
- source_text:
    - Constants: Represent specific entities in the world (e.g., Harry, Chicago).
    - Variables: Represent generic entities (e.g., x, y).
    - Predicates: Represent relations between entities (e.g., Booked, Near) or properties of entities (e.g., Red, Small).
    - Functions: Operations that map entities to other entities (e.g., FatherOf, PriceOf).
- korean_translation:
    - 상수(Constants): 세계에 존재하는 특정 개체들을 나타냄. (예: Harry, Chicago)
    - 변수(Variables): 범용 개체들을 나타냄. (예: x, y)
    - 술어(Predicates): 개체들 사이의 관계(예: Booked, Near)나 개체의 속성(예: Red, Small)을 나타냄.
    - 함수(Functions): 개체를 다른 개체로 대응시키는 연산. (예: FatherOf, PriceOf)
- study_note: 상수, 변수, 술어, 함수의 정의와 구체적인 예시를 설명합니다.
