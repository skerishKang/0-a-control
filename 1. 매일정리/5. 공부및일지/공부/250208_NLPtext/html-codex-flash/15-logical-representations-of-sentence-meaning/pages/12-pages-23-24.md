## Block 6
- type: paragraph
- source_text: The highest-profile role for Description Logics to date, has been as part of the development of the Semantic Web. The Semantic Web is an ongoing effort to provide a formally specified semantics of the contents of the Web (Fensel et al., 2003). A key component of this effort involves the creation and deployment of ontologies for various application areas of interest. The meaning representation language used to represent this knowledge is the Web Ontology Language (OWL) (McGuinness and van Harmelen, 2004). OWL embodies a Description Logic that corresponds roughly to the one we’ve been describing here.
- korean_translation: 지금까지 Description Logic이 가장 두드러지게 사용된 분야는 시맨틱 웹의 발전 과정입니다. 시맨틱 웹은 웹 콘텐츠에 대해 형식적으로 명시된 의미론을 제공하려는 지속적인 노력입니다(Fensel et al., 2003). 이 노력의 핵심 구성 요소 중 하나는 다양한 응용 영역을 위한 온톨로지를 만들고 배포하는 일입니다. 이 지식을 표현하는 의미 표현 언어가 바로 Web Ontology Language, 즉 OWL입니다(McGuinness and van Harmelen, 2004).
