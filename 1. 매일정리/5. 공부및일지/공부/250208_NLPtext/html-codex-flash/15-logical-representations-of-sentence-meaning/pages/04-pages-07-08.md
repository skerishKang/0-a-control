- study_note: 불완전한 논리 표현을 결합하여 완성된 의미를 형성하는 강력한 도구인 람다 계산식을 다루는 섹션입니다.

## Block 2
- type: paragraph
- source_text: Lambda calculus is a mathematical system for expressing computation based on function abstraction and application. In meaning representation, it is used to represent unsaturated predicates.
- korean_translation: 람다 계산식은 함수 추상화(abstraction)와 적용(application)을 바탕으로 계산을 표현하는 수학적 시스템입니다. 의미 표현에서는 이를 포화되지 않은 술어(unsaturated predicates)를 나타내는 데 사용합니다.
- study_note: 람다 계산식의 정의와 의미 표현에서의 주된 용도를 설명합니다.

## Block 3
- type: margin_note
- source_text: abstraction / application
- korean_translation: 추상화(abstraction) / 적용(application)
- study_note: 람다 연산의 두 가지 핵심적인 조작 방식입니다.

## Block 4
- type: margin_note
- source_text: lambda (λ)
- korean_translation: 람다 (λ)
- study_note: 인자를 받아들일 빈 자리를 표시하는 기호입니다.

## Block 5
- type: paragraph
- source_text: The symbol λ (lambda) is used to indicate that a predicate is waiting for an argument. For example, λx.Likes(x, Harry) represents the property of liking Harry.
- korean_translation: λ(람다) 기호는 술어가 인자를 기다리고 있음을 나타내는 데 사용됩니다. 예를 들어, λx.Likes(x, Harry)는 ‘Harry를 좋아함’이라는 속성을 나타냅니다.
- study_note: 람다 기호의 실질적인 의미와 사용 예시를 보여줍니다.
