Block 2
paragraph
Source Text
Lambda calculus is a mathematical system for expressing computation based on function abstraction and application. In meaning representation, it is used to represent unsaturated predicates.
Korean Translation
람다 계산식은 함수 추상화(abstraction)와 적용(application)을 바탕으로 계산을 표현하는 수학적 시스템입니다. 의미 표현에서는 이를 포화되지 않은 술어(unsaturated predicates)를 나타내는 데 사용합니다.
Study Note
람다 계산식의 정의와 의미 표현에서의 주된 용도를 설명합니다.

Source Text
The symbol λ (lambda) is used to indicate that a predicate is waiting for an argument. For example, λx.Likes(x, Harry) represents the property of liking Harry.
Korean Translation
λ(람다) 기호는 술어가 인자를 기다리고 있음을 나타내는 데 사용됩니다. 예를 들어, λx.Likes(x, Harry)는 'Harry를 좋아함이라는 속성을 나타냅니다.
