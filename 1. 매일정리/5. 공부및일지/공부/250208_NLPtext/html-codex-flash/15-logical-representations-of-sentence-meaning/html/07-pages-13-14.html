15.7 시간과 상의 표현
Study Note
문장에서 나타나는 시점(point in time)과 지속 기간(duration), 그리고 사건의 완료 여부와 같은 상(aspect) 정보를 논리적으로 포착하는 방법을 다릅니다.
Block 2
paragraph
Source Text
Time is an essential part of the meaning of many sentences. For example, "John is booking a flight" and "John booked a flight" describe the same event at different points in time.
Korean Translation
시간은 많은 문장의 의미 중에서 필수적인 부분입니다. 예를 들어, "John이 항공편을 예약하고 있다"와 "John이 항공편을 예약했다"는 서로 다른 시점에서의 같은 사건을 묘사합니다.
Study Note
시제(tense)의 변화에 따라 문장의 논리적 의미가 어떻게 달라지는지 화두를 던집니다.

Block 5
paragraph
Source Text
Temporal logic provides a formal framework for representing time using intervals and points. For example, we can use the relation Before(e1, e2) to represent that event e1 occurred before event e2.
Korean Translation
시제 논리(temporal logic)는 구간과 시점을 사용하여 시간을 표현하기 위한 형식적인 틀을 제공합니다. 예를 들어, 사건 e1이 e2보다 먼저 발생했음을 나타내기 위해 Before(e1, e2) 관계를 사용합니다.
Study Note
시제 논리에서 사건 간의 시간적 순서를 정의하는 기본 방식을 설명합니다.

Reichenbach's tense theory
Korean Translation
리헨바흐의 시제 이론(Reichenbach's tense theory)
Study Note
S, E, R이라는 세 가지 시점을 사용하여 영어의 12시제를 체계적으로 설명한 이론입니다.
Block 4
list
Source Text
- S (Speech Time): The time when the sentence is uttered.
- E (Event Time): The time when the event described by the sentence occurs.
- R (Reference Time): The time from which the event is viewed.
Korean Translation
- S (발화 시점): 문장이 말해지는 시점.
- E (사건 시점): 문장이 묘사하는 사건이 발생하는 시점.
- R (참조 시점): 사건을 바라보는 관점이 되는 특정 시점.
Study Note
리헨바흐 모델의 세 가지 핵심 시점에 대한 정의입니다.
