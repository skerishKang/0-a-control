15.6 사건과 관계의 표현
Study Note
단순한 속성이나 관계를 넘어, 시간과 상 등의 복잡한 정보를 가진 사건(event)을 어떻게 논리적으로 구조화할지 다릅니다.
Block 3
paragraph
Source Text
In natural language, many sentences describe events. For example, "John booked a flight from Chicago to New York." How can we represent such a complex event in FOL?
Korean Translation
자연어에서 많은 문장은 사건(event)을 묘사합니다. 예를 들어, "John이 시카고에서 뉴욕으로 가는 항공편을 예약했다"는 문장입니다. 이러한 복잡한 사건을 1차 논리(FOL)에서 어떻게 표현할 수 있을까요?
Study Note
복합적인 논항을 가진 사건을 표현할 때 직면하는 도전을 예시와 함께 소개합니다.
Block 4
margin_note
Source Text
reification
Korean Translation
실체화(reification)
Study Note
술어나 관계를 하나의 독립된 객체(개체)로 취급하여 다루는 기법입니다.
Block 5
paragraph
Source Text
Reification is a common technique for representing events in FOL. It involves introducing a new entity to represent the event itself.
Korean Translation
실체화(reification)는 1차 논리(FOL)에서 사건을 표현하기 위해 널리 쓰이는 기법입니다. 이는 사건 그 자체를 나타내는 새로운 개체를 도입하는 작업을 포함합니다.
Study Note
실체화의 정의와 사건 중심적 표현의 핵심 아이디어를 설명합니다.

데이비드슨의 사건 집합론을 확장하여 사건의 개별 구성 요소들을 술어로 쪼개어 표현하는 방식을 다릅니다.
Block 3
margin_note
Source Text
Neo-Davidsonian
Korean Translation
신-데이비드슨적(Neo-Davidsonian)
Study Note
사건을 개체로 보고 그 참여자들과의 관계를 별도의 술어로 연결하는 현대적인 분석 방식입니다.
Block 4
formula
Source Text
∃e Booking(e) ∧ Booker(e, John) ∧ Booked(e, Flight1) ∧ From(e, Chicago) ∧ To(e, New York)
Korean Translation
∃e Booking(e) ∧ Booker(e, John) ∧ Booked(e, Flight1) ∧ From(e, Chicago) ∧ To(e, New York)
Study Note
사건 e를 도입하여 예약자, 예약 대상, 출발지, 목적지를 각각 별도의 사실들로 연결한 신-데이비드슨적 논리 표현입니다.
Block 5
paragraph
Source Text
This representation is highly flexible. It allows us to represent events with a variable number of arguments and to add information about the time or location of the event easily.
Korean Translation
이 표현 방식은 매우 유연합니다. 이는 가변적인 수의 논항을 가진 사건을 표현할 수 있게 해주며, 사건의 시간이나 장소에 대한 정보를 쉽게 추가할 수 있도록 해줍니다.
Study Note
신-데이비드슨적 사건 표현이 갖는 유연성과 확장성이라는 강점을 강조합니다.
