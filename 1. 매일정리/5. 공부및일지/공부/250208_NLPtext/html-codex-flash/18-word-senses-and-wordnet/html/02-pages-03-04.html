Block 3
paragraph
Source Text
Synonymy is the relation between two word senses that can be substituted for each other in many contexts without changing the meaning of the sentence. For example, "couch" and "sofa" are synonyms.
Korean Translation
유의 관계는 문장의 의미를 바꾸지 않고 많은 문맥에서 서로 대체될 수 있는 두 단어 의미 사이의 관계입니다. 예를 들어, "couch"와 "sofa"는 유의어입니다.
Study Note
유의어의 핵심 속성인 '대체 가능성(substitutability)'을 강조합니다.

margin_note
Source Text
antonymy
Korean Translation
반의 관계(antonymy)
Study Note
상반되는 의미를 가지는 관계입니다.
Block 5
paragraph
Source Text
Antonyms are word senses that are opposite in meaning. For example, "big" and "small", or "hot" and "cold".
Korean Translation
반의어는 의미가 반대인 단어 의미들입니다. 예를 들어, "big"과 "small", 또는 "hot"과 "cold"가 있습니다.
Study Note
반의 관계가 성립하더라도 보통 같은 범주(예: 온도, 크기) 내에서 반대되는 속성을 공유함을 이해해야 합니다.
Block 6
margin_note
Source Text
hyponymy / hypernymy
Korean Translation
하위 관계 / 상위 관계
Study Note
포함 관계를 나타내는 계층적 구조입니다.
Block 7
paragraph
Source Text
Hyponymy and hypernymy describe a "kind-of" relationship. For example, "car" is a hyponym of "vehicle", and "vehicle" is a hypernym of "car".
Korean Translation
하위 관계(hyponymy)와 상위 관계(hypernymy)는 "A는 B의 일종이다"라는 관계를 설명합니다. 예를 들어, "car"는 "vehicle"의 하위어이고, "vehicle"은 "car"의 상위어입니다.
Study Note
계층적 구조에서 위(상위어)와 아래(하위어)의 관계를 명확히 정의합니다.

18.2 워드넷(WordNet): 영어 어휘 데이터베이스
Study Note
어휘 의미론 연구에서 가장 널리 사용되는 데이터베이스인 워드넷의 구체적인 구조를 다릅니다.
Block 3
paragraph
Source Text
WordNet is a large, hand-crafted lexical database for English. Its fundamental building block is the synset (synonym set), representing a single concept.
Korean Translation
워드넷은 수작업으로 구축된 대규모 영어 어휘 데이터베이스입니다. 워드넷의 기본 구축 단위는 하나의 개념을 나타내는 유의어 집합(synonym set), 즉 신셋(synset)입니다.
Study Note
워드넷이 단어 중심이 아닌 '개념(신셋)' 중심으로 설계되었음을 설명합니다.

margin_note
Source Text
synset
Korean Translation
신셋(synset)
Study Note
같은 개념을 공유하는 유의어들의 집합입니다.

Source Text
Each synset is associated with its part of speech, a definition (gloss), and example sentences.
