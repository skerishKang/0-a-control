Block 2
paragraph
Source Text
We can induce a CFG from a treebank by simply reading off the rules from the trees in the corpus. For example, if we have a tree where an NP has children Det and N, we add the rule NP → Det N to our grammar.
Korean Translation
우리는 코퍼스 내의 트리들로부터 규칙을 단순히 읽어냄으로써 트리뱅크에서 CFG를 유도할 수 있습니다. 예를 들어, NP가 Det와 N을 자식으로 갖는 트리가 있다면, 우리의 문법에 NP → Det N 규칙을 추가합니다.
Study Note
데이터로부터 문법을 자동으로 학습하는 기본적인 방법을 설명합니다.

핵심어(head)
Study Note
구(phrase)의 중심이 되어 그 구의 문법적 속성을 결정하는 단어입니다.
Block 5
paragraph
Source Text
The idea that every phrase has a central word called the head is fundamental to modern linguistics. The head of an NP is a noun, the head of a VP is a verb, and so on. The head determines the syntactic category and other important properties of the phrase.
Korean Translation
모든 구에는 핵심어(head)라 불리는 중심 단어가 있다는 아이디어는 현대 언어학의 기초입니다. NP의 핵심어는 명사이고, VP의 핵심어는 동사입니다. 핵심어는 해당 구의 구문 범주와 다른 중요한 속성들을 결정합니다.
Study Note
핵심어의 정의와 역할에 대해 설명합니다.
