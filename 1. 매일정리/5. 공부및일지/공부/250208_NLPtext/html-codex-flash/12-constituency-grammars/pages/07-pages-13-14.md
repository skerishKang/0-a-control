## Block 2
- type: margin_note
- source_text: dependency grammar
- korean_translation: 의존 문법(dependency grammar)
- study_note: 단어 간의 부모-자식 관계(의존 관계)를 통해 문장 구조를 표현하는 문법입니다.

## Block 3
- type: paragraph
- source_text: In constituency grammars, the focus is on how words group together into constituents. In dependency grammars, the focus is instead on the direct relations between the words in a sentence.
- korean_translation: 구 구조 문법에서는 단어들이 어떻게 성분으로 묶이는지에 초점을 맞춥니다. 반면, 의존 문법에서는 문장 내 단어들 사이의 직접적인 관계에 초점을 맞춥니다.
- study_note: 구 구조 문법과 의존 문법의 근본적인 관점 차이를 설명합니다.

## Block 4
- type: margin_note
- source_text: head / dependent
- korean_translation: 지배어(head) / 의존어(dependent)
- study_note: 의존 관계에서 핵심 역할을 하는 단어와 그에 딸린 단어를 부르는 용어입니다.

## Block 5
- type: paragraph
- source_text: A dependency relation is a binary relation between a head and a dependent. Usually, the head is a verb and its dependents are its arguments (like subject or object) or its modifiers.
- korean_translation: 의존 관계는 지배어(head)와 의존어(dependent) 사이의 이항 관계입니다. 보통 지배어는 동사이고, 그 의존어들은 그 동사의 논항(주어나 목적어 등)이나 수식어들입니다.
- study_note: 의존 관계의 핵심 구성 요소인 지배어와 의존어의 개념을 설명합니다.

## Block 2
- type: margin_note
- source_text: Universal Dependencies (UD)
- korean_translation: 보편 의존 관계(Universal Dependencies, UD)
- study_note: 구문 분석의 범언어적 표준을 목표로 하는 국제 프로젝트 명칭입니다.

## Block 3
- type: table
- source_text:
    | Relation | Description | Example |
    | :--- | :--- | :--- |
    | nsubj | nominal subject | [I] saw |
    | obj | direct object | saw [the man] |
    | nmod | nominal modifier | man [with a telescope] |
    | det | determiner | [the] man |
    | amod | adjectival modifier | [long] flight |
- korean_translation:
    | 관계명 | 설명 | 예시 |
    | :--- | :--- | :--- |
    | nsubj | 명사 주어(nominal subject) | [내(I)]가 보았다 |
    | obj | 직접 목적어(direct object) | [그 남자(the man)]를 보았다 |
    | nmod | 명사 수식어(nominal modifier) | [망원경(with a telescope)]을 가진 남자 |
    | det | 한정사(determiner) | [그(the)] 남자 |
    | amod | 형용사 수식어(adjectival modifier) | [긴(long)] 항공편 |
- study_note: UD에서 가장 흔히 사용되는 의존 관계 라벨들의 정의와 실제 예시입니다.

## Block 4
- type: paragraph
- source_text: The set of relations used depends on the formalism. Universal Dependencies (UD) provides a set of cross-linguistically consistent relations.
- korean_translation: 사용되는 관계의 집합은 형식 시스템에 따라 다릅니다. 보편 의존 관계(UD)는 언어 간에 일관되게 적용할 수 있는 관계 집합을 제공합니다.
- study_note: UD의 보편성(cross-linguistic consistency)을 장점으로 언급합니다.
