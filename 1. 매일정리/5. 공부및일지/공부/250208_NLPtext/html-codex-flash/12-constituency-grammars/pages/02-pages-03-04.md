- source_text: A context-free grammar is formally defined by a four-tuple (N, Σ, P, S):
- korean_translation: 문맥 자유 문법은 정식으로 4-튜플 (N, Σ, P, S)로 정의됩니다.
- study_note: CFG를 구성하는 네 가지 요소를 도입합니다.

## Block 4
- type: table
- source_text:
    | Symbol | Description |
    | :--- | :--- |
    | N | a set of non-terminal symbols |
    | Σ | a set of terminal symbols (the alphabet/lexicon), disjoint from N |
    | P | a set of productions or rules of the form A → α, where A ∈ N and α is any string of symbols from (N ∪ Σ)* |
    | S | a designated start symbol from N |
- korean_translation:
    | 기호 | 설명 |
    | :--- | :--- |
    | N | 비단말 기호들의 집합 |
    | Σ | 단말 기호(알파벳/어휘집)들의 집합, N과 서로소임 |
    | P | A → α 형태의 생성 규칙들의 집합 (여기서 A ∈ N이고, α는 (N ∪ Σ)*에 속하는 임의의 기호열) |
    | S | N에 속하는 지정된 시작 기호 |
- study_note: CFG의 구성 요소인 N, Σ, P, S에 대한 상세 설명입니다.

## Block 5
- type: paragraph
- source_text: A CFG generates a language, which is the set of all strings of terminals that can be derived from the start symbol S. A derivation is a sequence of rule applications that starts with S and ends with a string of terminals.
- korean_translation: CFG는 시작 기호 S로부터 유도될 수 있는 모든 단말 기호열의 집합인 언어를 생성합니다. 유도(derivation)란 S에서 시작하여 단말 기호열로 끝나는 일련의 규칙 적용 과정을 말합니다.
- study_note: CFG가 언어를 생성하는 원리와 유도의 개념을 설명합니다.

## Block 6
- type: paragraph
- source_text: We can represent a derivation as a parse tree, like the one we saw in Figure 12.1. In a parse tree, the root is the start symbol S, the leaves are the terminals, and the internal nodes are non-terminals.
- korean_translation: 우리는 유도 과정을 그림 12.1에서 보았던 것과 같은 구문 분석 트리로 표현할 수 있습니다. 구문 분석 트리에서 루트(root)는 시작 기호 S이고, 잎(leaves)은 단말 기호들이며, 내부 노드들은 비단말 기호들입니다.
- study_note: 유도 과정과 구문 분석 트리의 구조적 관계를 설명합니다.
