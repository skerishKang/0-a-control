- korean_translation: 17.4 부트스트래핑을 통한 반지도(Semi-Supervised) 관계 추출
- study_note: 소수의 규칙이나 종잣돈(seed) 데이터만으로 시스템 스스로 훈련 데이터를 생성해 나가는 부트스트래핑(bootstrapping) 기법을 다룹니다.

## Block 3
- type: paragraph
- source_text: Creating a large annotated corpus for every relation of interest is extremely expensive. Bootstrapping is a semi-supervised approach that addresses this bottleneck. It starts with a small set of "seed" instances or patterns.
- korean_translation: 관심 있는 모든 관계에 대해 방대한 주석 코퍼스(annotated corpus)를 구축하는 것은 극히 비용이 많이 듭니다. 부트스트래핑은 이러한 병목 현상을 해결하는 반지도(semi-supervised) 접근법입니다. 이는 소규모의 "종잣돈(seed)" 인스턴스 또는 패턴 세트로 시작합니다.
- study_note: 정답 구축 비용 문제라는 지도 학습의 근본적 한계와 그 돌파구로서 부트스트래핑 접근법을 소개합니다.

## Block 4
- type: margin_note
- source_text: bootstrapping / seeds
- korean_translation: 부트스트래핑(bootstrapping) / 시드(seeds)
- study_note: 초기 지식(시드)을 바탕으로 패턴을 찾고, 그 패턴으로 다시 새로운 지식을 추출하는 자기 주도적 반복 학습 과정입니다.

## Block 5
- type: paragraph
- source_text: The system uses these seeds to find new patterns in an unannotated corpus, and then uses the new patterns to find new instances, repeating the process iteratively.
- korean_translation: 시스템은 주석이 없는 코퍼스에서 새로운 패턴을 찾기 위해 이 시드(seeds)를 사용하며, 그 다음 새로운 인스턴스를 찾기 위해 새로운 패턴을 사용하여 이 과정을 반복적으로 수행합니다.
- study_note: 패턴 추출과 인스턴스 추출이 양방향 피드백 루프를 형성하는 부트스트래핑의 핵심 알고리즘 메커니즘을 설명합니다.

## Block 2
- type: section_title
- source_text: 17.5 Distant Supervision for Relation Extraction
- korean_translation: 17.5 관계 추출을 위한 원거리 지도(Distant Supervision)
- study_note: 기존에 구축된 거대한 지식 베이스(예: 위키데이터)를 텍스트 코퍼스와 결합하여 수많은 주석 데이터를 자동 생성(휴리스틱 기반)하는 효율적 학습 방식을 다룹니다.

## Block 3
- type: paragraph
- source_text: Distant supervision combines the advantages of supervised learning and bootstrapping. Instead of manual annotation or a small set of seeds, it uses a large existing knowledge base (like Freebase or Wikidata) to automatically label a training corpus.
- korean_translation: 원거리 지도(distant supervision)는 지도 학습과 부트스트래핑의 장점을 결합한 방식입니다. 수동 주석 작업이나 소규모의 시드 세트 대신 대규모 기존 지식 베이스(Freebase나 Wikidata 등)를 사용하여 훈련 코퍼스에 자동으로 레이블을 지정합니다.
- study_note: 기존 모델들의 제약을 극복하고 방대한 외부 지식을 활용하는 원거리 지도의 기본 철학을 설명합니다.

## Block 4
- type: margin_note
- source_text: distant supervision / knowledge base
- korean_translation: 원거리 지도(distant supervision) / 지식 베이스(knowledge base)
- study_note: 지식 베이스의 사실(A와 B의 관계)을 활용하여 "두 개체 A, B가 함께 등장하는 문장들은 모두 해당 관계를 나타낼 것이다"라는 가정을 토대로 대량의 학습 데이터를 얻어내는 기법입니다.

## Block 5
- type: paragraph
- source_text: The core assumption is: if two entities have a relation in the knowledge base, then any sentence containing both entities is assumed to express that relation. While noisy, this allows the creation of massive training datasets.
- korean_translation: 핵심 가정은 다음과 같습니다. 만약 특정 지식 베이스에서 두 개체가 어떤 관계를 맺고 있다면, 두 개체가 함께 포함된 모든 문장은 해당 관계를 표현하고 있다고 전제하는 것입니다. 이 방식에는 잡음(노이즈)이 섞이겠지만, 대규모 훈련 데이터세트를 구축할 수 있게 해줍니다.
- study_note: 데이터 생성에 쓰이는 강한 가정("Distant Supervision Assumption")과, 이로 인한 장단점(스케일 확장 vs. 노이즈 데이터)을 지적합니다.
