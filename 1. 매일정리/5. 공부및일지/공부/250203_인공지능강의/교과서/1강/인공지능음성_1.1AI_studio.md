## Part 1: 음성 생성의 기초

### 1.1 음성 신호 처리의 기본

#### 1.1.1 음성 신호 분석 및 처리 기법

- **켑스트럼 분석 (Cepstrum Analysis)**

    - **정의:** 켑스트럼 분석은 음성 신호의 주파수 영역 정보를 시간 영역으로 변환하여 음성 신호의 특성을 분석하는 기법입니다. 푸리에 변환된 스펙트럼의 로그 값을 취한 후, 다시 역푸리에 변환을 수행하여 얻습니다. 이를 통해 음원의 특성(성도, 피치)과 스펙트럼 포락선을 분리할 수 있습니다.
    
    - **수학적 정의:**
        
        1. 음성 신호 (x(t))에 푸리에 변환(Fourier Transform, $\mathcal{F}$)을 적용하여 주파수 영역으로 변환합니다.
            
            $$
            X(f) = \mathcal{F}\{x(t)\}
            $$
            
        2. 변환된 주파수 영역 신호의 크기(Magnitude)를 구합니다.
            
            $$
            |X(f)|
            $$
            
        3. 크기에 로그를 취합니다.
            
            $$
            \log |X(f)|
            $$
            
        4. 로그 스펙트럼에 역푸리에 변환(Inverse Fourier Transform, $\mathcal{F}^{-1}$)을 적용하여 켑스트럼 영역으로 변환합니다.
            
            $$
            C(t) = \mathcal{F}^{-1} \{\log |X(f)|\}
            $$
            
        * 여기서 (X(f))는 주파수 영역에서의 신호, (C(t))는 켑스트럼입니다.
    
    - **켑스트럼의 활용:**
        - 음성 신호에서 음원의 정보(피치)와 스펙트럼 포락선(성도) 정보를 분리하여 분석합니다.
        - 음성 합성 시, 켑스트럼을 사용하여 톤과 음색을 조절할 수 있습니다.(피치를 조절하여 톤을, 스펙트럼 포락선을 조절하여 음색을 변경)
        - 음성 인식 분야에서, 켑스트럼은 특징 벡터로 활용되어 음성 인식 성능을 향상시킵니다. (피치와 스펙트럼 포락선 정보를 모두 활용)
    
    - **실험적 예시:**

        | 단계 | 설명                                                                           | 코드 (Python 예시)                                                                                                                                                                   |
        | --- | ---------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
        | 1   | 음성 신호 로드                                                               | ```python import librosa import numpy as np audio_file = 'speech.wav' y, sr = librosa.load(audio_file, sr=None) ```                                                            |
        | 2   | 푸리에 변환 (FFT)                                                               | ```python n_fft = 2048 D = librosa.stft(y, n_fft=n_fft) ```                                                                                                               |
        | 3   | 스펙트럼 크기 계산                                                               | ```python mag = np.abs(D) ```                                                                                                                                                   |
        | 4   | 로그 스펙트럼 계산                                                               | ```python log_mag = np.log(mag) ```                                                                                                                                             |
        | 5   | 역푸리에 변환 (IFFT)                                                            | ```python cepstrum = librosa.istft(log_mag) ```                                                                                                                                      |
        | 6   | 켑스트럼 시각화                                                                | ```python import matplotlib.pyplot as plt plt.figure(figsize=(10, 4)) plt.plot(cepstrum) plt.title('Cepstrum') plt.xlabel('Quefrency') plt.ylabel('Amplitude') plt.show() ``` |
        
        * **실험 결과 분석:** 다양한 음성 데이터에 대해 위 코드를 실행하여 켑스트럼을 추출하고 시각화하여 음성 데이터의 특징을 분석합니다. 켑스트럼은 음성 신호의 저주파 성분, 즉 음색이나 스펙트럼 포락선과 관련된 정보를 나타냅니다.
        * **다양한 음성 신호 분석:** 남성, 여성, 아이 등 다양한 음성의 켑스트럼을 비교 분석하여 음성의 특징을 파악합니다. 음높이, 음색, 발음 등 음성 신호의 다양한 특징이 켑스트럼에 어떻게 나타나는지 분석합니다.

- **필터 설계**

    - **FIR (Finite Impulse Response) 필터**
        - **정의:** 유한 임펄스 응답 필터로, 현재 및 과거 입력 신호의 유한한 가중 합으로 출력을 생성합니다. 안정성이 보장되며 선형 위상 특성을 가지기 쉽습니다.
        - **수식:** 출력 (y[n])은 입력 (x[n])과 필터 계수 (b[k])의 컨볼루션으로 계산됩니다.
        
        $$
        y[n] = \sum_{k=0}^{M} b[k] x[n-k]
        $$
        
        * 여기서 (M)은 필터의 차수(Order)입니다.
        - **설계 방법:** 창 함수(Window function) 기반 설계 (Hamming, Hanning, Blackman 창 등), 주파수 샘플링 방법, 최적화 방법 등이 있습니다.
        - **특징:** 안정성 보장, 선형 위상 특성 구현 용이, 메모리 및 계산량이 많을 수 있음.

    - **IIR (Infinite Impulse Response) 필터**
        - **정의:** 무한 임펄스 응답 필터로, 현재 및 과거 입력 신호뿐만 아니라 과거 출력 신호도 사용하여 출력을 생성합니다. 동일한 필터 성능을 위해 FIR 필터보다 적은 차수를 사용할 수 있어 계산 효율성이 높습니다.
        - **수식:** 출력 (y[n])은 입력 (x[n])과 필터 계수 (b[k]) 및 과거 출력 (y[n-k])의 가중 합으로 계산됩니다.
        
        $$
        y[n] = \sum_{k=0}^{M} b[k] x[n-k] - \sum_{k=1}^{N} a[k] y[n-k]
        $$
        
        * 여기서 (M)과 (N)은 각각 입력 및 출력 피드백 필터의 차수입니다.
        - **설계 방법:** 아날로그 필터(Butterworth, Chebyshev, Elliptic)를 디지털 필터로 변환하는 방법(Impulse Invariance, Bilinear Transform) 등이 있습니다.
        - **특징:** 계산 효율성 높음, 위상 특성 설계 복잡, 불안정할 가능성 존재.
    
    - **필터 설계 실습**

        | 필터 종류 | 설계 방법                                                                                              | 코드 (Python 예시)                                                                                                                                 |
        | --- | ------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------- |
        | FIR  | 창 함수 (Hamming)                                                                                           | ```python import scipy.signal as signal import numpy as np N = 50 b = signal.firwin(N, cutoff=0.5, window='hamming') ```                |
        | IIR  | 버터워스 필터 설계                                                                                       | ```python from scipy.signal import butter, lfilter order = 5 cutoff_freq = 0.5 b, a = butter(order, cutoff_freq, btype='lowpass') ```      |
        | 필터 적용 | 설계된 필터를 음성 신호에 적용                                                                                     | ```python y_filtered = lfilter(b, a, y) # IIR 필터 적용 y_filtered = np.convolve(b, y, mode='same')  # FIR 필터 적용 ```     |

        * **필터 성능 분석:** 필터 설계 결과의 주파수 응답을 시각화하여 필터의 차단 주파수 및 감쇠율을 분석하고, 필터가 음성 신호에 미치는 영향을 분석합니다. 다양한 차수의 필터를 설계하여 성능을 비교하고, 각 필터의 특징과 장단점을 상세히 분석합니다.

### 시간-주파수 분석 방법론
- **STFT (Short-Time Fourier Transform)**
    - **정의:** 짧은 시간 구간별로 푸리에 변환을 수행하여 시간-주파수 정보를 얻는 방법입니다. 음성 신호의 시간 변화에 따른 주파수 성분 변화를 분석할 수 있습니다.
    - **수학적 정의:** 음성 신호 (x[n])을 윈도우 함수 (w[n])를 사용하여 분할한 후, 각 프레임에 푸리에 변환을 적용합니다.
        
        $$
        X(m, \omega) = \sum_{n=-\infty}^{\infty} x[n] w[n - mR] e^{-j \omega n}
        $$
        
        * 여기서 (m)은 프레임 인덱스, (R)은 홉 사이즈(Hop Size)입니다.
    - **특징:** 시간 및 주파수 해상도가 상호 연관됨(불확정성 원리). 창 함수(Window Function) 선택에 따라 시간-주파수 해상도 조절 가능.
    - **실습:** librosa 라이브러리를 사용하여 STFT를 계산하고 스펙트로그램을 시각화합니다.

- **웨이블릿 변환 (Wavelet Transform)**
    - **정의:** 다양한 스케일(Scale)과 시간 이동을 가지는 웨이블릿 함수를 사용하여 음성 신호를 분석하는 방법입니다. STFT에 비해 다중 해상도 분석이 가능하며, 시간 및 주파수 정보를 더 정확하게 얻을 수 있습니다.
    - **수학적 정의:** 음성 신호 (x(t))를 웨이블릿 함수 (ψ(t))를 사용하여 변환합니다.
        
        $$
        W(a, b) = \frac{1}{\sqrt{a}} \int_{-\infty}^{\infty} x(t) \psi \left(\frac{t-b}{a}\right) dt
        $$
        
        * 여기서 (a)는 스케일, (b)는 시간 이동입니다.
    - **특징:** 다중 해상도 분석 가능, 시간 및 주파수 해상도 조절 유연, 비정상 신호 분석에 효과적.
    - **실습:** pywt 라이브러리를 사용하여 웨이블릿 변환을 수행하고 스케일로그램을 시각화합니다.

- **시간-주파수 분석 비교**

    | 분석 방법 | 장점                                                    | 단점                                                         | 활용 분야                                                        |
    | -------- | ------------------------------------------------------- | ------------------------------------------------------------ | --------------------------------------------------------------- |
    | STFT     | 구현 용이, 계산 효율 높음, 직관적 해석 가능                 | 시간-주파수 해상도 제한, 창 함수 선택에 따른 성능 변화             | 음성 분석, 스펙트로그램 생성                                             |
    | 웨이블릿 변환 | 다중 해상도 분석 가능, 비정상 신호 분석에 효과적, 시간-주파수 정보 정밀 | 계산 복잡, 웨이블릿 함수 선택에 따른 성능 변화, 해석 어려움 가능 | 음성 분리, 잡음 제거, 신호 분석                                        |

    * **실험:** 실제 음성 신호에 STFT와 웨이블릿 변환을 적용하여 각 방법의 장단점을 비교 분석합니다. 다양한 파라미터 설정에 따른 결과 변화를 분석하고, 각 방법의 특징을 명확히 이해합니다.

### 시간 영역 분석
- **Zero-Crossing Rate (ZCR)**
    - **정의:** 신호가 0을 교차하는 횟수를 시간 단위로 나타내는 값입니다. 주로 무성음과 유성음을 구분하는 데 사용됩니다.
    - **수학적 정의:** 주어진 시간 구간 내에서 신호의 부호가 바뀌는 횟수를 계산합니다.
        
        $$
        ZCR = \frac{1}{T} \sum_{n=1}^{T-1} \mathbb{I} \{ \mathrm{sign}(x[n]) \neq \mathrm{sign}(x[n-1]) \}
        $$
        
        * 여기서 \(T\)는 시간 구간의 길이, \(\mathbb{I}\)는 지시 함수입니다.
    - **특징:** 계산이 간단하고 실시간 처리 가능, 노이즈에 민감하게 반응.

- **Autocorrelation (자기 상관)**
    - **정의:** 신호 자신과 시간 이동된 신호 사이의 유사성을 나타내는 값입니다. 주로 음성의 피치(기본 주파수)를 추정하는 데 사용됩니다.
    - **수학적 정의:** 신호 (x[n])에 대해 시간 지연 (τ)에 따른 유사도를 계산합니다.
        
        $$
        R(\tau) = \sum_{n=0}^{N-\tau-1} x[n] x[n + \tau]
        $$
        
        * 여기서 (N)은 신호의 길이입니다.
    - **특징:** 음성 신호의 주기성을 파악하는 데 효과적, 계산 비용이 높을 수 있음.

- **시간 영역 분석 실습**
    - **Python 코드 예시:**
        
        ```python
        import numpy as np
        
        def calculate_zcr(signal, frame_length):
            zcr = []
            for i in range(0, len(signal)-frame_length, frame_length):
                frame = signal[i:i+frame_length]
                count = np.sum(np.abs(np.diff(np.sign(frame))))/2
                zcr.append(count/frame_length)
            return np.array(zcr)
        
        def calculate_autocorrelation(signal, max_lag):
            autocorr = np.correlate(signal, signal, mode='full')
            mid = autocorr.size // 2
            return autocorr[mid:mid + max_lag]
        
        
        # 음성 신호 로드 및 시간 영역 분석 수행
        # ZCR 및 자기상관 분석 결과 시각화 및 해석
        
        ```
    
    * **실험 결과 분석:** 다양한 음성 신호에 대해 ZCR과 자기 상관 함수를 계산하고 시각화합니다. ZCR을 통해 유성음과 무성음 구간을 식별하고, 자기 상관 함수를 통해 음성의 피치를 추정합니다.

- **주파수 영역 분석**
    - **FFT (Fast Fourier Transform)**
        - **정의:** 이산 푸리에 변환(DFT)을 빠르게 계산하는 알고리즘입니다. 음성 신호의 주파수 성분을 분석하는 데 필수적입니다.
        - **수학적 정의:**
            
            $$
            X(k) = \sum_{n=0}^{N-1} x[n] e^{-j 2 \pi \frac{kn}{N}}
            $$
            
            * 여기서 (x[n])은 입력 신호, (X(k))는 주파수 성분, (N)은 샘플 개수입니다.
        - **특징:** 빠른 계산 속도, 다양한 주파수 분석 가능, 시간 정보를 잃음.
        - **실습:** numpy 또는 librosa 라이브러리를 사용하여 FFT를 계산하고 스펙트럼을 시각화합니다.

    - **Spectrogram**
        - **정의:** STFT 결과를 시각화한 것으로, 시간 축과 주파수 축으로 구성된 2차원 그래프입니다. 음성 신호의 시간 변화에 따른 주파수 성분 변화를 직관적으로 파악할 수 있습니다.
        - **특징:** 음성 신호의 시간-주파수 특성을 시각적으로 표현, 음성 분석 및 합성 연구에 널리 사용됨.
        - **실습:** librosa를 사용하여 스펙트로그램을 생성하고 시각화합니다.

    - **Mel-Spectrogram**
        - **정의:** 스펙트로그램을 멜 스케일로 변환한 것입니다. 사람의 청각 특성에 맞게 주파수 축을 비선형적으로 변환하여 주파수 정보를 표현합니다.
        - **수학적 정의:** 주파수 (f)를 멜 스케일(m)로 변환합니다.
            
            $$
            m = 2595 \log_{10} \left(1 + \frac{f}{700}\right)
            $$
            
        - **특징:** 사람의 청각에 더 적합한 주파수 표현, 음성 인식 및 음성 합성에 활용.
        - **실습:** librosa 라이브러리를 사용하여 멜 스펙트로그램을 생성하고 시각화합니다.

    - **주파수 영역 분석 실습**

### 1.1.2 음성 특징 추출 심화

- **MFCC (Mel Frequency Cepstral Coefficients)**
    - **정의:** 사람의 청각 특성을 반영하여 음성 신호의 특징을 추출하는 방법입니다. 멜 스케일로 변환된 스펙트럼에 로그를 취한 후, 이산 코사인 변환(DCT)을 수행하여 얻습니다.
    - **수학적 배경:**
        - 멜 스케일은 사람의 청각이 주파수를 인지하는 특성을 반영한 비선형 척도입니다. 낮은 주파수 영역에서는 주파수 변화에 민감하게 반응하고, 높은 주파수 영역에서는 덜 민감하게 반응합니다.
        - 멜 스케일 변환을 통해 음성 신호의 주요 특징을 효과적으로 추출할 수 있습니다.
    - **수식:**
        1. 음성 신호 (x(t))를 시간 영역에서 주파수 영역으로 변환하기 위해 FFT를 수행합니다.
            
            $$
            X(f) = \mathcal{F}\{x(t)\}
            $$
            
        2. FFT 결과의 크기를 계산합니다.
            
            $$
            |X(f)|
            $$
            
        3. 스펙트럼의 크기를 멜 스케일로 변환합니다.
            
            $$
            \log (|Mel(f)|)
            $$
            
        4. 멜 스케일로 변환된 스펙트럼에 이산 코사인 변환 (DCT)을 적용하여 MFCC를 추출합니다.
            
            $$
            MFCC(t) = \mathcal{DCT} \left( \log(Mel(f)) \right)
            $$
            
    - **MFCC 특징 추출 과정**
        1. **프레임 분할 (Framing):** 음성 신호를 짧은 구간(프레임)으로 나눕니다.
        2. **윈도우 적용 (Windowing):** 각 프레임에 윈도우 함수를 적용하여 프레임 양 끝의 불연속성을 완화합니다.
        3. **FFT (Fast Fourier Transform):** 각 프레임에 FFT를 적용하여 주파수 영역으로 변환합니다.
        4. **멜 필터뱅크 적용 (Mel Filterbank):** 멜 스케일에 따라 설계된 삼각 필터뱅크를 적용하여 주파수 성분을 멜 스케일로 변환합니다.
        5. **로그 변환 (Log Transform):** 각 필터뱅크 출력에 로그를 취합니다.
        6. **DCT (Discrete Cosine Transform):** 로그 변환된 값에 DCT를 적용하여 MFCC 계수를 추출합니다.
    - **실습:**

        | 단계 | 설명                                                                     | 코드 (Python 예시)                                                                                                             |
        | --- | ------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
        | 1   | 음성 신호 로드                                                             | ```python import librosa import numpy as np audio_file = 'speech.wav' y, sr = librosa.load(audio_file, sr=None) ```        |
        | 2   | MFCC 추출                                                               | ```python mfccs = librosa.feature.mfcc(y=y, sr=sr, n_mfcc=40) ```                                                            |
        | 3   | MFCC 시각화                                                                | ```python import matplotlib.pyplot as plt plt.figure(figsize=(10, 4)) librosa.display.specshow(mfccs, sr=sr, x_axis='time') plt.colorbar() plt.title('MFCC') plt.show() ``` |
        
    * **실험 결과 분석:** 다양한 음성 신호에 대해 MFCC를 추출하고 시각화하여 음성의 특징을 분석합니다. 남성, 여성, 아이 등 다양한 음성의 MFCC를 비교 분석하여 음성 특징을 파악합니다.

- **PLP (Perceptual Linear Prediction)**
    - **정의:** 사람의 청각 인지 모델을 기반으로 음성 신호의 스펙트럼을 모델링하는 방법입니다. 음성의 스펙트럼을 청각적으로 중요한 특징만 남기도록 변형합니다.
    - **수학적 배경:**
        - 사람의 청각은 모든 주파수에 대해 동일한 감도를 가지지 않습니다. PLP는 이러한 청각 특성을 고려하여 음성 신호의 주파수 영역 정보를 변형합니다.
        - 음성 신호의 스펙트럼을 인간의 청각 인지 모델에 맞게 왜곡하여 분석합니다.
        - 음성 인식 분야에서 강인한 특징 추출 방법으로 활용됩니다.
    - **특징:**
        - 청각 인지 특성을 반영하여 음성 특징 추출
        - 음성 인식 시스템의 성능 향상
        - 잡음에 강인한 특징 추출
    - **PLP 특징 추출 과정**
        1. **전처리:** 음성 신호의 전처리 과정 (프레임 분할, 윈도우 적용, FFT)
        2. **주파수 워핑:** 주파수 스케일을 청각 임계 값에 맞게 변형합니다.
        3. **등가 사운드 압력 계산:** 변형된 스펙트럼을 기반으로 청각 인지 모델에 따라 등가 사운드 압력을 계산합니다.
        4. **자동 상관 분석:** 스펙트럼의 자동 상관 분석 수행합니다.
        5. **선형 예측 분석:** 선형 예측 계수를 계산하여 스펙트럼 포락선을 모델링합니다.
        6. **PLP 계수 추출:** 선형 예측 계수를 기반으로 PLP 계수를 추출합니다.
    - **실습:** 다음 코드를 사용하여 PLP 특징을 추출하고 시각화합니다.

        ```python
        import librosa
        import numpy as np
        from scipy.signal import lfilter
        
        def perceptual_filterbank(n_fft, sr, n_bands=20):
            freqs = np.linspace(0, sr // 2, n_fft // 2 + 1)
            bands = np.linspace(0, sr // 2, n_bands + 1)
            filterbank = np.zeros((n_bands, n_fft // 2 + 1))
            for b in range(n_bands):
                low = bands[b]
                high = bands[b+1]
                for f_idx, freq in enumerate(freqs):
                    if low <= freq <= high:
                        filterbank[b, f_idx] = 1 - abs(freq - (low + high) / 2) / ((high - low) / 2)
            return filterbank
        
        def linear_prediction(spectrum, order=12):
            acorr = np.fft.ifft(spectrum**2).real[:order+1]
            r = acorr[1:]
            R = np.array([acorr[:order] for _ in range(order)]).T
            R = R - np.diag(np.diag(R))
            R[0, :] = np.roll(R[0,:], shift=1)
            R[-1, :] = np.roll(R[-1,:], shift=-1)
            
            a = -np.linalg.solve(R, r)
            return np.concatenate(([1], a))
        
        def lpcc_from_lpc(lpc):
            order = len(lpc) -1
            lpcc = np.zeros_like(lpc)
            
            for m in range(order+1):
                if m == 0:
                    lpcc[m] = 1
                else:
                   for k in range(1, m):
                       lpcc[m] += lpcc[k] * lpc[m-k] * k / m
                   lpcc[m] += lpc[m]
            
            return lpcc
        
        def plp(y, sr, n_fft=2048, hop_length=512, order = 12):
          
           frames = librosa.util.frame(y, frame_length=n_fft, hop_length=hop_length)
           magnitude = np.abs(np.fft.rfft(frames*np.hamming(n_fft), axis=0))
           
           filterbank = perceptual_filterbank(n_fft, sr)
           filtered = np.dot(filterbank, magnitude)
           
           eq_loudness = filtered**0.33
           
           plp_coefs = [lpcc_from_lpc(linear_prediction(eq_loudness[:,t], order)) for t in range(frames.shape[1])]
           return np.array(plp_coefs).T
           
           
        # 음성 신호 로드
        y, sr = librosa.load('speech.wav', sr=None)
        
        # PLP 특징 추출
        plp_coef = plp(y, sr)
        
        # PLP 특징 시각화
        import matplotlib.pyplot as plt
        plt.figure(figsize=(10, 4))
        plt.imshow(plp_coef, aspect='auto', origin='lower')
        plt.title('PLP Features')
        plt.xlabel('Frame')
        plt.ylabel('Coefficient')
        plt.colorbar()
        plt.show()
        ```
    
    * **실험 결과 분석:** 다양한 음성 신호에 대해 PLP를 추출하고 시각화하여 음성의 특징을 분석합니다. 남성, 여성, 아이 등 다양한 음성의 PLP를 비교 분석하여 음성 특징을 파악합니다.

- **Gammatone 필터뱅크**
    - **정의:** 사람의 청각 필터 특성을 모방한 감마톤 함수를 사용하여 음성 신호의 특징을 추출하는 방법입니다. 청각 인지 모델을 기반으로 다양한 주파수 대역에서 특징을 추출할 수 있습니다.
    - **수학적 배경:**
        - 감마톤 함수는 인간의 달팽이관 내에서 진동을 일으키는 기저막의 움직임을 모델링한 함수입니다.
        - 청각 필터 특성을 반영하여 음성 신호에서 특정 주파수 성분을 강조할 수 있습니다.
    - **특징:**
        - 청각 필터 특성을 모방하여 음성 특징 추출
        - 음성 인식 시스템의 성능 향상
        - 주파수 분해능 조절 가능
    - **Gammatone 필터뱅크 구현:**
        - 감마톤 필터의 중심 주파수와 대역폭을 설정합니다.
        - 각 중심 주파수에 대해 감마톤 필터를 생성합니다.
        - 음성 신호에 감마톤 필터를 적용하여 필터뱅크 출력을 생성합니다.
    - **실습:** 다음 코드를 사용하여 감마톤 필터뱅크 특징을 추출하고 시각화합니다.

        ```python
        import numpy as np
        import librosa
        import matplotlib.pyplot as plt
        from scipy.signal import convolve
        
        def gammatone_filter(center_frequency, bandwidth, order, sample_rate, duration):
            t = np.linspace(0, duration, int(duration * sample_rate), endpoint=False)
            a = 2 * np.pi * bandwidth
            g = ((a**order / np.math.factorial(order-1)) * (t**(order-1)) * np.exp(-a * t) * np.cos(2*np.pi * center_frequency * t))
            return g
        
        def gammatone_filterbank(num_filters, sample_rate, duration, low_freq=50, high_freq=8000, order=4):
            
            center_frequencies = np.linspace(low_freq, high_freq, num_filters)
            filterbank = [gammatone_filter(cf, 0.74 * 24.7 * (4.37 * cf / 1000 + 1), order, sample_rate, duration) for cf in center_frequencies]
            return center_frequencies, filterbank
        
        def gammatone_features(y, sr, num_filters, duration):
            center_frequencies, filterbank = gammatone_filterbank(num_filters, sr, duration)
            filtered = [convolve(y, f, mode='same') for f in filterbank]
            return np.array(filtered)
        
        
        # 음성 신호 로드
        y, sr = librosa.load('speech.wav', sr=None)
        duration = len(y)/sr
        
        # Gammatone 필터뱅크 특징 추출
        num_filters = 40
        gammatone_feat = gammatone_features(y, sr, num_filters, duration)
        
        
        # Gammatone 필터뱅크 특징 시각화
        plt.figure(figsize=(10, 4))
        plt.imshow(gammatone_feat, aspect='auto', origin='lower', extent=[0, duration, 0, num_filters])
        plt.title('Gammatone Filterbank Features')
        plt.xlabel('Time (sec)')
        plt.ylabel('Channel')
        plt.colorbar()
        plt.show()
        ```
    
    * **실험 결과 분석:** 다양한 음성 신호에 대해 감마톤 필터뱅크 특징을 추출하고 시각화하여 음성의 특징을 분석합니다. 남성, 여성, 아이 등 다양한 음성의 감마톤 필터뱅크 특징을 비교 분석하여 음성 특징을 파악합니다.

### 1.1.3 잡음 제거 및 음성 향상 알고리즘
- **잡음 제거 알고리즘:**
    - **스펙트럼 감산 (Spectral Subtraction):**
        - **정의:** 잡음이 섞인 음성 신호에서 잡음 스펙트럼을 추정하여 제거하는 방법입니다.
        - **수학적 정의:** 잡음이 포함된 음성 신호 스펙트럼에서 잡음 스펙트럼 추정치를 감산하여 순수한 음성 신호를 복원합니다.
            
            $$
            \hat{S}(f) = |Y(f)| - \alpha \sqrt{N(f)}
            $$
            
            * 여기서 (Y(f))는 잡음이 포함된 신호 스펙트럼, (N(f))는 잡음 스펙트럼, (α)는 감산 계수입니다.
        - **특징:** 구현이 간단하고 계산량이 적음, 음성 왜곡 및 잔류 잡음 문제 발생.
        - **실습:** Python을 사용하여 스펙트럼 감산 알고리즘을 구현하고, 음성 신호에 적용하여 잡음 제거 성능을 분석합니다.
    - **위너 필터 (Wiener Filter):**
        - **정의:** 최소 평균 제곱 오차(MMSE) 기준에 따라 최적의 필터를 설계하여 잡음을 제거하는 방법입니다.
        - **수학적 정의:** Wiener 필터는 음성 신호와 잡음 신호의 스펙트럼 파워 비를 사용하여 설계됩니다.
            
            $$
            H(f) = \frac{|S(f)|^2}{|S(f)|^2 + |N(f)|^2}
            $$
            
            * 여기서 (S(f))는 음성 신호 스펙트럼, (N(f))는 잡음 신호 스펙트럼입니다.
        - **특징:** 스펙트럼 감산보다 높은 성능 제공, 잡음 및 음성 신호에 대한 사전 정보 필요.
        - **실습:** Python을 사용하여 위너 필터 알고리즘을 구현하고, 음성 신호에 적용하여 잡음 제거 성능을 분석합니다.
    - **칼만 필터 (Kalman Filter):**
        - **정의:** 시스템의 상태를 추정하는 알고리즘으로, 시간 변화에 따라 상태를 예측하고 측정값을 기반으로 상태를 보정합니다.
        - **수학적 정의:** 칼만 필터는 예측 단계와 보정 단계로 구성됩니다.
            
            * **예측 단계:**
                
                $$
                \hat{x}_{k|k-1} = F_k \hat{x}_{k-1|k-1} + B_k u_k
                $$
                $$
                P_{k|k-1} = F_k P_{k-1|k-1} F_k^T + Q_k
                $$
                
            * **보정 단계:**
                
                $$
                K_k = P_{k|k-1} H_k^T (H_k P_{k|k-1} H_k^T + R_k)^{-1}
                $$
                $$
                \hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k (z_k - H_k \hat{x}_{k|k-1})
                $$
                $$
                P_{k|k} = (I - K_k H_k) P_{k|k-1}
                $$
                
            * 여기서 (x)는 시스템의 상태, (P)는 공분산 행렬, (F)는 상태 전이 행렬, (B)는 제어 입력 행렬, (u)는 제어 입력, (H)는 측정 행렬, (R)는 측정 노이즈 공분산, (Q)는 프로세스 노이즈 공분산, (z)는 측정값입니다.
        - **특징:** 시간 변화에 따른 잡음 특성 반영, 안정적인 성능 제공, 초기 값 설정 및 시스템 모델링 필요.
        - **실습:** Python을 사용하여 칼만 필터 알고리즘을 구현하고, 음성 신호에 적용하여 잡음 제거 성능을 분석합니다.
    - **딥러닝 기반 잡음 제거 모델:**
        - **정의:** 딥러닝 모델 (CNN, RNN, Transformer 등)을 사용하여 잡음을 제거하는 방법입니다.
        - **특징:** 대량의 데이터 학습을 통해 높은 성능 제공, 복잡한 잡음 환경에 강인함, 학습 및 모델링 복잡.
