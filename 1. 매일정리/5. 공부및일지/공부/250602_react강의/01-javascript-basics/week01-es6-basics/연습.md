## [] 의 의미

```javascript
const users = [];
```

**`[]`는 "빈 배열"을 만드는 문법입니다.**

```javascript
const users = [];        // 빈 배열 (아무것도 없음)
const numbers = [1, 2, 3]; // 숫자가 들어있는 배열
const names = ["김철수", "이영희"]; // 문자가 들어있는 배열
```

## "배열 자체는 변경 안 함"의 의미

### 배열 자체를 바꾸는 것 (❌ 불가능)
```javascript
const users = [];

// 이런 건 안 됨 - 배열 자체를 바꾸려고 함
users = ["새로운", "배열"]; // TypeError! 
users = []; // TypeError!
```

**"배열 자체를 바꾼다" = 변수에 완전히 새로운 배열을 대입하는 것**

### 배열 내용을 바꾸는 것 (✅ 가능)
```javascript
const users = [];

// 이런 건 됨 - 배열 내용만 바꾸는 것
users.push("김철수");        // 요소 추가
users.push("이영희");        // 요소 추가
console.log(users);         // ["김철수", "이영희"]

users.pop();               // 마지막 요소 제거
console.log(users);        // ["김철수"]

users[0] = "박철수";        // 첫 번째 요소 변경
console.log(users);        // ["박철수"]
```

## 쉬운 비유

**배열을 "상자"라고 생각해보세요:**

```javascript
const 상자 = []; // 빈 상자
```

**❌ 안 되는 것:**
- 상자 자체를 다른 상자로 바꾸기
- `상자 = 새로운상자` (TypeError!)

**✅ 되는 것:**
- 상자 안에 물건 넣기: `상자.push("물건")`
- 상자 안 물건 빼기: `상자.pop()`
- 상자 안 물건 바꾸기: `상자[0] = "새물건"`

## 실제 예시

```javascript
const 장바구니 = []; // 빈 장바구니

// 상품 추가 (가능)
장바구니.push("사과");
장바구니.push("바나나");
console.log(장바구니); // ["사과", "바나나"]

// 상품 제거 (가능)  
장바구니.pop(); // "바나나" 제거
console.log(장바구니); // ["사과"]

// 장바구니 자체를 바꾸려고 하면 (불가능)
// 장바구니 = []; // TypeError!
```

**정리하면:**
- `[]`: 빈 배열을 만드는 문법
- "배열 자체는 변경 안 함": 변수에 새로운 배열을 대입하는 건 안 되지만, 배열 안의 내용물을 바꾸는 건 OK!
