## 1. 변수 선언: let, const

### 1.1 기존 var의 문제점
```javascript
// 1. 함수 스코프 (블록 스코프 X)
if (true) {
    var x = 1;
}
console.log(x); // 1 (블록 밖에서도 접근 가능)

// 2. 호이스팅으로 인한 혼란
console.log(y); // undefined (에러가 아님!)
var y = 5;

// 3. 재선언 가능 (의도하지 않은 덮어쓰기)
var name = "김철수";
var name = "이영희"; // 문제없이 실행됨
```

### 1.2 ES6+ 방식: let과 const
```javascript
// let: 재할당 가능, 블록 스코프
let age = 25;
age = 26; // OK

// const: 재할당 불가능, 블록 스코프
const PI = 3.14159;
// PI = 3.14; // TypeError!

// 블록 스코프
{
    let blockVar = "블록 안";
    const BLOCK_CONST = "상수";
}
// console.log(blockVar); // ReferenceError!
```

### 1.3 언제 무엇을 사용할까?
- **const**: 기본적으로 사용 (값이 변하지 않을 때)
- **let**: 값이 변경될 필요가 있을 때
- **var**: 사용하지 않음 (레거시 코드 제외)

## 2. 화살표 함수 (Arrow Functions)

### 2.1 기본 문법
```javascript
// 기존 함수 선언
function add(a, b) {
    return a + b;
}

// 화살표 함수 (동일한 기능)
const add = (a, b) => {
    return a + b;
};

// 더 간결하게 (한 줄 표현)
const add = (a, b) => a + b;
```

### 2.2 다양한 형태
```javascript
// 매개변수가 하나일 때 (괄호 생략 가능)
const double = x => x * 2;
const square = (x) => x * x; // 괄호 유지해도 됨

// 매개변수가 없을 때
const greet = () => "안녕하세요!";

// 객체 반환 시 (괄호로 감싸기)
const makePerson = (name, age) => ({ name, age });

// 여러 줄 함수
const complexFunction = (x, y) => {
    const result = x + y;
    console.log(`결과: ${result}`);
    return result;
};
```

### 2.3 화살표 함수의 특징
- `this` 바인딩이 다름 (나중에 React에서 중요!)
- `arguments` 객체 없음
- 생성자 함수로 사용 불가
- 더 간결한 문법

## 3. 템플릿 리터럴 (Template Literals)

### 3.1 기본 사용법
```javascript
const name = "홍길동";
const age = 30;

// 기존 방식 (문자열 연결)
const message1 = "안녕하세요! 제 이름은 " + name + "이고, 나이는 " + age + "살입니다.";

// 템플릿 리터럴 (백틱 사용)
const message2 = `안녕하세요! 제 이름은 ${name}이고, 나이는 ${age}살입니다.`;
```

### 3.2 다양한 활용
```javascript
// 표현식 사용
const a = 10;
const b = 20;
console.log(`${a} + ${b} = ${a + b}`); // "10 + 20 = 30"

// 함수 호출
const getGreeting = () => "안녕하세요";
console.log(`${getGreeting()}, 반갑습니다!`);

// 여러 줄 문자열
const multiLine = `
    첫 번째 줄
    두 번째 줄
    세 번째 줄
`;

// HTML 템플릿 (나중에 JSX와 비슷!)
const createCard = (title, content) => `
    <div class="card">
        <h2>${title}</h2>
        <p>${content}</p>
    </div>
`;
```

## 4. 구조분해할당 (Destructuring)

### 4.1 배열 구조분해할당
```javascript
// 기존 방식
const colors = ["red", "green", "blue"];
const first = colors[0];
const second = colors[1];

// 구조분해할당
const [first, second, third] = colors;
console.log(first);  // "red"
console.log(second); // "green"
console.log(third);  // "blue"

// 일부만 가져오기
const [primaryColor] = colors; // "red"
const [, , tertiaryColor] = colors; // "blue"
```

### 4.2 객체 구조분해할당
```javascript
// 기존 방식
const person = { name: "김철수", age: 25, city: "서울" };
const name = person.name;
const age = person.age;

// 구조분해할당
const { name, age, city } = person;

// 변수명 변경
const { name: userName, age: userAge } = person;

// 기본값 설정
const { name, age, country = "한국" } = person;
```

## 5. 단축 속성명 (Shorthand Properties)

```javascript
const name = "이영희";
const age = 28;

// 기존 방식
const person1 = {
    name: name,
    age: age
};

// 단축 속성명
const person2 = { name, age }; // 동일한 결과
```
