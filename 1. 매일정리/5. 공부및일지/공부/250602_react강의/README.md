# React 강의 자료 구조 및 안내

이 저장소는 10~12주간의 React 강의 및 실습을 위한 자료를 체계적으로 정리한 공간입니다.

## 폴더 구조

```
📁 01-javascript-basics/
  ├─ week01-es6-basics/
  ├─ week02-async-programming/
📁 02-react-basics/
  ├─ week03-jsx-components/
  ├─ week04-props-state/
  ├─ week05-conditional-rendering/
📁 03-react-hooks/
  ├─ week06-useState-useEffect/
  ├─ week07-advanced-hooks/
📁 04-routing-state/
  ├─ week08-react-router/
  ├─ week09-state-management/
📁 05-final-projects/
  ├─ week10-12-main-project/
  ├─ mini-challenges/
📁 resources/
```

## 각 폴더 설명
- **01-javascript-basics/**: React 입문 전 필수 JS 기초/심화 학습
- **02-react-basics/**: React의 기본 개념, 컴포넌트, Props/State, 조건부 렌더링 등
- **03-react-hooks/**: useState, useEffect, useContext 등 주요 Hook 실습
- **04-routing-state/**: React Router, 상태관리(Context, Zustand 등)
- **05-final-projects/**: 실전 프로젝트 및 미니 챌린지
- **resources/**: 전체 가이드, 유용한 링크, 면접 대비 자료 등

## 학습 및 실습 안내
- 각 주차별로 교과서, 강의내용, 실습/예제 폴더가 제공됩니다.
- 실습 프로젝트는 각 주차별 폴더 내에 위치합니다.
- 최종 프로젝트는 10~12주차에 진행하며, 개발일지와 프로젝트 파일을 별도 관리합니다.

## 평가 기준
- 각 주차별 미니 프로젝트 완성
- 3주마다 복습
- 최종 프로젝트로 학습 완료도 평가
