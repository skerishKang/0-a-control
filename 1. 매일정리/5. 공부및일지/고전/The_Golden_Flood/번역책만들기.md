# 번역책 만들기 가이드

## 프로젝트 개요
- **프로젝트명**: The Golden Flood 번역 웹사이트
- **원작**: Edwin Lefevre의 "The Golden Flood" (1905)
- **목표**: 원본 영어 소설을 한국어로 번역하여 이중 언어 웹사이트로 제작

## 프로젝트 구조

```
The_Golden_Flood/
├── html/
│   └── The_Golden_Flood.html    # 메인 웹페이지
├── css/
│   └── styles.css               # 스타일시트
├── js/
│   └── script.js               # 인터랙티브 기능
└── 번역책만들기.md              # 이 가이드 파일
```

## 주요 기능

### 1. 이중 언어 표시
- **좌측**: 영어 원문
- **우측**: 한국어 번역
- **토글 기능**: English/한국어/Both 모드 전환

### 2. 사이드 네비게이션
- Part별 구분 (Part 1, 2, 3)
- 각 Part 내 섹션별 링크
- 진행률 표시기

### 3. 반응형 디자인
- 데스크톱/태블릿/모바일 지원
- 모바일에서는 상하 배치로 전환

### 4. 콘텐츠 구분
- **대화**: 배경색과 들여쓰기로 구분
- **서술**: 일반 텍스트 형태
- **제목**: 각 Part와 섹션별 명확한 구분

## 번역 가이드라인

### 1. 번역 원칙
- **직역보다는 의역**: 한국어로 자연스럽게
- **시대적 배경 고려**: 1905년 미국 금융계
- **인물 특성 보존**: 각 캐릭터의 개성 유지
- **전문 용어**: 금융/경제 용어의 정확한 번역

### 2. 인물명
- **George K. Grinnell**: 조지 K. 그리넬
- **Richard Dawson**: 리처드 도슨
- **William Mellen**: 윌리엄 멜렌

### 3. 기관명
- **Metropolitan National Bank**: 메트로폴리탄 국립은행
- **Assay Office**: 검정소
- **International Distributing Syndicate**: 국제유통신디케이트
