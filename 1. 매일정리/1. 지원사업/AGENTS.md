# AGENTS.md

이 폴더에서 작업할 때는 아래 기준을 따른다.

## 목적

지원사업 자료를 마감일 기준으로 빠르게 찾고, 공고문부터 제출본까지 한 흐름으로 관리한다.

## 폴더 생성 규칙

- 새 사업은 `YYYY-MM-DD_지원사업명` 형식의 폴더로 만든다.
- 날짜는 마감일 기준이다.
- 기존 폴더가 있어도 날짜 형식이 다르면 새 규칙으로 다시 정리하는 것을 우선한다.

## 파일 정리 규칙

- 공고 PDF, 신청 서식, 작성본, 제출본, 참고자료를 같은 사업 폴더 아래에 둔다.
- 초안은 `작성중`, 최종본은 `제출본`에 둔다.
- 동일 문서의 여러 버전은 파일명 끝에 날짜나 `v1`, `v2`를 붙인다.

## 문서 확인 규칙

- 공고문이나 신청서식에서 사업명, 마감일, 제출처를 먼저 확인한다.
- 확인된 사업명과 마감일을 기준으로 폴더명을 정한다.
- 마감일이 불명확하면 임시로 `9999-12-31_지원사업명` 템플릿을 복사해 쓰고, 나중에 이름을 바꾼다.

## 주의

- 원본 파일은 바로 덮어쓰지 않는다.
- 사업명이 길면 핵심 명칭만 남기고 폴더명은 짧게 정리한다.
- 제출 완료 후에는 `제출본` 기준으로 최종 상태를 유지한다.
