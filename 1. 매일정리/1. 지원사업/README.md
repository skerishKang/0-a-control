# 지원사업 폴더 운영 기준

이 폴더는 지원사업 공고, 신청서류, 작성본, 제출본, 참고자료를 마감일 기준으로 정리하는 곳이다.

## 폴더명 규칙

- 기본 형식: `YYYY-MM-DD_지원사업명`
- 예시: `2026-04-30_고성능컴퓨팅지원사업`
- 날짜는 공고 마감일 기준으로 쓴다.
- 동일한 사업이 여러 차수면 뒤에 기관명이나 차수를 덧붙인다.

## 권장 하위 구조

각 사업 폴더 안에는 아래 하위 폴더를 기본으로 쓴다.

- `공고문`
- `제출서류`
- `작성중`
- `제출본`
- `참고자료`

## 정리 원칙

- 공고 원문, 서식, 작성본, 제출본을 한 폴더 안에 모은다.
- 최종 제출 후에는 `작성중`의 핵심 파일만 남기고 정리한다.
- 사업과 직접 관련 없는 개인 메모는 넣지 않는다.
- 파일명은 가능한 한 날짜나 버전을 포함해 쓴다.

## 템플릿 폴더

- `9999-12-31_지원사업명` 폴더는 새 사업을 만들 때 복사해서 쓰는 템플릿이다.
