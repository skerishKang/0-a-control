이 영상은 n8n 자동화 툴을 사용하여 **AI Agent를 중심으로 워크플로우를 구성하고 다양한 기능을 연동하는 방법**에 대해 설명합니다.

핵심 내용은 다음과 같습니다:

1.  **n8n 워크플로우의 기본 구성 요소:**
    *   워크플로우를 처음 생성하는 방법 (Start from scratch 또는 Create 버튼 사용)
    *   워크플로우의 작업 영역 (중앙의 빈 공간)
    *   **노드(Node)**: 워크플로우를 구성하는 개별 작업 단위 (영상에서는 다양한 아이콘으로 표시됨)
    *   노드 연결: 각 노드 간의 데이터 흐름을 화살표로 연결.
    *   노드의 **Input(입력)** 과 **Output(출력)**: 데이터가 노드로 들어가고 나오는 지점 (각 노드 양쪽에 사각형 또는 원형 모양으로 표시됨)
    *   노드 실행/활성화/삭제: 각 노드 위에 나타나는 버튼으로 제어.
    *   실행 이력 확인: Editor 옆 Executions 탭에서 과거 실행 이력 확인 가능.

2.  **AI Agent 노드 상세 설명:**
    *   AI Agent 노드는 n8n의 고급 AI 기능 중 하나로, 외부 툴을 사용하거나 특정 작업을 수행하도록 설계할 수 있습니다.
    *   **필수 입력:** AI Agent 노드가 작동하려면 반드시 **Chat Model** (언어 모델)이 연결되어야 합니다 (연결 없이는 오류 발생).
    *   **선택 입력:** **Memory**(이전 대화 기억), **Tool**(특정 외부 기능 사용) 등은 필요에 따라 연결합니다.

3.  **AI Agent 기능 시연:**
    *   **기본 텍스트 입력 및 출력:** AI Agent 노드에 직접 텍스트를 입력하여 답변을 받는 간단한 테스트를 보여줍니다.
    *   **System Message 활용:** AI Agent의 기본 성격이나 역할을 설정하는 System Message를 변경하여 답변 스타일을 제어하는 것을 시연합니다.
    *   **Prompt Source 설정:** 이전 노드에서 자동으로 입력을 가져올지 (Take from previous node automatically), 직접 텍스트를 입력할지 (Define below) 선택할 수 있습니다.
    *   **Required Specific Output Format:** 출력 형식을 강제하여 원하는 JSON 형태로 결과를 받을 수 있도록 설정하는 방법을 보여줍니다.
    *   **Memory 연결:** **Window Buffer Memory** 노드를 AI Agent의 Memory 입력에 연결하여 이전 대화 맥락을 기억하도록 합니다. 이를 통해 이전 질문을 기억하고 답변하는 것을 시연합니다.
    *   **Tool 연결:** **Google Sheets** 노드를 AI Agent의 Tool 입력에 연결합니다. Google Cloud Platform에서 Google Sheets API를 활성화하고 OAuth 인증 정보를 설정하는 과정이 간략히 소개됩니다. AI Agent가 Google Sheet에 있는 데이터를 읽어서 질문에 답변하는 것을 시연합니다. Tool은 Google Sheets 외에 계산기, 위키피디아 등 다양하게 연결 가능합니다.

4.  **복합 워크플로우 구축 (설문지 응답 처리 예시):**
    *   **On Form Submission (설문지 응답) 트리거**를 사용하여 워크플로우를 시작합니다.
    *   **Edit Fields 노드**를 사용하여 설문지 응답 데이터에서 필요한 필드(이름, 음식 후기 등)를 추출하거나 가공합니다.
    *   AI Agent가 음식 후기 내용을 분석하여 "만족" 또는 "불만족"으로 평가합니다 (Prompt Source 및 Output Parser 설정으로 출력 형식 제어).
    *   **Structured Output Parser 노드**를 사용하여 AI Agent의 자유로운 텍스트 응답에서 핵심 정보("만족" 또는 "불만족")만 구조화된 형태로 추출합니다.
    *   **Switch 노드**를 사용하여 Output Parser에서 나온 결과 값("만족" 또는 "불만족")에 따라 워크플로우를 두 갈래로 나눕니다.
    *   각 갈래에 **Google Sheets 노드**를 배치하여 "만족"인 응답은 '만족' 시트에, "불만족"인 응답은 '불만족' 시트에 각각 데이터를 추가하도록 설정합니다.

결론적으로 영상은 n8n의 AI Agent를 소개하고, Chat Model, Memory, Tool이라는 핵심 기능을 이해하며, 나아가 설문지 응답을 받아 AI로 분석한 후 결과를 Google Sheets에 분류하여 저장하는 복합적인 자동화 워크플로우를 구축하는 과정을 단계별로 보여주고 있습니다.
