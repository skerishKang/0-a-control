## [1교시] Canvas 개념 이해와 도구 소개 (50분)

### 실습 1-1: 미니 실습 (15분)

**과제**: '업무 시작 전 팀에 보내는 간단한 공지 메일' 초안을 만드세요.

**방법**:
1. ChatGPT Canvas 또는 Gemini Canvas 중 하나를 선택
2. 아래 프롬프트를 입력하여 초안 생성
3. 생성된 문서를 확인

**프롬프트 예시**:
```
업무 시작 전 팀에 보내는 간단한 공지 메일을 작성해줘.
주제: [본인의 업무 관련 주제]
```

**작업 시간**: 3분

## [2교시] 공통점과 차이점 비교 실습 (50분)

### 실습 2-1: 비교 실습 (20분)

**과제**: 같은 주제를 두 도구에서 각각 생성해 비교하세요.

**주제**: "중원로지스 내부 AI 활용 가이드"

**프롬프트** (두 도구 모두 동일하게 입력):
```
당신은 실무 문서 작성 도우미입니다.
주제: 중원로지스에서 AI 도구를 활용하는 기본 가이드
대상: AI를 처음 사용하는 실무 직원
형식: 아래 순서
1. 도입 (왜 필요한가)
2. 실제 사용 예시 3가지
3. 바로 해볼 일 3가지
```

**작업 시간**: 10분 (각 도구 5분씩)
