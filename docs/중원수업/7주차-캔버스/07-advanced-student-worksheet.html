심화: Canvas 확장 활용 워크숍 실습지

데이터 정리, PPT 기획, 화면 설계, 실무 도구 제작 실습 활동지

1교시: 데이터 마스터링 (표 활용)

[과제 1] 흩어진 배차 정보 정리하기

Gemini Canvas를 열고, 제공된 난잡한 배차 텍스트를 붙여넣습니다.
프롬프트를 사용하여 [차량번호, 이름, 노선, 운송비]가 포함된 깔끔한 표로 변환합니다.
심화 지시: "운송비가 높은 순으로 정렬하고, 총합계 행을 추가해줘"라고 요청해봅니다.

2교시: 비주얼 기획 (PPT 스토리보드)

[과제 2] 보고서를 발표 슬라이드로 구성하기

ChatGPT Canvas를 열고, 긴 텍스트 보고서를 입력합니다.
"슬라이드 5장 분량의 PPT 스토리보드로 짜줘"라고 요청합니다.
심화 지시: "3번 슬라이드에 어울리는 추천 이미지나 도표 구성을 설명해줘"라고 요청해봅니다.

3교시: 디지털 화면 설계 (레이아웃)

[과제 3] 우리 팀만의 화면 구조 잡기

Gemini Canvas에서 "현장 기사님용 배차 확인 화면 구조를 잡아줘"라고 요청합니다.
브랜드 색상(Navy, White)과 가장 중요한 버튼(배차 수락)을 강조해달라고 합니다.
심화 지시: "모바일에서 볼 때 더 편하도록 버튼 크기를 키우고 문구를 다듬어줘"라고 요청합니다.

4교시: 나만의 실무 계산기 (도구 제작)

[과제 4] 자동 운임 계산기 만들기

Canvas에게 우리 회사만의 운임 계산 로직(거리당 단가, 할증 등)을 설명합니다.
"실제로 작동하는 계산 도구로 만들어줘"라고 요청합니다.
심화 지시: 기사님 성함을 입력하면 해당 기사님의 최종 정산액이 나오도록 기능을 추가해봅니다.
