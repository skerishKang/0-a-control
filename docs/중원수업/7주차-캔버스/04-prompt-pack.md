# 7주차 프롬프트 모음
## ChatGPT Canvas & Gemini Canvas 활용

---

## 1. 초안 생성 프롬프트

### A-1. 기본 초안 생성
```
당신은 실무 문서 작성 보조자입니다.
주제는 "[주제]"입니다.
대상 독자는 [대상]입니다.
형식: [원하는 구조]
```

### A-2. 보고서 초안
```
당신은 실무 문서 작성 전문가입니다.
목적: [주제]에 대한 보고서 작성
대상: [대상] 담당자
형식: 아래 섹션 순서
1. 현재 상황
2. 문제점
3. 해결 방안
4. 기대 효과
5. 추진 일정
```

### A-3. 안내문 초안
```
당신은 사내 안내 문서 작성 전문가입니다.
주제: [주제]
대상: [대상]
형식: 친근하면서도 격식 있는 어조, 항목형
```

### A-4. 제안서 초안
```
당신은 제안서 작성 전문가입니다.
목적: [주제]에 대한 제안서 목차 작성
대상: [고객사/팀] 담당자
형식: 아래 섹션 순서
1. 현재 문제 상황
2. 해결 방안
3. 기대 효과
4. 추진 일정
5. 비용 개요
```

## 2. 구조 변경 프롬프트

### B-1. 문단 → 항목 변환
```
생성된 내용을 항목 형식으로 다시 정리해 주세요.
각 항목은 2~3문장 이내로 짧게 해주세요.
```

### B-2. 항목 → 문단 변환
```
항목 형태의 내용을 문단형으로 바꿔주세요.
각 문단은 3~4문장으로 구성해 주세요.
```

### B-3. 요약 변환
```
생성된 내용을 핵심만 요약해 주세요.
총 5개 이내의 핵심 포인트로 정리해 주세요.
```

### B-4. 상세화 요청
```
생성된 내용을 더 상세하게 작성해 주세요.
구체적인 예시와 데이터를 포함해 주세요.
```

### B-5. 축소 요청
```
생성된 내용을 더 간결하게 줄여주세요.
핵심 메시지만 남겨주시고, 불필요한 표현은 제거해 주세요.
```

## 3. 톤/어조 조정 프롬프트

### C-1. 격식체로 변경
```
이 문서를 격식 있는 보고서 톤으로 다시 작성해 주세요.
공식적인 표현을 사용하고, 경어를 적절히 포함해 주세요.
```

### C-2. 친근하게 변경
```
이 문서를 친근한 어조로 다시 작성해 주세요.
가벼운 느낌이 나도록, 어려운 용어는 피하고 일상어를 사용해 주세요.
```

### C-3. 초보자용으로 변경
```
이 문서를 [주제]에 익숙하지 않은 초보자 대상으로 다시 작성해 주세요.
기술적 용어는 풀어서 설명하고, 단계별로 자세히 알려주세요.
```

### C-4. 전문적으로 변경
```
이 문서를 전문적인-tone로 다시 작성해 주세요.
적절한 기술 용어를 사용하고, 설득력 있는 표현을 포함해 주세요.
```

## 4. 표/목록 변환 프롬프트

### D-1. 실행 계획표 변환
```
생성된 내용을 실행 계획표로 변환해 주세요.
형식: | 순번 | 할 일 | 담당자 | 기한 | 진행 상황 |
```

### D-2. 비교표 생성
```
[주제]에 대해 ChatGPT Canvas와 Gemini Canvas를 비교하는표를 만들어주세요.
비교 항목: 초안 작성, 구조화, 문장 다듬기, 수정 반복, 실무 적합성
```

### D-3. 체크리스트 변환
```
생성된 내용을 실행 체크리스트로 변환해 주세요.
각 항목은 [ ] 형태의 체크리스트로 작성해 주세요.
```

### D-4.timeline 작성
```
생성된 내용을 시간순 진행 표(timeline)로 변환해 주세요.
형식: | 단계 | 내용 | 기한 | 비고 |
```

[특정 섹션]의 순서를 [새 순서]로 변경해 주세요.

## 6. Gemini 전용 프롬프트

### F-1. 구조 설계
```
[주제]에 대한 문서 구조를 설계해 주세요.
대상: [대상]
목표: [목표]
출력: 섹션별 개요 (상세 내용 없이 제목과 설명만)
```

### F-2. 아이디어 확장
```
다음 [내용]을 바탕으로 추가 아이디어를 제안해 주세요.
실무에서 바로 적용 가능한 수준으로 3가지를 제안해 주세요.
```

### F-3. 다중 자료 정리
```
아래 여러 자료를 하나의 문서로 정리해 주세요.
[자료1], [자료2], [자료3] …
형식: 주제별 카테고리 분류
```

## 7. ChatGPT 전용 프롬프트

### G-1. 글쓰기 코칭
```
이 문서의 설득력을 높여주세요.
감정적 호소와 논리적 근거를 적절히 추가해 주세요.
```

### G-2. 문장 개선
```
각 문장의 표현을 더 강력하고 명확하게 개선해 주세요.
장문 같은 표현은 줄이고, 핵심 메시지를 부각시켜 주세요.
```

### G-3. 버전 관리
```
이 문서의 버전을 만들어 주세요.
변경 사항을 기록하고, 이전 버전과 비교해 주세요.
```

## 8. 실전 조합 프롬프트

### H-1. Gemini + ChatGPT 조합 (뼈대→살)
```
[1단계 - Gemini]
"주제 [ ]에 대한 문서 구조를 설계해 주세요."

[2단계 - ChatGPT]
"위 구조를 바탕으로 각 섹션의 상세 내용을 작성해 주세요."

[3단계 - Gemini]
"생성된 내용을 표와 체크리스트로 변환해 주세요."

[4단계 - ChatGPT]
"최종 문장을 다듬고, 최종 검수해 주세요."
```

### H-2. 4단계 실무 문서 작성
```
[초안]
"당신은 [역할]입니다. [주제]에 대한 [형식] 문서를 작성해 주세요."

[구조]
"생성된 내용을 [원하는 구조]로 재구성해 주세요."

[톤]
"이 문서를 [대상] 대상으로 [원하는 톤]으로 다시 작성해 주세요."

[표]
"최종 결과를 [원하는 표 형식]으로 변환해 주세요."
```

## 9. 나쁜 프롬프트 vs 좋은 프롬프트

### 나쁜 프롬프트 예시
- ❌ "이거 해줘"
- ❌ "문서 좀 써줘"
- ❌ "보고서 작성"

### 좋은 프롬프트 예시
- ✅ "중원로지스 1분기 매출 보고서를 작성해줘. 매출, 배송 정시율, 고객 만족도 포함. A4 1장, 격식 체."
- ✅ "사내 AI 활용 안내문을 작성해주세요. 대상: 신규 직원, Tone: 친근함, Format: 항목형 5개"

## 사용 팁

1. **한 번에 하나씩**: 수정 요청은 한 번에 하나씩 하세요.
2. **구체적으로**: "더 잘 써줘"보다 "친근한 톤으로"가 더 낫습니다
3. **대상 명시**: 누가 읽는지 항상 명시하세요.
4. **형식 명시**: 표, 항목, 문단 등 원하는 형태를 말하세요.
5. **실패하면 다시**: 결과가 만족스럽지 않으면 다시 시도하세요.
