# 중원로지스 7주차 수업 계획

## 주제
- ChatGPT Canvas와 Gemini Canvas를 활용한 실무형 문서 작성, 수정, 확장 실습

## 수업 목표
- Canvas형 작업 공간의 개념을 이해한다.
- ChatGPT Canvas와 Gemini Canvas의 공통점과 차이점을 설명할 수 있다.
- 하나의 업무 주제를 두 도구에서 각각 확장하고 비교할 수 있다.
- 프롬프트를 단계별로 써서 초안 생성, 구조 개선, 수정 지시, 최종 정리를 수행할 수 있다.

## 수업 운영 방향
- 설명보다 실습 비중을 높인다.
- 같은 과제를 두 도구에서 반복해 보게 한다.
- "한 번에 완성"보다 "초안 -> 수정 -> 비교 -> 개선" 흐름을 강조한다.
- 결과물은 업무 문서 중심으로 잡는다.

## 4시간 전체 구성

### 1교시: Canvas 개념 이해와 도구 소개 (50분)
- 목표
  - Canvas가 일반 채팅과 어떻게 다른지 이해
  - ChatGPT Canvas와 Gemini Canvas의 역할 감 잡기
- 내용
  - Canvas란 무엇인가
  - 일반 채팅형 AI와 작업형 AI의 차이
  - 문서 초안 작성, 구조 수정, 재작성, 협업 관점 소개
  - ChatGPT Canvas 기본 사용 흐름
  - Gemini Canvas 기본 사용 흐름
- 데모
  - 같은 주제로 두 도구에서 초안 생성
  - 초안 일부를 수정 지시로 고치는 예시
- 미니 실습
  - "업무 보고 메일 초안" 또는 "회의 결과 정리" 1개 주제로 각자 초안 생성

### 2교시: 공통점과 차이점 비교 실습 (50분)
- 목표
  - 두 도구의 차이를 체감
  - 어떤 상황에 어떤 도구가 더 맞는지 판단
- 내용
  - 공통점
    - 긴 글 초안 작성 가능
    - 문단 단위 수정과 재작성 가능
    - 사용자 지시를 반영해 버전 개선 가능
    - 문서 작업의 반복 시간을 줄일 수 있음
  - 차이점
    - ChatGPT Canvas
      - 문장 다듬기, 구조 재조정, 버전 수정에 강함
      - 기존 초안을 붙여 넣고 손보는 흐름이 자연스러움
      - 글쓰기 코치처럼 점진 수정하기 좋음
    - Gemini Canvas
      - 아이디어 확장, 구조화, 정리, 자료형 변환에 강함
      - 큰 틀 설계나 문서 뼈대 확장에 유리함
      - Google 생태계와 함께 쓸 때 연결감이 좋음
  - 비교 기준
    - 초안 생성 속도
    - 수정 지시 반응
    - 표/목차/구조화 능력
    - 한국어 문장 자연스러움
    - 실무 문서화 적합성
- 비교 실습 과제
  - 주제: "중원로지스 내부 AI 활용 가이드 초안"
  - 두 도구로 각각 생성
  - 결과 비교표 작성

### 3교시: 실무 문서 제작 실습 (60분)
- 목표
  - 실제 업무 문서를 Canvas로 만든다
  - 초안 작성과 수정 요청 프롬프트를 구분해서 써본다
- 실습 주제 후보
  - 고객사 제안서 목차 초안
  - 사내 AI 활용 교육 안내문
  - 물류/운영 개선 아이디어 메모
  - 보고서용 실행 계획서
- 진행 방식
  - 1단계: 초안 만들기
  - 2단계: 문서 구조 바꾸기
  - 3단계: 대상 독자에 맞게 톤 조정
  - 4단계: 표, 체크리스트, 실행계획으로 재구성
- 강사용 체크 포인트
  - 한 번의 프롬프트로 끝내지 않게 한다
  - "추가", "정리", "요약", "재작성"을 구분해 지시하게 한다
  - 문서 목적과 독자를 먼저 적게 한다

### 4교시: 프롬프트 실전과 결과물 피드백 (60분)
- 목표
  - 쓸 만한 프롬프트 패턴을 손에 익힌다
  - 두 도구를 조합하는 방식을 익힌다
- 내용
  - 잘 되는 프롬프트 구조
    - 역할
    - 목적
    - 대상 독자
    - 원하는 산출물 형식
    - 수정 기준
  - 실패하는 프롬프트 특징
    - 목표가 없음
    - 독자가 없음
    - 산출물 형식이 없음
    - 수정 지시가 너무 뭉뚱그려짐
  - 도구 조합 예시
    - Gemini Canvas로 뼈대 설계
    - ChatGPT Canvas로 문장 다듬기
    - 다시 Gemini Canvas로 표/요약/확장
- 마무리 실습
  - 각자 1개 결과물 완성
  - 발표 또는 짧은 공유
  - 어떤 도구가 더 맞았는지 한 줄 평가

## 도구 공통점과 차이점 정리

### 공통점
- 긴 문서를 단순 채팅보다 안정적으로 다룰 수 있다.
- 초안을 만든 뒤 부분 수정과 전체 재구성이 가능하다.
- 업무 문서, 안내문, 계획서, 보고서 초안에 적합하다.
- 사용자가 지시를 잘게 나눌수록 결과가 좋아진다.

### 차이점

#### ChatGPT Canvas
- 글의 품질 개선과 재작성 흐름이 자연스럽다.
- 문장 다듬기, 표현 변경, 설득력 보강에 유리하다.
- 기존 문서가 있을 때 보완 작업이 편하다.

#### Gemini Canvas
- 구조 설계와 확장 아이디어 도출에 강하다.
- 문서를 표, 항목, 계획 형태로 재조직하기 좋다.
- 전체 프레임을 먼저 잡는 실습에 어울린다.

## 실습용 프롬프트 세트

### A. 초안 생성 프롬프트
```text
당신은 실무 문서 작성 보조자입니다.
주제는 "중원로지스에서 AI 도구를 활용하는 기본 가이드"입니다.
대상 독자는 AI를 처음 쓰는 실무 직원입니다.
너무 기술적으로 쓰지 말고, 바로 이해할 수 있는 한국어로 작성해 주세요.
형식은 아래 순서로 작성해 주세요.
1. 도입
2. 왜 필요한가
3. 실제 사용 예시 3개
4. 주의할 점
5. 바로 해볼 일 3개
```

### B. 구조 변경 프롬프트
```text
방금 작성한 내용을 설명문이 아니라 교육자료 형식으로 바꿔 주세요.
문단형 글보다 항목형 정리가 많도록 수정하고,
각 항목은 한눈에 읽히게 짧게 정리해 주세요.
마지막에는 강사가 바로 읽을 수 있는 진행 순서도 추가해 주세요.
```

### C. 톤 조정 프롬프트
```text
이 문서를 초보 직원 대상 안내문으로 다시 써 주세요.
너무 거창한 표현은 빼고,
실제로 오늘 바로 따라할 수 있는 수준으로 단순하게 고쳐 주세요.
```

### D. 비교 실습 프롬프트
```text
같은 주제를 두 도구로 작업한다고 가정할 때,
어떤 부분은 ChatGPT Canvas가 더 유리하고 어떤 부분은 Gemini Canvas가 더 유리한지
실무 문서 작성 관점에서 비교표로 정리해 주세요.
비교 항목은 초안 작성, 구조화, 문장 다듬기, 수정 반복, 발표자료 전환으로 잡아 주세요.
```

### E. 결과물 개선 프롬프트
```text
현재 초안의 문제점을 먼저 5가지 짚어 주세요.
그 다음 그 문제를 반영해 더 실무적으로 다시 작성해 주세요.
결과물은 보고용 문서 톤으로 정리해 주세요.
```

### F. 강사용 확장 프롬프트
```text
아래 수업 주제를 바탕으로 4시간짜리 실습 수업안을 더 촘촘하게 확장해 주세요.
각 시간대별로
1. 강의 포인트
2. 데모 내용
3. 학생 실습 과제
4. 질문 유도 포인트
를 나눠 작성해 주세요.
주제: ChatGPT Canvas와 Gemini Canvas 비교 및 실무 활용
```

## 강사용 진행 팁
- 먼저 완벽한 설명보다 바로 데모를 보여준다.
- 학생이 직접 두 도구를 모두 써보게 한다.
- 같은 프롬프트를 두 도구에 넣고 차이를 말하게 한다.
- "어떤 도구가 더 좋냐"보다 "어떤 작업에 더 맞냐"를 질문한다.
- 마지막 20분은 각자 자기 업무에 가까운 문서로 적용하게 한다.
