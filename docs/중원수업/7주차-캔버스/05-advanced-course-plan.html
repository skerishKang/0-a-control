심화: Canvas 확장 활용 워크숍 과정 계획서

데이터 정리, PPT 기획, 화면 설계, 맞춤형 계산기까지 — 실전을 위한 4시간 심화 세션

1. 교육 목표

  • Canvas의 표(Table) 기능을 통해 흩어진 물류 운영 데이터를 한눈에 보이는 리포트로 변환할 수 있다.
  • PPT 스토리보드 설계 기능을 활용해 보고서 내용을 발표 슬라이드 구조로 즉시 전환한다.
  • 디지털 화면 설계(Layout) 기능을 통해 현장용 웹 페이지나 앱 화면의 구조와 문구를 직접 설계한다.
  • 실무 전용 계산기 생성 로직을 체험하며, 복잡한 운임 산정 등을 자동화하는 나만의 도구를 만든다.

2. 교육 시간표

교시 시간 주제 핵심 수행 내용
1교시 50분 데이터 마스터링 비어있는 텍스트를 표로 변환, 운영 KPI 요약 생성
2교시 50분 비주얼 기획 보고서를 PPT 목차로 전환, 슬라이드별 시각 요소 설계
3교시 60분 디지털 화면 설계 차주 모집용 홍보 페이지 구조 기획, 앱 가이드 화면 설계
4교시 60분 나만의 실무 계산기 할증 운임 계산기 로직 생성, 실전 도구 제작 및 테스트

3. 교시별 세부 내용

1교시: 데이터 마스터링

카톡이나 메모장에 적힌 난잡한 배차 일지를 논리적인 표(Table)로 즉시 정리하고, 수익성 지표를 추가하여 분석 리포트로 업그레이드하는 기법을 배웁니다.

2교시: 비주얼 기획

텍스트 위주의 보고서를 발표용 스토리보드로 빠르게 변환하고, 슬라이드별 핵심 문구와 적절한 시각화 아이디어를 얻는 방법을 연습합니다.

3교시: 디지털 화면 설계

전문 디자인 기술 없이도 텍스트만으로 깔끔한 웹 시안 구조를 잡고, 버튼 배치와 핵심 문구를 AI와 상의하며 현장용 화면을 설계해봅니다.

4교시: 나만의 실무 계산기

매번 수기로 계산하던 복잡한 할증 로직을 자동 계산해주는 툴을 만들어보고, 코딩 지식 없이도 실무 도구를 생성해내는 과정을 체험합니다.
