  4. 제한능력자(미성년자 등)의 법률행위인 경우
  5. 착오, 사기, 강박에 의한 의사표시인 경우
  6. 소멸시효가 완성된 경우
  7. 동시이행의 항변권이 있는 경우

2. 주요 채권의 소멸시효기간 예시

  1. 일반채권: 민법상 10년, 상법상 5년
  2. 상인이 판매한 상품의 대가: 3년

  6. 파산/재판상 화해 등으로 확정된 채권: 10년 (단, 변제기 미도래 채무 제외)
