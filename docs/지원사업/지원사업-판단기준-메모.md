# 지원사업 판단기준 메모

파디엠 관련 지원사업을 검토할 때 쓰는 1차 필터 메모.
긴 대화나 LLM 분석 결과는 참고만 하고, 실제 판단은 공고 원문과 회사 팩트시트로 재검증한다.

## 핵심 원칙

- 무작위 공고 스크랩을 반복하지 않는다.
- 공고별 세부 검토 전에 `업력`, `지역`, `도메인 적합성`, `실증/인프라`, `파트너 구조`로 1차 컷한다.
- 신규 법인/지역 이전 카드는 아무 공고에나 쓰지 않는다.
- LLM 분석은 `1차 분류기`로만 쓰고, 최종 Go/No-Go 판정은 사람이 한다.

## 1차 컷 기준

### 1. 업력
- 창업 3년/7년 이내 제한이 있는지 확인
- 개인사업 개업일, 법인 설립일, 법인 전환일을 구분
- 제출 직전 최신 등기와 사업자등록증 기준으로 최종 확정

### 2. 지역
- 본점 기준인지, 사업장/지점/연구소 기준인지 구분
- 지점/연구소로 우회 가능한지, 신규 법인이 필요한지 따로 본다
- 지역 이전으로 해결 가능한 문제인지, 도메인 불일치라 해결 불가인지 구분한다

### 3. 도메인 적합성
- 파디엠 본체 기준 적합 영역:
  - AI
  - SW
  - 데이터
  - 공공안전
  - 디지털콘텐츠
  - 기업용 솔루션/API
- 기본적으로 불리한 영역:
  - 공장/제조라인 필수
  - 특수 하드웨어 실증 필수
  - 전통 제조업/농식품/지역 소상공인 전용
  - 의료기기 인허가 전제 사업

### 4. 실증/인프라
- 선박, 차량, 공장, 제조장비 등 실증 인프라가 필수면 단독 지원은 불리하다
- 이런 사업은 수요기업/제조사/공공기관과의 컨소시엄이 전제다

### 5. 파트너 구조
- 단독 지원형인지
- 대기업/공공기관 파트너 필수인지
- 지역 제조사/대학 컨소시엄형인지
- 파디엠이 주관기관이 되어야 하는지, 참여기관이 더 현실적인지 구분

### 6. 재무/행정 부담
- 민간부담금 비율
- 보증보험 필요 여부
- 신규채용 의무
- 기술료
- 보고/정산/증빙 부담

## 전략 판단

### 본체 법인으로 우선 검토
- 업력 무관 또는 현재 업력에 걸리지 않는 사업
- 전국 단위 또는 광주 수용
- AI/SW/데이터 중심
- 과도한 제조 인프라가 필요 없는 사업

### 신규 법인 카드 검토
- 업력 제한이 핵심 장벽일 때
- 지역 제한도 함께 풀어야 할 때
- 기존 파디엠 기술을 스핀오프해도 도메인 정합성이 유지될 때

### 과감히 패스
- 도메인 자체가 맞지 않을 때
- 지역/업력만 바꿔도 해결되지 않을 때
- 지원금은 작고 행정/이전/설립 비용이 더 클 때

## 주의

- `100% 불가`, `무조건 된다` 같은 표현은 피한다.
- 공고 원문에 예외조항, 참여기관 허용, 지점 인정 기준이 있을 수 있다.
- 판단 메모는 전략적 참고용이지 법적 판정문이 아니다.
