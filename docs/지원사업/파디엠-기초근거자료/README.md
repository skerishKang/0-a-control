# 파디엠 지원사업 기초근거자료

지원사업 적합성 판단, 자격 검토, 컨소시엄/신규법인 전략 검토 시 먼저 참조할 기본 문서 모음.

포함 파일
- `250801_파디엠법인사업자등록증.pdf`: 사업자등록증
- `250207_파디엠등기.pdf`: 법인 등기 관련 문서
- `2023파디엠재무제표.pdf`: 재무 상태 확인용
- `파디엠R&D평가서.pdf`: R&D/사업 역량 참고 자료
- `이력사항-강철원.pdf`
- `이력사항-강철원.hwp`
- `이력사항-강혜림.pdf`
- `이력사항-강혜림.hwp`

사용 원칙
- 새 지원사업 검토 시 이 폴더의 자료를 기본 회사 팩트로 본다.
- LLM 분석 결과만 믿지 말고, 공고 원문과 이 자료를 교차 검증한다.
- 신규 법인, 지역 이전, 컨소시엄 판단 시에도 이 자료를 공통 근거로 사용한다.
