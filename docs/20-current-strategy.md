# 현재 전략

## 현재 단계

**전체 조망 + 퀘스트성 강화 동시 진행**

- 전체 조망: Support Zone 패널로 상황 압축
- 퀘스트성: Hero Card로 "지금 1개만 잡기"

---

## 핵심 철학

### 1. 전체를 보되 한 번에 1개 퀘스트만 잡기
- 오늘 판단 요약: 5초 내에 전체 상황 파악
- 현재 퀘스트: "지금 이것만" 집중

### 2. Support Zone 패널 순서 (경보 → 판단 → 계획 → 기록 → 참고)
1. 기한 임박 (경보)
2. 오늘 판단 요약 (판단)
3. 프로젝트에서 온 추천 퀘스트 (판단)
4. 우선 검토 후보 (판단)
5. 단기 플랜 (계획)
6. 장기 플랜 (계획)
7. 반복 플랜 (계획)
8. 미완료 플랜 (기록)
9. 최근 세션 (기록)
10. 최근 브리프 (참고)
11. 텔레그램/이메일 (참고)
12. workdiary 폴더 (참고)

### 3. AI 추천의 위치
- AI 추천은 **상단 확정 영역이 아니라 지원영역**에서 판단 보조 역할
- 사용자 계획(plan-item)보다 위에 위치하지 않음
- "사용자 확정 > AI 추천" 원칙 유지

## 아직 부족한 것

1. **전체 조망의 압축**
   - 12개 패널이 너무 많음
   - 더 간결한 "오늘 핵심" 필요

2. **퀘스트 완료 → 다음 퀘스트 전환감**
   - 현재: Quest 보고 → 다음후보 직접 이동
   - desired: Quest 완료 후 "오늘 판단 요약"에서 흐름 확인

3. **새로고침 없이 실시간 반영**
   - 현재: 페이지 새로고침 필요
   - desired: 보고 제출 후 즉시 UI 반영
