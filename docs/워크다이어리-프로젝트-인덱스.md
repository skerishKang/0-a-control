# 워크다이어리 프로젝트 인덱스

워크다이어리 최상위 폴더가 너무 많아서, 자주 다시 찾을 가능성이 높은 프로젝트만 먼저 묶어 둔 인덱스.
이 문서는 `이 아이디어 예전에 했었나?`, `비슷한 폴더가 어디였지?`를 빠르게 찾기 위한 용도다.

## 현재 가장 중요한 연결

### LLM 프라이버시 프록시 아이디어
- 가장 가까운 기존 프로젝트: [24-secure-bridge](/mnt/g/Ddrive/BatangD/task/workdiary/24-secure-bridge)
- 이유:
  - 로컬 처리
  - Zero-Leakage
  - PII 익명화/복원
  - 보안 브리지/프록시 성격
- 보조 연결 프로젝트:
  - [87localllmui](/mnt/g/Ddrive/BatangD/task/workdiary/87localllmui): 로컬 LLM UI/Ollama 브리지
  - [102-limone-voice-safety](/mnt/g/Ddrive/BatangD/task/workdiary/102-limone-voice-safety): 음성 보안/보이스피싱 탐지

## 운영 메모

- 이 인덱스는 전체 워크다이어리 완전목록이 아니라 `다시 찾을 가능성이 높은 축`만 우선 정리한 것이다.
- 이름만 있고 README가 없는 폴더는 추후 필요할 때 한 줄 설명을 보강한다.
- 현재 기준으로 `LLM 프라이버시 프록시` 아이디어는 새 폴더를 파기보다, 먼저 `24-secure-bridge`와의 연결을 중심으로 보는 것이 맞다.
