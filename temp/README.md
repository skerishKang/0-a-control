# temp 폴더 안내

이 폴더는 `0-a-control`의 임시 보관 구역이다.

## 목적
- 루트에 임시 파일, 로그, 스크린샷, 실험 스크립트가 쌓이지 않게 한다.
- 당장 삭제하기는 아깝지만 프로젝트 핵심 구조에 속하지 않는 파일을 비파괴적으로 옮겨 둔다.

## 여기로 옮기는 것
- 일회성 로그
- 수동 점검 스크린샷
- OCR/알고리즘 같은 임시 실험 스크립트
- 외부에서 잠깐 가져온 zip, 폴더, 개인 메모
- 잘못된 경로로 생성된 stray 파일

## 여기 두지 않는 것
- 실제 실행 진입점
- 공식 문서
- 프로젝트 코드
- 장기 운영 규칙

## 원칙
- `temp/`는 기본적으로 git 추적 대상이 아니다.
- 필요 시 사람이 직접 다시 꺼내 쓸 수 있도록 의미 있는 하위 폴더로만 정리한다.
